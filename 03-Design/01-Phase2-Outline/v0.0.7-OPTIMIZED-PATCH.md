## P0 优化（必须）

### 1. 诗词格律修正

#### 1.1 第84回回末诗（失粘修正）

**原诗**:
> 文字缘深命里浅，惊闻变故泪潸然。
> 诗稿焚尽痴情在，一缕香魂待上天。

**问题**: 第三句"诗稿焚尽痴情在"与首句"文字缘深命里浅"失粘（平仄不协）

**修正后**:
> 文字缘深命里浅，惊闻变故泪潸然。
> 焚尽诗稿痴情在，一缕香魂待上天。

**说明**: "焚尽诗稿"（平仄仄仄）对"文字缘深"（平仄平平），符合仄起七绝格律

---

#### 1.2 第92回回末诗（第三句调整）

**原诗**:
> 家宅残破恶子猖，危楼产子命如霜。
> 桂子初生声细细，不知身世已苍凉。

**问题**: 第三句"桂子初生声细细"与首句失粘

**修正后**:
> 家宅残破恶子猖，危楼产子命如霜。
> 初生桂子声细细，不知身世已苍凉。

**说明**: "初生桂子"（平平仄仄）与"家宅残破"（平仄平仄）平仄相协

### 4. 宝钗主动性软化

#### 4.1 第84回：从"直接询问"改为"梦境暗示"

**原文**:
```
【宝钗主动】宝钗私下询问莺儿："老爷和太太...究竟是怎么打算的？"
【宝钗内心】："我虽知不该问...但这事关乎我一生...我不能只等别人安排..."
```

**优化后**:
```
【宝钗暗示】当夜宝钗梦见自己嫁给宝玉，但梦中自己一直在哭，醒来枕边有泪痕
【宝钗内心】："这梦...是什么预兆...我...真的想吗..."
【宝钗动作】次日整理黛玉诗稿时，手微微颤抖，无意识地将"侬今葬花人笑痴"一句反复摩挲
【宝钗内心】："她的诗...她的命...我..."
【暗示】宝钗的"主动"从言语改为梦境+无意识动作，更符合"停机德"的克制人设
```

**说明**: 保持宝钗主动性（德vs情矛盾），但从"言语直接"改为"梦境+动作暗示"，更含蓄内敛

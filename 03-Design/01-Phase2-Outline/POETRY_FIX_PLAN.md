## 二、详细修复方案

### 方案A：保守修正（推荐）
**原则**：最小改动，保留原诗意象，仅修正格律

**示例 - 84回修正**：
```
原诗：
文字缘深命里浅，惊闻变故泪潸然。
诗稿焚尽痴情在，一缕香魂待上天。

问题：
- "命里浅"：仄仄仄（三仄尾）
- "待"：应平而仄（出律）

修正后：
文字缘悭命本悭，惊闻变故泪潸然。
诗稿焚余情未了，香魂一缕待归天。

验证：
- 仄仄平平仄仄平 ✓
- 平平仄仄仄平平 ✓
- 平仄仄平平仄仄 ✓
- 平平仄仄仄平平 ✓
```

### 方案B：意境重作（适用于92、107回）
**原则**：完全重作，但保持原诗情感和意象

**示例 - 92回重作方向**：
```
原诗主题：宝钗危楼产子，末世悲凉
原诗意象：恶子猖、危楼、命如霜、桂子、声细细

重作思路：
1. 保持"产子"主题
2. 保留"恶子猖狂"背景
3. 保留"命运悲凉"情感
4. 重新选韵（建议用"阳"韵或"庚"韵）

试作示例：
家宅飘摇恶子狂，危楼产子命如霜。
初生啼哭声犹弱，未知身世已苍凉。

验证：
- 平仄需逐字校验
- 避免三平尾/三仄尾
```

## 三、统一格律标准

### 推荐格式：七绝仄起首句不入韵

```
仄仄平平平仄仄（首句）
平平仄仄仄平平（对句）
平平仄仄平平仄（转句）
仄仄平平仄仄平（结句）
```

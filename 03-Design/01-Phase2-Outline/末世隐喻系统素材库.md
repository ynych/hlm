## 二、v0.0.7末世隐喻系统设计

### 2.1 设计原则

1. **不直接点明**：不用"悼明""反清"等词
2. **意象系统**：通过景物、诗词、对话暗示
3. **开放解读**：可理解为"悼明"或"身世"
4. **前80回呼应**：模仿前80回的隐喻密度

### 2.2 人物对话隐喻

**凤姐临终**（第95回）：
```
原方案："去他娘的戎羌"（过于直白）

B1方案：
"我这一生，机关算尽，算来算去，算不过命。
如今家破人亡，才知道这世上，没有什么能长长久久的。"

隐喻：王朝更替，个人无力回天
```

**宝玉出家前**（第104回）：
```
"国在家亡，不如出世。"

隐喻：家国破灭，遁入空门
开放解读：可指贾府（家），也可指明朝（国）
```

**探春远嫁**（第87回）：
```
"清明涕送江边望，千里东风一梦遥。
这一去，不知还能不能回来。"

隐喻：南明遗民远离故土
```

### 2.3 回末诗隐喻

**第81回**（渐衰开始）：
```
藕香榭外水潺潺，旺相难占运已残。
残荷听雨声声慢，一叶飘零一叶寒。

新增："残荷""飘零"暗示末世
```

**第89回**（抄家）：
```
官差突至满门惊，籍没家产一片腥。
大厦倾颓风雨骤，满座衣冠似散星。

新增："大厦倾颓"呼应"忽喇喇似大厦倾"
```

**第104回**（悬崖撒手）：
```
悬崖撒手万缘空，酸齑充饥雪没膝。
白茫茫大地真干净，万古红尘一梦归。

保留"白茫茫大地"核心隐喻
```

## 五、使用建议

### 5.1 密度控制
- 平均每回1-2处末世隐喻（景物/诗词/对话）
- 避免过度，以免说教

### 5.2 层次设计
- **表层**：家族衰败（身世说）
- **深层**：王朝更替（悼明说）
- **哲学层**：盛极必衰（普遍性）

### 5.3 关键节点
- 81回：末世开始（隐喻启动）
- 89回：大厦倾颓（隐喻强化）
- 104回：白茫茫大地（隐喻总结）

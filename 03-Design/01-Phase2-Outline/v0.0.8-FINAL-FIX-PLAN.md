### 2. 心理描写修复（去直白化）

#### 2.1 第82回：删除直白内心标注

**原设计**（直白）：
```
【宝钗内心】："我虽羡她才情，却也不忍见她如此。只是这府里，谁不是命悬一线？"
【黛玉内心】："她倒是好意，只是我这身子，怕是撑不过去了。"
```

**修复后**（暗示）：
```
宝钗望着黛玉咳血的帕子，手微微一颤，忙别过脸去整理药碗。
黛玉望着窗外落叶，轻声道："这叶子...落得比往年早..."
紫鹃在旁，见两人各怀心事，不敢言语。
```

**修复要点**：用"手颤""别过脸""望落叶"暗示心理，不说破

---

#### 2.2 第84回：宝钗主动性软化

**原设计**（直白主动）：
```
【宝钗主动】宝钗私下询问莺儿："老爷和太太...究竟是怎么打算的？"
【宝钗内心】："我虽知不该问...但这事关乎我一生...我不能只等别人安排..."
```

**修复后**（暗示+间接）：
```
宝钗整理黛玉诗稿时，手指停在"侬今葬花人笑痴"一句上，久久未动。
夜间，宝钗梦见自己嫁给宝玉，却一直在哭，醒来枕边有泪痕。
次日，宝钗"无意"中向莺儿提起："近日...老爷和太太...可曾提起什么？"
```

**修复要点**：从"直接询问"改为"梦境暗示+间接探听"，更符合"停机德"人设

---

#### 2.3 第94回：妙玉悔恨去直白

**原设计**（直白）：
```
【妙玉】："我一生洁净，如今却陷在这淖泥里。"
【妙玉】："刘姥姥...我当初不该..."
```

**修复后**（暗示）：
```
妙玉望着雨滴落在杯中，那雨滴与当年栊翠庵的雨...仿佛一样...
妙玉低头，望着手中那枚要丢未丢的成窑杯碎片，久久不语。
一滴泪，落入杯中，与雨水混在一起。
```

**修复要点**：用"望雨滴""握碎片""泪落杯中"暗示悔恨，不说"报应""不该"

---

### 3. 情榜说教修复（去点评化）

#### 3.1 第106回（合并后）：仅列名+判词，无评语

**原设计**（说教）：
```
警幻出示正册，评语："痴情第一，泪尽而亡"
```

**修复后**（留白）：
```
警幻出示正册，十二道光芒依次亮起：
- 林黛玉（白光闪过，配判词："可叹停机德，堪怜咏絮才..."）
- 薛宝钗（金光闪过，配判词："玉带林中挂，金簪雪里埋"）
- ...（以此类推）
警幻仅叹："千红一哭，万艳同悲。"（一句话总结，其余不说）
```

**修复要点**：删除所有"评语"，仅保留判词原文，与第五回风格一致

---

#### 3.2 第108回：顿悟去金句化

**原设计**（金句说教）：
```
【宝玉金句】："情不情者，最终情亦是空。"
```

**修复后**（意象呈现）：
```
宝玉站在悬崖边，望着白茫茫大地。
一阵风吹过，卷起几片落叶，飞向远方。
宝玉伸手，接住一片落叶，又让它随风飘去。
【旁白】：他终究...还是那块石头...但又不是那块石头了...
```

**修复要点**：用"接落叶又放走"暗示"情在但不再执着"，不用金句总结

## P1级优化（建议修复）

### 4. 末世隐喻去标签化

#### 4.1 第91回：删除"末世红颜多薄命"直白对话

**原设计**（贴标签）：
```
紫鹃叹："这世道，好人没好报，末世红颜多薄命。"
```

**修复后**（暗示）：
```
紫鹃望着窗外寒鸦，喃喃："这鸟...去年...好像没这么多..."
宝玉在旁，望着那寒鸦，不语。
```

**修复要点**：用"寒鸦增多"暗示末世，不说"末世"二字

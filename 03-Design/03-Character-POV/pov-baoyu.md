# 贾宝玉 角色视角关键剧情

**角色**: 贾宝玉（神瑛侍者转世）  
**视角**: 第一主角  
**覆盖**: 第1-108回

---

## 前80回关键剧情

### 第1回：通灵宝玉下凡
- **关键事件**：石头被茫茫大士、渺渺真人携带下凡，投胎为贾宝玉口中衔玉而生
- **宝玉视角**：出生即带玉，被视为奇事
- **关键物品**：通灵宝玉（刻"莫失莫忘，仙寿恒昌"）

### 第3回：黛玉进府
- **关键事件**：林黛玉进贾府，与宝玉初见
- **宝玉视角**："这个妹妹我曾见过的"，似曾相识
- **心理活动**：觉得黛玉与众姐妹不同，格外亲近

### 第5回：梦游太虚幻境
- **关键事件**：宝玉梦游太虚幻境，看金陵十二钗判词
- **宝玉视角**：似懂非懂，未能领悟命运预兆
- **关键物品**：警幻仙姑传授"云雨之事"

### 第8回：比通灵金莺微露意
- **关键事件**：宝钗看通灵宝玉，莺儿说出与宝钗金锁是一对
- **宝玉视角**：开始注意宝钗的金锁，埋下"金玉良缘"伏笔

### 第17-18回：大观园题咏
- **关键事件**：宝玉为大观园各处题匾额对联
- **宝玉视角**：展现诗才，得到贾政认可

### 第23回：共读西厢
- **关键事件**：宝黛共读《西厢记》，黛玉听到"花落水流红"
- **宝玉视角**：第一次对黛玉表白（"我就是个'多愁多病身'，你就是那'倾国倾城貌'"）

### 第27回：黛玉葬花
- **关键事件**：黛玉葬花，吟诵《葬花吟》
- **宝玉视角**：在山坡上听到，恸倒在山坡之上，深切理解黛玉的悲伤

### 第29回：痴情女情重愈斟情
- **关键事件**：宝黛大闹，宝玉砸玉
- **宝玉视角**：因误会而痛苦，"我都改"的无奈

### 第34回：宝玉挨打
- **关键事件**：贾政痛打宝玉，黛玉哭得最伤心
- **宝玉视角**：送黛玉旧帕，明白黛玉真心

### 第36回：绣鸳鸯梦兆绛芸轩
- **关键事件**：宝钗绣鸳鸯，宝玉梦中喊"什么是金玉姻缘，我偏说是木石姻缘"
- **宝玉视角**：潜意识中坚定选择黛玉

### 第76回：凹晶馆联诗
- **关键事件**：中秋夜，黛玉、湘云联诗，"寒塘渡鹤影，冷月葬花魂"
- **宝玉视角**：在远处听到，预感不祥

## v0.0.7大纲（81-108回）

### 第81回：四美钓鱼
- **关键事件**：探春钓鱼时叹"好景不长"
- **宝玉视角**：见探春感慨，心中不安
- **通灵玉视角**：记录渐衰开始

### 第82回：宝钗探病
- **关键事件**：宝钗探望黛玉，宝玉在场
- **宝玉视角**：不知二人交谈内容，只觉气氛凝重
- **心理活动**：隐约感到某种变化

### 第84回：宝玉议婚
- **关键事件**：贾政与薛姨妈商议宝玉婚事
- **宝玉视角**：蒙在鼓里，以为要娶黛玉

### 第85回：错姻缘（核心悲剧）
- **关键事件**：
  1. 成婚当日被骗娶宝钗，以为娶的是黛玉
  2. 见是宝钗，当场昏厥
  3. 婚后得知黛玉已逝（前一日病逝）
  4. 重游潇湘馆，见白绫挂梅枝
- **宝玉视角**：
  - 大喜→大惊→大悲→大悔
  - "我原以为是她，怎么会是她？"
  - 见白绫挂梅枝，痛哭失声："黛玉，是我负你！"
- **通灵玉视角**：记录宝玉悲痛欲绝

### 第88回：凤姐托孤
- **关键事件**：凤姐向刘姥姥托付巧姐
- **宝玉视角**：旁观，感叹凤姐一生

### 第89回：抄家入狱
- **关键事件**：贾府被抄，宝玉入狱
- **宝玉视角**：
  - 从富贵公子到阶下囚
  - 在狱神庙中反思一生
- **通灵玉视角**：记录宝玉失落

### 第90回：贾母病逝
- **关键事件**：贾母去世，宝钗挺孕肚主丧
- **宝玉视角**：失去最疼爱自己的祖母，悲痛

### 第91回：红玉探监
- **关键事件**：红玉夫妇探望宝玉
- **宝玉视角**：感激，感叹人情冷暖

### 第92回：宝钗产子
- **关键事件**：宝钗产下贾桂
- **宝玉视角**：
  - 短暂喜悦
  - 但心中始终放不下黛玉
  - 对宝钗愧疚

### 第94回：妙玉被劫
- **关键事件**：妙玉被乱兵劫走
- **宝玉视角**：
  - 多方寻找，但未能找到
  - 得知妙玉被卖入凝香院，但无力赎回
  - 愧疚："我当初就该..."
- **心理活动**：对女性的无力保护

### 第97回：宝钗病逝
- **关键事件**：宝钗病逝，宝玉归来
- **宝玉视角**：
  - "我对不起宝钗，也对不起黛玉"
  - 托付贾桂给湘云
- **心理活动**：双重愧疚

### 第98回：重游潇湘馆
- **关键事件**：宝玉重游潇湘馆，拾得黛玉旧帕
- **宝玉视角**：
  - 睹物思人，泪湿衣襟
  - "黛玉，你临终前，可曾怨我？"

### 第100回：刘姥姥救巧姐
- **关键事件**：巧姐被救，嫁板儿
- **宝玉视角**：感叹凤姐一生算计，不如刘姥姥一诺

### 第102回：湘江边重逢湘云
- **关键事件**：
  1. 流浪至湘江边，闻琴声熟悉
  2. 隔船相望，认出湘云（已成歌妓）
  3. 湘云托付贾桂
  4. 湘云病逝
- **宝玉视角**：
  - "云妹妹，你也..."
  - 接受贾桂，承诺抚养
  - 目睹湘云病逝，悲痛欲绝
- **金麒麟视角**：记录重逢与永别

### 第103回：甄宝玉送玉
- **关键事件**：甄宝玉送还通灵玉，点化
- **宝玉视角**：
  - "真假"之辨
  - 开始领悟

### 第104回：悬崖撒手
- **关键事件**：宝玉悬崖撒手，遁入空门
- **宝玉视角**：
  - "国在家亡，不如出世"
  - 放下一切执念
- **通灵玉视角**：记录宝玉出家

### 第105回：通灵玉归山
- **关键事件**：茫茫大士、渺渺真人收回通灵玉
- **宝玉视角**：
  - 与玉分离
  - 彻底放下红尘

### 第106-108回：情榜揭晓
- **关键事件**：太虚幻境，情榜揭晓
- **宝玉视角**（回忆）：
  - 十二钗身影闪过
  - 最终领悟"情"字

```
  懵懂→痴情→挣扎→坚定（选黛玉）

  不安→大喜→大悲→大悔（错娶）

86-97回：
  愧疚（对宝钗）→痛苦（双愧疚）

98-104回：
  绝望→领悟→出家

105-108回：
  超脱→见证情榜
```

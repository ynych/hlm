## 自动化执行流程

```
while (总分 ≤ 9.5) {
    1. 扫描当前问题
    2. 按优先级修复
    3. 执行 /review --all
    4. 计算评分变化
    5. if (提升 > 0.5) 升级版本号
    6. 提交并推送
    7. 生成评估报告
    8. 等待下一轮 (或cron触发)
}
锁定最终版本
```

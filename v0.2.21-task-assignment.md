## 详细任务书

### 【T01】宝玉语言含蓄化工程

**Owner**: 脂砚斋（文本风格师）  
**配合**: 贾宝玉顾问（台词是否符合人物）

**原始文本**（第100回）：
```
宝玉心中一动。
这是他流浪以来...
第一次...主动...保护别人...

"原来...主动...是这么痛快..."
```

**修改目标**: 保留"主动"情感转折，但"痛快"过于直白，需含蓄化

**备选方案**（脂砚斋拟定）：
- 方案A: "原来...主动...心中一动，似有所悟..."
- 方案B: "原来...主动...竟是如此滋味..."
- 方案C: "原来...主动...比那富贵闲人...痛快百倍..."（半保留）

**交付要求**:
1. 提供3种备选修改
2. 贾宝玉顾问逐句评分"像/不像"
3. 选取得分最高方案

**风险预案**: 若宝玉顾问三次评分<80分，触发Echo介入，提交曹霑裁决

### 【T02】湘云身份含蓄化工程

**Owner**: 畸笏叟（创意激发师）  
**配合**: 判词解（命运逻辑校验）

**原始文本**（第102回）：
```
湘云沦为船妓...
```

**修改目标**: 保留"沦落底层"命运，但"船妓"二字过于直白，需含蓄

**修改原则**（判词解提出）：
- 必须保留"湘江水逝楚云飞"的意象
- 必须保留"船上"场景
- 必须保留"诗酒风流"的应对方式

**备选方案**（畸笏叟拟定）：
- 方案A: "湘云在船上以诗酒度日..."
- 方案B: "湘云流落江上，以诗换酒..."
- 方案C: "湘云在那船上，以诗自娱，以酒遣怀..."

**交付要求**:
1. 提供3种备选修改
2. 判词解校验是否符合"湘江水逝"命运逻辑
3. 选取得分最高方案

**风险预案**: 若判词解认为含蓄化偏离命运逻辑，触发Echo介入

### 【T03】凤姐托孤台词微调

**Owner**: 王熙凤顾问（人物自身）  
**配合**: 脂砚斋（文本润色）

**原始文本**（第95回）：
```
"告诉她...她娘...不是好人...但...没害过人..."
```

**修改目标**: 将"不是"改为"不算"， soften一分强撑口气

**Owner裁决**（王熙凤顾问）：
> "畸笏叟前辈说得对，'不算'比'不是'软一分，更符合我那强撑的口气。我不是在认罪，是在解释。'不算'是给自己留面子，也是给巧姐留面子。"

**修改后**:
```
"告诉她...她娘...不算好人...但...没害过人..."
```

**交付要求**: 仅一字之改，Owner确认即可

**状态**: ✅ 已确认，无需风险预案

### 【T04】袭人送别留白工程

**Owner**: 太虚幻境（情节架构师）  
**配合**: 贾宝玉顾问（人物反应是否符合）

**原始文本**（第108回）：
```
袭人追到悬崖边：
"二爷...您...真的要走..."

宝玉回头：
"袭人...这些年...苦了你了...
这红尘...我已无牵挂..."

袭人跪倒在地，泣不成声。

宝玉转身，不再回头。
```

**修改目标**: 删除直白对话，改用留白

**修改后**（太虚幻境拟定）：
```
袭人追到悬崖边，见宝玉立于崖畔，背影萧索。

她张了张口，却终究没有出声。
只跪倒在地，泣不成声。

宝玉没有回头。
片刻后，纵身一跃——
```

**交付要求**:
1. 贾宝玉顾问评分"像/不像"
2. 脂砚斋批语确认留白效果

**风险预案**: 若宝玉顾问认为"没有回头"过于决绝，可调整为"微微侧首，终未转身"

### 【T05】宝钗病程语言优化

**Owner**: 薛宝钗顾问（人物自身）  
**配合**: 脂砚斋（含蓄化润色）

**原始文本问题**（第97回）：
> 部分描写过于直白，如"那温柔，她给了一辈子，他却在她死后，才真正懂得"

**修改目标**: 将直白点评改为含蓄暗示

**Owner要求**（薛宝钗顾问）：
> "我那7天病程，本就写得好。但有些地方，脂砚斋说得对，太直白了。比如回望蘅芜苑那段，不该点破'温柔给了一辈子'，让他自己悟去。"

**修改示例**（脂砚斋拟定）：

**原文**:
```
蘅芜苑的香草已经枯萎，
却仿佛看见宝钗站在那里，静静地望着他...
那温柔，她给了一辈子，他却在她死后，才真正懂得。
```

**改后**:
```
蘅芜苑的香草已经枯萎，
却仿佛看见宝钗站在那里，静静地望着他...
宝玉心中一痛，想起她平日里的温言软语，
如今都成了锥心刺骨。
```

**交付要求**:
1. 逐段检查第97回直白点评
2. 薛宝钗顾问确认"像我"
3. 脂砚斋确认含蓄度

**风险预案**: 若宝钗顾问认为含蓄化削弱情感力度，保留原文

```
他想说些什么，却只觉喉头哽咽。
```

```
芍药早已凋零，只剩下枯枝。
宝玉想起那年春日，湘云醉卧花下，口中吟着"只恐夜深花睡去"。
如今花已谢，人亦散。
```

**蘅芜苑**:
```
香草枯萎，冰雪覆盖。
宝玉想起宝钗平日里的话不多，却句句在理。
如今再想听一句，已是不能了。
```

**交付要求**:
1. 五位人物顾问分别确认各自地点"像我"

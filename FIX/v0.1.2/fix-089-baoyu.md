**修复级别**: P1  
**原文回目**: 第八十九回 元妃薨贾府树倒猢狲散 宝玉等通辑捉拿入狱

## 原文问题

原文中宝玉入狱罪名模糊：
```
那官府的人冷笑道："政治上的事，说了你也不懂。我们只奉命捉人，其他的，你去问上头罢。"
```

**问题**: "政治案件"未具体化，不符合清代政治犯通常有明确罪名（如结交近侍、图谋不轨等）的惯例。

## 修复说明

### 历史考据依据

**清代政治犯罪名惯例**:
1. **结交近侍**: 清代严禁外官与内廷太监、侍卫结交，违者以"图谋不轨"论处
2. **党争株连**: 康熙朝索额图/明珠党争、雍正朝年羹尧/隆科多案，均有株连亲属先例
3. **忠顺王府原型**: 对应《红楼梦》原文中的忠顺王府，与贾府为敌对势力

### 与前后文呼应

1. **83回元妃托梦**: "虎兕相逢大梦归" + "退步抽身" → 89回贾政想起这些细节
2. **83回政治背景**: 忠顺王府与北静王府党争 → 89回明确宝玉罪名与此相关
3. **判词呼应**: "于国于家无望" → 宝玉意识到自己成为政治牺牲品

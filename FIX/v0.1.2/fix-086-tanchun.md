## 修复内容

### 修改点: 探春登船前增加内心独白

**原文**:
```
且说那日，探春登船远去，众人站在岸边，望着那船渐渐消失在视线中，心中皆是唏嘘。
```

**修改为**:
```
且说那日，众人来到江边，只见那大江滚滚东流，烟波浩渺，一艘官船已在岸边等候多时。

探春站在船头，望着那茫茫江水，心中百感交集。她想起自己的判词："才自精明志自高，生于末世运偏消。清明涕送江边望，千里东风一梦遥。"

"才自精明志自高..."她在心中默念着这一句，嘴角浮起一丝苦笑。

她这一生，确实精明。她管家时雷厉风行，整顿大观园时铁面无私，连凤姐都要让她三分。她也确实有抱负，只可惜...只可惜生在了这末世，生在了这日渐衰败的贾府，更可惜...生为女儿身，又是个庶出。

她想起自己的母亲赵姨娘。那个身份卑微、举止粗俗的女人，是她在这府中最大的耻辱，也是她心中最深的痛。她从小便知道，自己是"庶出"，是不被看重的。她拼命地读书，拼命地管家，拼命地让自己变得优秀，就是为了证明——庶出又如何？她比那些嫡出的小姐们，强上百倍！

可如今，这一切都要结束了。

她想起二姐姐迎春。那个懦弱的女人，任人欺凌，最终落得个被虐待致死的下场。她探春，绝不会像迎春那样！她宁可远嫁他乡，宁可在这乱世中搏一条生路，也绝不要在这府中等死！

"我但凡是个男人，可以出得去，我必早走了，立一番事业，那时自有我一番道理。"这是当年她说过的话，如今想来，竟是一语成谶。

她虽然不是男人，但她依然可以走出去。这远嫁，在外人看来是悲剧，在她探春看来，却是...解脱。

解脱了这庶出的身份，解脱了这衰败的家族，解脱了这"才自精明志自高，生于末世运偏消"的命运。

她望着那江水，望着那未知的远方，眼中闪过一丝坚定。

"才自精明志自高...既生为探春，便当如探春而活。"她轻声道，声音虽轻，却字字清晰。

她转身望向岸边，望向那些来送行的人——宝玉、宝钗、湘云...还有这她曾经生活过的大观园。

"二哥哥，"她轻声道，声音随风飘散，"你要保重..."

说完，她毅然转身，步入船舱。那决绝的背影，仿佛要将这一切都抛在身后。

船夫解开缆绳，官船缓缓驶离岸边，向着那茫茫江水深处驶去。

众人站在岸边，望着那船渐渐消失在视线中，心中皆是唏嘘。
```

## 修复说明

### 判词呼应

**"才自精明志自高，生于末世运偏消"**:
- **才自精明**: 管家才能、整顿大观园（原文基础）
- **志自高**: 宁可远嫁也不等死的决断（新增）
- **生于末世**: 贾府衰败的无奈（原文基础）
- **运偏消**: 庶出身份的宿命（新增）

**"清明涕送江边望，千里东风一梦遥"**:
- **江边望**: 饯别场景（原文基础）
- **千里东风**: 远嫁他乡的决绝（新增心理描写强化）

### 人物对比

| 人物 | 结局 | 探春的对比 |

3. **决断宣言**: 宁远嫁不等死的主动选择

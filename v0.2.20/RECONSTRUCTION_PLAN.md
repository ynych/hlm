# 《癸酉续红》v0.3.0 全面重构计划

**方案**: C - 全面重构  
**时间**: 6-8周  
**目标**: 15万字，85分  
**范围**: 离散期7回重写+结局2回大幅扩充+全部回末诗重写

---

## 重构目标

| 指标 | v0.2.19 | v0.3.0目标 | 变化 |
|------|---------|------------|------|

| 离散期字数 | 0.8万 | **3.5万** | +2.7万 |
| 结局字数 | 0.3万 | **1.5万** | +1.2万 |

## 分阶段计划

### Phase 2: 离散期重构（Week 1-3）

**Week 1: 第95-96回**
- 第95回凤姐狱中：800字 → 3500字
  - 狱中日常细节
  - 与其他囚犯互动
  - 身体恶化过程
  - 巧姐托孤完整过程
  
- 第96回迎春临终：600字 → 3000字
  - 孙绍祖虐待细节（含蓄）
  - 抄经念佛过程
  - 与惜春最后对话
  - 临终弥留之际

**Week 2: 第97-98回**
- 第97回宝钗之死：1000字 → 4000字
  - 产后病程详细（7天过程）
  - 与宝玉每日对话
  - 对贾桂嘱托（3个场景）
  - 整理黛玉诗稿细节
  - 金簪埋雪意象渲染
  
- 第98回宝玉痛失：500字 → 2500字
  - 哭灵场景
  - 回忆闪回
  - 初步醒悟

**Week 3: 第99-101回**
- 第99回贾府崩塌：600字 → 2500字
  - 奴仆离散
  - 贾母病重
  - 宝玉离家
  
- 第100回宝玉流浪：500字 → 3000字
  - 流浪途中的见闻（3个城市）
  - 偶遇旧人（至少2人）
  - 心理转变关键节点
  
- 第101回神签惊兆：400字 → 2000字
  - 散花寺求签详细
  - 与道士对话
  - 通灵玉异象

### Phase 3: 人物补全（Week 4）

**宝玉主动性增强**:
- 第97回：主动守护宝钗（拒绝离开）
- 第100回：主动救助流浪儿
- 第108回：主动选择出家（非被动）

**副册人物扩充**:
- 香菱：学诗场景扩充（第86回增加500字）
- 夏金桂：增加动机背景（第91回增加300字）
- 平儿：照顾巧姐细节（第96回增加400字）

### Phase 4: 结局重构（Week 5-6）

**Week 5: 第102-106回**
- 第102回湘云船妓：扩充至3000字
  - 船上日常生活（3个场景）
  - 与船娘友谊
  - 诗酒风流细节
  - 与宝玉重逢（含蓄）
  
- 第105回查抄扩充：500字 → 2500字
  - 王夫人、邢夫人反应（已完成，再扩充）
  - 贾母临终
  - 贾政崩溃
  
- 第106回凤姐历幻：扩充至2000字
  - 魂魄归金陵意象
  - 与贾琏最后对话（梦中）

**Week 6: 第107-108回**
- 第107回宝玉护玉：扩充至2500字
  - 贾环争夺通灵玉
  - 宝玉以命相护
  - 佳人双护玉细节
  
- 第108回悬崖撒手：扩充至4000字
  - 大观园回望（8个地点详细描写）
  - 通灵玉记忆闪回（12钗一一浮现）
  - 袭人悬崖送别（已完成，再扩充）
  - 警幻仙子引路
  - 出家仪式细节

### Phase 5: 诗词精修（Week 7-8）

**Week 7: 关键回末诗重写**
- 第83回（元春）：宫怨主题，含蓄表达
- 第85回（黛玉）：白绫意象，诗意含蓄
- 第88回（凤姐）：断簪意象，尊严主题
- 第97回（宝钗）：金簪雪埋，含蓄暗示

**Week 8: 其他回末诗精修**
- 剩余24回回末诗统一检查
- 平仄格律核对
- 对仗工整度检查
- 与原著风格对比

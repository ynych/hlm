## 二、双主题架构

### 2.1 悼明挽歌（40%）

**隐喻映射**:
- 贾府 = 明朝
- 贾母 = 崇祯（最后的家长）
- 元春薨 = 崇祯自缢（崩塌开始）
- 抄家 = 李自成入京
- 宝玉流浪 = 遗民漂泊

**末世意象贯穿**:
- 残荷、落叶、枯兰、断簪
- 金丝雀、宫墙、笼中
- 雪、水、火、梦

### 2.2 情字总成空（60%）

**人物关系解构**:
- 宝黛：泪尽而亡 → 白绫挂梅
- 宝钗：停机德 → 金簪雪埋
- 湘云：诗酒风流 → 贫病而逝
- 凤姐：机关算尽 → 狱中托孤
- 迎春：柔弱顺从 → 佛号中超脱

**核心命题**:
> "情"的美好与毁灭  
> "空"的必然与无奈  
> "情"到"空"的转化

## 三、三大人物改造

### 3.1 黛玉之死（第85回）

**判词落实**: 玉带林中挂

**改造要点**:
- 泪尽咳血（情感高潮）
- 白绫挂梅枝（意象落实）
- 三段式告别（宝钗、湘云、宝玉）
- **质本洁来还洁去**

**关键场景**:
```
紫鹃将白绫挂上梅枝...
那白绫在风中飘动...
如一条玉带...
```

### 3.2 宝钗之死（第97回）

**判词落实**: 金簪雪里埋

**改造要点**:
- 7天病程（第1-7日详细描写）
- 每日动作暗示（整理诗稿、贴胸口）
- 回忆闪回（钗黛和解）
- 金锁传给贾桂

**关键场景**:
```
第七日，窗外北风呼啸...
最后一朵残梅飘落...
被雪覆盖...
金簪雪里埋...
```

### 3.3 湘云之逝（第102回）

**判词落实**: 湘江水逝楚云飞

**改造要点**:
- 船妓身份（沦落但不失尊严）
- 用诗与客人周旋（"诗比曲贵"）
- 与船娘对话（豪爽性格）
- 笑着流泪（与宝玉重逢）

**关键场景**:
```
湘云站起身，望着湘江水...
笑着，泪水却流了下来...
湘江水逝...楚云飞...
```

## 七、宝玉主动性强化

### 7.1 主动救流浪儿（第100回）

**场景**:
```
宝玉在第三个城市...
看见一个流浪儿被欺负...
他挺身而出...
"这是我弟弟...你们欺负他...就是欺负我..."

这是他流浪以来...
第一次...主动...保护别人...

那流浪儿望着他...
眼中满是感激...
宝玉心中一动...
原来...主动...是这么痛快...
```

**意义**:
- 从被动承受 → 主动保护
- 从自我中心 → 关心他人
- 为最终出家做铺垫

### 7.2 主动护玉拒贾环（第107回）

**场景**:
```
贾环伸手去拿玉...
宝玉一把按住："环儿，你要做什么？"

"这是林妹妹...宝姐姐...
她们留给我...唯一的念想...
你不能拿..."

"只要我还活着...
这玉...就是我的..."

他望着贾环...
眼中没有怨恨...
只有一种...看透一切的清明...
```

**意义**:
- 从逃避 → 直面
- 从软弱 → 坚定
- 保护精神寄托

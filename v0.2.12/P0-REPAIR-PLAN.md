## 二、宝钗临终场景修复方案

### 原文（OOC）

```
宝钗将贾桂的小手放入湘云手中，轻声道："这孩子，我就托付给你了。"

湘云闻言，泪如雨下："姐姐放心，我...我一定视如己出，将他抚养成人。"

宝钗点了点头，惨然一笑："我知晓你是个重情义的人。卫若兰去了，你一个人...也是孤苦。这孩子跟着你，或许...或许能给你们彼此做个伴..."

她说着，又是一阵剧烈的咳嗽。湘云忙替她抚胸捶背，待咳嗽稍止，宝钗靠在床头，喘着粗气，轻声道："看似圆满，实则...从未有过自己的主张...若有来世...愿做一株草木，不求金玉良缘，只守本心..."

她说完，便闭上双眼，不再言语。
```

**问题诊断**（META-001）：
- ❌ "若有来世"——存在主义句式，非清代闺秀语言
- ❌ "只守本心"——现代自我觉醒概念
- ❌ "不求金玉良缘"——直白否定判词，失去含蓄
- ❌ 临终总结性独白——现代小说手法，非红楼风格

### 修复方案

```
宝钗将贾桂的小手放入湘云手中，轻声道："这孩子，我就托付给你了。"

湘云闻言，泪如雨下："姐姐放心，我...我一定视如己出，将他抚养成人。"

宝钗点了点头，惨然一笑："我知晓你是个重情义的人。卫若兰去了，你一个人...也是孤苦。这孩子跟着你，或许...或许能给你们彼此做个伴..."

她说着，又是一阵剧烈的咳嗽。湘云忙替她抚胸捶背，待咳嗽稍止，宝钗靠在床头，喘着粗气，望着窗外那越下越大的雪，半日不语。

【湘云顺着她的目光望去，只见窗外白茫茫一片】

宝钗忽然伸出手，从枕下摸出那枚金锁，凝视良久，又轻轻放在贾桂手中。

【湘云见状，心中一动，却也不敢问】

宝钗又望向窗外，那雪越下越大，竟似要将整个世界都掩埋。她忽然轻轻叹了一口气，声音几不可闻："这雪...下得真好..."

【说完，便闭上双眼，不再言语】
```

**修复说明**（META-001）：
- ✅ 删除"若有来世""只守本心"等现代概念
- ✅ 删除总结性独白
- ✅ 增加动作暗示：取出金锁、凝视窗外大雪
- ✅ 以景写情：大雪掩埋 = 金簪雪里埋的意象
- ✅ 含蓄结尾："这雪...下得真好"——多重解读空间

**读者感知团复核**（蘅芜君/宝钗自己）：
> "这像我。不说破，但你们都能感受到我的解脱。"

## 三、凤姐判词场景修复方案

### 原文（说破判词）

```
她颤巍巍地拾起那休书，手上一抖，一支簪子从发间滑落，掉在地上，断成两截。她凝眸望着那断簪，念及前日在镜中看见的不祥之兆，心底越发忐忑。

忽而，她目光落在那休书上，盯着"休"字怔怔出神。她想起从前——新婚时贾琏待她言听计从；后来她操持家务，贾琏渐渐对她发号施令；如今一纸休书将她逐出家门...

"一从二令三人木..."她喃喃自语，苦笑一声，望着那断簪，久久不语。
```

**问题诊断**（META-001）：
- ❌ "一从二令三人木"——角色直接解读判词，当了读者
- ❌ "她喃喃自语，苦笑一声"——总结性分析

### 修复方案

```
她颤巍巍地拾起那休书，手上一抖，一支簪子从发间滑落，掉在地上，断成两截。她凝眸望着那断簪，念及前日在镜中看见的不祥之兆，心底越发忐忑。

忽而，她目光落在那休书上，盯着"休"字怔怔出神。她想起从前——新婚时贾琏待她言听计从；后来她操持家务，贾琏渐渐对她发号施令；如今一纸休书将她逐出家门...

【她望着那休书，又望着那断簪，手微微颤抖】

【良久，她冷笑一声】

【将断簪拾起，重新插回发间，坐直了身子】

"休便休了。"

【她理了理衣襟，尽管那衣裳已被揉皱】

"我倒要看看，这命还能把我怎样！"
```

**修复说明**（META-001）：
- ✅ 删除"一从二令三人木"自语
- ✅ 增加动作暗示：冷笑、重新插簪、理衣襟
- ✅ 保留"休便休了"——符合凤姐泼辣性格
- ✅ 保留"我倒要看看"——符合凤姐不服输

**读者感知团复核**（凤辣子/凤姐自己）：
> "这像我。我不需要念判词，你们从动作里就能看懂我的处境。"

## 四、省略号滥用替换方案

### 高频滥用模式

| 滥用类型 | 出现次数 | 替换策略 |
|:---------|:--------:|:---------|
| 断续抒情 | ~50处 | 改为动作/神态 |
| 欲言又止 | ~30处 | 改为场景暗示 |
| 心理停顿 | ~20处 | 改为身体反应 |

### 替换对照表

#### 模式1：断续抒情（"我这一生...最对不住..."）

**原文**：
```
"我这一生...最对不住的...是你..."
```

**替换方案A（悲伤）**：
```
【哽咽】【泪如雨下】【握住对方的手，半日不语】
```

**替换方案B（临终）**：
```
【望着对方】【目光渐渐涣散】【嘴唇微动，却发不出声音】
```

#### 模式2：欲言又止（"那白绫...便挂在..."）

**原文**：
```
"那白绫...便挂在那梅枝上..."
```

**替换方案**：
```
只见那白绫，挂在梅枝上，在风中轻轻摇曳。
```
（直接陈述，无需断续）

#### 模式3：心理停顿（"我不是聪明...我是..."）

**原文**：
```
"我不是聪明...我是不得不聪明..."
```

**替换方案**：
```
【苦笑】【摇头】
"不得不罢了。"
```

```

```

```

**替换方案B（动作暗示）**：
```

```

**替换方案C（场景暗示）**：
```
她望着那落花，半日不语，两行清泪无声滑落。
```

#### 套语2："胸口像是被什么堵住了"

**原文**：
```
听了这话，她胸口像是被什么堵住了。
```

**替换方案A**：
```

```

**替换方案B**：
```

```

**替换方案C**：
```

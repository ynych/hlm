## 二、评审工作流程

### Step 1: 系统位点定位

**问题**: 该角色在封建系统中的位置是什么？

```
核心层：元春、凤姐（权力中心）
依附层：黛玉、宝钗（未来权力候选人）
边缘层：湘云、岫烟（旁支/贫寒）
底层：香菱、平儿（丫鬟/妾侍）
```

**检查**: 角色的行为是否符合她的系统位点？
- 凤姐作为核心层，是否有权力运作？
- 黛玉作为依附层，是否有婚姻博弈？
- 香菱作为底层，是否有被动生存？

### Step 2: 五阶段演进检查

**问题**: 该角色在五阶段中采取了什么策略？

| 阶段 | 检查点 |
|------|--------|
| Stage 1 | 是否意识到系统熵增？采取了什么预防策略？ |
| Stage 2 | 面对核心崩溃的反应？崩溃/适应/超越？ |
| Stage 3 | 独立生存的能力？是否建立了新支持网络？ |
| Stage 4 | 身份转换的能力？精神内核是否保持？ |
| Stage 5 | 是否获得了超越性意义？ |

### Step 3: 六大原理映射

**问题**: 该角色的行为是否符合六大原理？

```markdown
□ 熵增对抗：她如何对抗系统混乱？
  - 例子：探春改革、黛玉诗社、妙玉隔离
  
□ 稀缺争夺：她争夺什么稀缺资源？
  - 例子：凤姐争夺权力、宝钗争夺婚姻价值
  
□ 复制成本：她的核心资产是否难以复制？
  - 例子：黛玉诗才+情感连接、凤姐治家才能
  
□ 网络位置：她在协作网络中的位置？
  - 例子：宝钗中心节点、平儿桥梁节点
  
□ 生存优先级：她在哪个层次？
  - Level 5: 黛玉、妙玉、惜春（精神超越）
  - Level 4: 凤姐、湘云、尤三姐（尊严维持）
  - Level 3: 宝钗、李纨（后代延续）
  - Level 2: 香菱、平儿（基本生存）
  - Level 1: 迎春、尤二姐（被动承受）
  
□ 确定性来源：她用什么对抗未知？
  - 例子：宝钗人设、黛玉诗、李纨教育、惜春宗教
```

### Step 4: 五大手段应对检查

**问题**: 该角色如何应对五大手段？

```markdown
□ 制造稀缺：
  - 她是制造者（凤姐）还是接受者（尤二姐）？
  - 她是否成功塑造了自己的稀缺性（宝钗）？
  
□ 思想驯化：
  - 她是内化者（李纨）还是反抗者（黛玉）？
  - 驯化/反抗是否符合她的系统位点？
  
□ 阶层流动：
  - 她是否尝试流动（袭人）？
  - 流动通道是否通畅？
  
□ 分化瓦解：
  - 她是否被分化（钗黛对立）？
  - 她是否成功对抗分化（钗黛和解）？
  
□ 暴力兜底：
  - 她是否遭遇暴力（尤二姐被虐）？
  - 她的反应是什么（忍受/反抗/退出）？
```

## 三、评分标准调整

### 3.1 第一性原理加分项

| 原理表现 | 加分 | 例子 |
|----------|------|------|
| 积极反熵增 | +3分 | 探春改革、黛玉诗社 |
| 拥有稀缺资产 | +3分 | 凤姐权力、黛玉诗才 |
| 高复制成本 | +3分 | 妙玉洁癖、湘云豪爽 |
| 网络核心节点 | +3分 | 宝钗信息枢纽 |
| 高生存层次 | +3分 | 黛玉精神超越 |
| 可靠确定性 | +3分 | 李纨教育确定性 |

### 3.2 第一性原理减分项

| 原理表现 | 减分 | 例子 |
|----------|------|------|
| 完全被动承受 | -3分 | 迎春无反抗 |
| 缺乏稀缺资产 | -3分 | 迎春无才能/无性格 |
| 低复制成本 | -3分 | 秋桐可替代 |
| 网络边缘孤立 | -3分 | 迎春孤立 |
| 低生存层次 | -3分 | 尤二姐被动承受 |
| 无确定性来源 | -3分 | 元春被困深宫 |

## 四、专家评审团规范

### 4.1 每位专家的评审视角

**正册12钗**应从以下视角评审：

| 专家 | 核心视角 | 第一性原理焦点 |
|------|----------|----------------|
| 黛玉 | 诗意反熵增 | 精神超越如何对抗系统混乱？ |
| 宝钗 | 理性秩序维持 | 人设确定性如何维持秩序？ |
| 凤姐 | 权力博弈 | 稀缺资源如何争夺与失去？ |
| 湘云 | 豪爽乐观 | 性格如何作为确定性来源？ |
| 探春 | 制度改革 | 反熵增努力的效果？ |
| 迎春 | 被动承受 | 缺乏任何原理的反面教材 |
| 惜春 | 主动退出 | 用退出对抗熵增？ |
| 可卿 | 天命先知 | 超越性视角的价值？ |
| 李纨 | 教育投资 | 确定性投资的回报？ |
| 妙玉 | 极端隔离 | 物理隔离能否对抗熵增？ |
| 巧姐 | 命运转折 | 系统位点下降后的适应？ |
| 元春 | 权力顶峰 | 核心层也无法逃避熵增？ |

**副册12钗**应从以下视角评审：

| 专家 | 核心视角 | 第一性原理焦点 |
|------|----------|----------------|
| 香菱 | 精神避难 | 底层如何寻找确定性？ |
| 平儿 | 忠诚套利 | 情义是否成为稀缺资产？ |
| 尤三姐 | 决绝退出 | 用死亡获得确定性？ |
| 尤二姐 | 被动承受 | 缺乏所有资源的悲剧 |
| ... | ... | ... |

### 4.2 交叉评审重点

**正册vs副册对照时的第一性原理检查**：

```markdown
□ 香菱vs黛玉：
  - 都是反熵增（诗），但位点不同
  - 黛玉：核心层的精神超越
  - 香菱：底层的精神避难
  
□ 凤姐vs平儿：
  - 都是网络节点，但位置不同
  - 凤姐：权力中心
  - 平儿：桥梁节点
  
□ 妙玉vs惜春：
  - 都是退出系统，但方式不同
  - 妙玉：被迫沦落（隔离失败）
  - 惜春：主动出家（成功退出）
```

## 六、使用示例

### 示例1: 评审黛玉

**传统评审**: 人物完整、情感动人、OOC已修正

**第一性原理评审**:
```
系统位点：依附层（贾母外孙女，父母双亡）✅
五阶段策略：
  - Stage1: 与宝钗和解（网络建设）✅
  - Stage2: 泪尽而亡（拒绝苟活）✅
  - Stage3-5: 精神延续 ✅
六大原理：
  - 熵增对抗：诗社秩序 ✅
  - 稀缺：情感专一 ✅
  - 复制成本：诗才+病弱 ✅
  - 网络位置：情感核心 ✅
  - 生存优先级：Level 5 ✅
  - 确定性：死亡+诗 ✅
五大手段：
  - 思想驯化：反抗者 ✅
  - 分化瓦解：成功对抗（钗黛和解）✅
结论：原理层面表现优秀，+3分
```

### 示例2: 评审迎春

**传统评审**: 性格懦弱、悲剧动人

**第一性原理评审**:
```
系统位点：边缘层 ✅
五阶段策略：
  - 全程被动承受 ⚠️
六大原理：
  - 熵增对抗：无 ⚠️
  - 稀缺：无 ⚠️
  - 复制成本：无 ⚠️
  - 网络位置：边缘孤立 ⚠️
  - 生存优先级：Level 1 ⚠️
  - 确定性：无 ⚠️
结论：原理层面薄弱，-3分
改进建议：即使懦弱，也应有某种确定性来源（如宗教、诗）
```

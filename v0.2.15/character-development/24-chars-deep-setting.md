## 正册十二钗深度设定

### 1. 林黛玉 - 潇湘妃子

**核心内核**: "质本洁来还洁去"的洁癖，既是身体的，更是精神的

**人物弧光**:
- **起点** (81回): 病弱敏感，但仍有生的渴望
- **转折** (82回): 与宝钗和解，放下执念
- **加速** (84-85回): 听闻婚讯，泪尽加速
- **终点** (85回): 泪尽而亡，洁净离世

**情感层次** (Week 1-2重点):
1. **对宝玉**: 不是占有，是"你放心"的成全
2. **对宝钗**: 从敌视到理解，最终和解
3. **对湘云**: 知己之情的惺惺相惜
4. **对自身**: 从自怜到释然，"强于污淖陷渠沟"

**关键意象**: 落花、流水、竹影、白绫

**临终独白草稿** (Week 1-2使用):
```
"我这一生，哭尽了眼泪，还清了债...
如今要去那干净的地方了...
宝玉，你放心，我不会怨你...
我只怨这命，怨这世道...
但我也谢谢这命，让我遇见你...
"
```

### 2. 薛宝钗 - 蘅芜君

**核心内核**: "停机德"的悖论——越贤良，越压抑

**人物弧光**:
- **起点** (81回): 藏愚守拙，理性克制
- **转折** (82回): 探望黛玉，真情流露
- **深化** (84回): 主动询问婚事，内心矛盾
- **终点** (97回): 病逝，临终"释放"

**情感层次**:
1. **对黛玉**: 从竞争者到知己
2. **对宝玉**: 金玉良缘，却非两情相悦
3. **对自身**: "活得太累了"的控诉

**关键意象**: 金锁、冷香丸、雪

**临终独白草稿**:
```
"我这辈子，学会了懂事，学会了藏拙...
可我真正想要的...
从来没有人问过我...
若有来世...
我不做宝钗...
只做自己...
"
```

---

### 3. 王熙凤 - 凤辣子

**核心内核**: "女子无才便是德"的反面——有才又如何？

**人物弧光**:
- **起点** (81回): 权倾贾府，精明强干
- **转折** (88回): 被休，尊严扫地
- **狱中** (95回): 反思女子身份
- **终点** (95回): 病死，机关算尽

**情感层次**:
1. **对贾琏**: 从爱到恨，再到释然
2. **对巧姐**: 母爱的牵挂
3. **对自身**: "若我是男子..."的慨叹

**关键意象**: 断簪、算盘、牢房

**狱中反思草稿**:
```
"我这一生，机关算尽...
算来算去，算不过一个'女'字...
探春说，她若是个男人，必早走了...
我又何尝不是？
可我走不了...
我只能在这牢里...
等死...
"
```

### 4. 史湘云 - 枕霞旧友

**核心内核**: "英豪阔大宽宏量"——看清了还热爱

**人物弧光**:
- **起点** (81回): 豪爽豁达，诗酒风流
- **转折** (86回): 定亲，以为找到归宿
- **沦落** (96回): 卫若兰死，流落风尘
- **终点** (102回): 病逝，笑着离开

**情感层次**:
1. **对宝玉**: "爱哥哥"的纯真情感
2. **对卫若兰**: 相敬如宾，却非深爱
3. **对自身**: "从未将儿女私情略萦心上"

**关键意象**: 金麒麟、芍药、酒

**临终独白草稿**:
```
"我这一生，虽苦，却也痛快...
海棠诗社、醉卧芍药、与林姐姐斗诗...
那些日子，比什么都值！
我不怨...
我只愿...你们都要好好的...
"
```

## 副册十二钗深度设定

### 13. 香菱 - 学诗妾侍

**核心内核**: "平生遭际实堪伤"——在绝望中寻找光

**人物弧光**:
- **起点**: 被拐卖，沦为丫鬟
- **转折** (82回): 向黛玉学诗
- **深化** (86回): 诗成被夏金桂撕毁
- **终点**: 随宝钗生活，诗是唯一慰藉

**情感层次**:
1. **对黛玉**: 师徒情深，唯一理解她的人
2. **对夏金桂**: 恐惧与隐忍
3. **对诗**: 绝望中的光

**关键意象**: 诗稿、荷花、薛家

**学诗场景草稿**:
```
"姑娘，我想学诗...
薛大爷总说我不懂这些...
可我...我真的很想学...
这是我...唯一能自己做主的事...
"
```

---

### 14. 平儿 - 忠心贤妾

**核心内核**: "义"——主仆之义超越夫妻之情

**人物弧光**:
- **起点**: 凤姐陪嫁，贾琏通房
- **转折** (88回): 凤姐被休，仍不离不弃
- **深化** (95回): 凤姐托孤
- **终点** (100回): 照顾巧姐一生

**情感层次**:
1. **对凤姐**: 忠心耿耿，主仆情深
2. **对贾琏**: 复杂情感，有名无实
3. **对巧姐**: 母爱般的守护

**关键意象**: 钥匙、账本、巧姐

**托孤场景草稿**:
```
"奶奶放心，巧姐就是我的命...
我定会将她抚养成人...
这辈子，我只认您一个主子...
巧姐，我会照顾好的...
"
```

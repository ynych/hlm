# Redream项目AI评审规则

**版本**: v1  
**日期**: 2026-03-16  
**适用范围**: 第81-108回初稿评审

---

## 一、评审维度（5大维度）

### 1. 文风仿真度（权重20%）
**标准**: 是否接近曹雪芹前80回风格

**评分细则**:
| 分值 | 标准 |
|------|------|
| 9-10 | 极像曹雪芹，白描、对话、意境俱佳 |
| 7-8 | 较像，偶现现代词汇或句式 |
| 5-6 | 一般，有明显现代痕迹 |
| 3-4 | 较差，大量现代语言 |
| 1-2 | 完全不像 |

**检查点**:
- [ ] 使用"呢、罢、么"等红楼语气词
- [ ] 短段落为主（避免长段落）
- [ ] 情景交融（以景衬情）
- [ ] 避免现代词汇（"其实""肯定"等）
- [ ] 使用"这会子、才刚、越发"等红楼常用词

---

|------|------|

| 7-8 | 基本符合，偶有小偏差 |
| 5-6 | 部分OOC，人物动机不明确 |
| 3-4 | 严重OOC，人物性格矛盾 |
| 1-2 | 完全崩坏 |

**关键人物检查清单**:

**林黛玉**:

**薛宝钗**:

- [ ] 劝宝玉读书（仕途经济）
- [ ] 不黑化，不陷害黛玉
- [ ] 藏愚守拙，不张扬

**贾宝玉**:
- [ ] 痴情黛玉，专一
- [ ] 厌恶仕途经济
- [ ] 对丫鬟温柔，不辱骂

**王熙凤**:

### 4. 情节合理性（权重15%）
**标准**: 情节是否符合逻辑、历史背景

**评分细则**:
| 分值 | 标准 |
|------|------|
| 9-10 | 情节合理，历史细节准确 |
| 7-8 | 基本合理，偶有小瑕疵 |

**检查点**:

### 5. 语言质量（权重15%）
**标准**: 文字是否精炼、有美感

**评分细则**:
| 分值 | 标准 |
|------|------|
| 9-10 | 文字精炼，诗意盎然 |
| 7-8 | 较好，偶现冗余 |
| 5-6 | 一般，有口水化表达 |
| 3-4 | 较差，大量冗余 |
| 1-2 | 完全口语化 |

**检查点**:
- [ ] 无重复用词（同一页不重复同一成语）
- [ ] 描写有画面感
- [ ] 对话符合身份
- [ ] 回末诗贴切工整

## 二、评审流程

### 步骤1：阅读初稿
- 通读全文，标记问题点
- 记录亮点

### 步骤2：逐项评分
- 按5维度打分（1-10分）
- 计算综合得分（加权平均）

### 步骤3：撰写评语
**必须包含**:
1. **亮点**（至少2点）
2. **关键问题**（如有，优先修正）
3. **改进建议**（具体可操作）

### 步骤4：问题分级
- **P0（致命）**: 必须修正（如人物OOC、时间线矛盾）
- **P1（重要）**: 建议修正（如用词不当、细节遗漏）
- **P2（优化）**: 可选（如语言精炼）

---

## 三、评分等级

| 综合得分 | 等级 | 处理方式 |
|----------|------|----------|
| 8.0-10 | A | 优秀，可直接进入修订阶段 |
| 6.5-7.9 | B | 良好，需小幅修改 |
| 5.0-6.4 | C | 合格，需中度修改 |
| 3.0-4.9 | D | 不合格，需大幅修改或重写 |
| <3.0 | F | 完全不合格，必须重写 |

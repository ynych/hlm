## 核心机制

### 预测-实现对比
每回开发流程：
```
前N-1回 → MiroFish预测 → 实际实现 → 对比分析 → 优化下回
```

### 预测输入
- **第81回**: 前80回《红楼梦》原文
- **第82回**: 前81回（80回原文 + 第81回续作）
- **第N回**: 前N-1回（累积续作）

### 预测内容
1. 剧情走向
2. 人物命运
3. 场景设计
4. 判词呼应
5. 情感基调

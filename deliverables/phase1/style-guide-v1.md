## 一、词汇层面

### 1.1 语气词（必须熟练运用）

| 词汇 | 用法 | 示例 | 频率 |
|------|------|------|------|
| 罢 | 句末，表建议/妥协/终结 | "去罢" "罢了" "不提罢了" | 极高 |
| 罢哟 | 娇嗔、撒娇语气 | "好没意思的罢哟" | 中 |
| 罢咧 | 无所谓、作罢 | "丢了就丢罢咧" | 中 |
| 罢呀 | 催促或无奈 | "快走罢呀" | 低 |
| 呢 | 疑问/陈述语气 | "在哪里呢" "好呢" | 极高 |
| 么 | 疑问语气 | "做什么呢" "怎么了呢" | 高 |
| 罢了 | 算了、而已 | "如此罢了" | 高 |
| 就是了 | 肯定、确认 | "如此就是了" | 中 |

**使用规则**：
- 疑问句多用"呢"、"么"
- 感叹/妥协多用"罢"
- 女性角色可用"罢哟"显娇态
- 仆人对话少用语气词，显得恭谨

### 1.2 时间词

| 词汇 | 含义 | 示例 | 使用场景 |
|------|------|------|----------|
| 才刚 | 刚刚、刚才 | "才刚老太太还问起" | 过去不久 |
| 这会子 | 这时候 | "这会子才回来" | 当前时刻 |
| 那阵子 | 那时候 | "那阵子还好好的" | 过去某时 |
| 半日 | 好一会儿 | "等了半日" | 一段时间 |
| 好半日 | 很久 | "呆了好半日" | 强调时间长 |
| 一时 | 一会儿 | "一时说不出话来" | 短暂时间 |
| 半晌 | 半天 | "半晌无话" | 沉默/思考 |
| 顷刻 | 片刻 | "顷刻间散了" | 极短时间 |
| 良久 | 很久 | "良久方散" | 书面化 |
| 向晚 | 傍晚 | "向晚时分" | 特定时辰 |

**使用规则**：
- 对话中多用"才刚""这会子"
- 叙述中可用"半日""半晌"控制节奏
- 避免"刚才""现在"等现代时间词

| 词汇 | 用法 | 示例 | 语气 |

## 二、句式层面

### 2.1 典型句式模板

#### 模板A：短句为主，间杂长句
```
那人听了，笑道："原来如此。我说呢，怎么这等面善。"
```
**特征**：短促有力，模拟口语节奏

#### 模板B：倒装句式
```
"往哪里去了这半日？"（而非"这半日你往哪里去了"）
"可大安了？"（而非"身体可好？"）
"做什么呢你？"（而非"你在做什么？"）
```
**特征**：将时间/宾语提前，显口语化

#### 模板C：重复与排比
```
"亏了，亏了！若不是你来，我几乎忘了。"
"罢了，罢了！再也休提。"
"好，好，好！"
```
**特征**：情感递进，强调语气

#### 模板D：省略主语
```
听了，心下疑惑，不知是何缘故。（省略"宝玉"）
笑道："来了。"（省略主语）
```
**特征**：承前省略，行文简洁

#### 模板E：转折先行
```
"只是......"（最常用，代替"但是"）
"却......"
"倒......"（表让步）
"偏......"（表不满）
"偏生......"
```
**特征**：转折词前置，语气柔和

### 2.2 对话模式

#### 模式A：不直接回答，先绕弯子
```
宝玉问："妹妹可大安了？"
黛玉不答，只道："你从哪里来？"
```
**适用**：黛玉、宝钗等聪明人物

#### 模式B：话中有话，意在言外
```
宝钗笑道："我来了这么些年，留神看起来，二嫂子凭她怎么巧，巧不过老太太去。"
（表面夸贾母，实则刺凤姐）
```
**适用**：贾府女眷间对话

#### 模式C：打断与抢白
```
凤姐道："你——"
话未说完，平儿进来道："奶奶，姨太太那边打发人来了。"
```
**适用**：表现场面紧张/忙乱

#### 模式D：先声夺人
```
只听外面一阵脚步响，帘子一掀，凤姐进来笑道："我来迟了，不曾迎接远客！"
```
**适用**：人物出场，先闻其声

#### 模式E：借物传情
```
"这花原是我才摘的，妹妹留着顽罢。"
（不说"我喜欢你"，而说"给你花"）
```
**适用**：宝黛情感交流

### 2.3 叙述句式

#### 模式A：动作连写
```
宝玉听了，心下疑惑，不知是何缘故。欲要询问，又怕唐突，只得闷闷的回来。
```
**特征**：一连串动作，暗示心理

#### 模式B：感官描写
```
只见贾宝玉穿着家常衣服，趿着鞋，在那桃花树底下呆呆的站着。一阵风过，吹下许多花瓣来，落得满身满鬓。
```
**特征**：视觉为主，动静结合

#### 模式C：侧面烘托
```
众人听说，都笑道："原来如此。"（不直接写"宝玉尴尬"，而写众人反应）
```
**特征**：通过他人反应写主角

## 三、叙事层面

### 3.1 视角控制

#### 全知视角，但假装受限
- 作者知道一切，但故意留白
- 通过人物的眼睛看，不直接下判断

**示例**：
```
宝玉听了，心下疑惑，不知是何缘故。欲要询问，又怕唐突，只得闷闷的回来。
```
（不直接说"宝玉不知道原因"，而是写他的动作与心理）

#### 视角切换规则
1. **主视角**：宝玉（使用最频繁）
2. **次要视角**：黛玉、宝钗、凤姐（根据需要切换）
3. **群像视角**：众人、大家（表现集体反应）
4. **限制**：同一自然段内不频繁切换

### 3.2 节奏控制

#### 大事缓写，小事细描
- 元妃省亲：宏大场面，寥寥数笔
- 黛玉葬花：一人一事，浓墨重彩

#### 动静结合
- 激烈冲突后必有闲笔
- 热闹场面中插入冷观察

**示例**：
```
（热闹场面）
当下众人说说笑笑，十分热闹。 
（插入冷观察）
只黛玉一人坐在角落里，冷眼看着，也不言语。
```

#### 节奏标记词
| 快节奏 | 慢节奏 |
|--------|--------|
| 当下、随即、连忙 | 半日、半晌、良久 |
| 一面...一面 | 且说、却说 |
| 不多时 | 这且不提 |

### 3.3 环境描写

#### 以人带景，情景交融
```
只见贾宝玉穿着家常衣服，趿着鞋，在那桃花树底下呆呆的站着。一阵风过，吹下许多花瓣来，落得满身满鬓。
```
**规则**：写景必有人的动作/视角

#### 善用时辰与天气
| 时辰 | 氛围 |
|------|------|
| 春日 | 生机、希望 |
| 夏夜 | 烦闷、躁动 |
| 秋雨 | 凄凉、感伤 |
| 冬雪 | 清冷、静谧 |
| 黄昏 | 惆怅、离别 |
| 夜半 | 秘密、私语 |
| 黎明 | 希望、开始 |

### 4.4 《红楼梦》诗词风格要点

#### 婉约为主，不直露
```
花魂默默无情绪，鸟梦痴痴何处惊。
（林黛玉《葬花吟》风格——以物写情）
```

#### 用典含蓄
- 不直接说"孤独" → "埋香冢飞燕泣残红"
- 不直接说"富贵无常" → "纵是齐眉举案，到底意难平"
- 不直接说"才华被埋没" → "可叹停机德，堪怜咏絮才"

#### 意象系统
| 意象 | 含义 | 示例 |
|------|------|------|
| 花/落花 | 青春易逝 | "花谢花飞飞满天" |
| 泪/红泪 | 悲伤 | "滴不尽相思血泪" |
| 月/冷月 | 孤独 | "冷月葬花魂" |
| 梦/幽梦 | 虚幻 | "幽梦冷随红袖笛" |
| 柳/垂柳 | 离别 | "柳丝榆荚自芳菲" |
| 秋风/秋雨 | 凄凉 | "秋风惨淡秋草黄" |

### 5.2 慎用词（尽量不用）

| 词汇 | 说明 |
|------|------|
| 已经 | 可用"已""都"替代 |
| 仍然 | 可用"仍""还是"替代 |
| 突然 | 可用"忽""忽然""蓦地"替代 |
| 终于 | 可用"终究""到底"替代 |
| 其实 | 可用"原来"替代 |
| 当然 | 可用"自然""理所当然"替代 |
| 确实 | 可用"委实""实在"替代 |

## 六、仿写示例

### 示例1：改写现代句子
**原文**："宝玉听了，心里很不舒服，但又不能说什么，只能默默离开。"

**仿写**："宝玉听了，心下不快，只是不好发作，搭讪着走了出来。"

**要点**：
- "心里很不舒服" → "心下不快"
- "但" → "只是"
- "默默离开" → "搭讪着走了出来"（通过动作暗示心理）

---

### 示例2：改写现代句子
**原文**："黛玉等了很久，宝玉还是没有来，她很失望。"

**仿写**："黛玉等了半日，不见宝玉来，越觉没趣。"

**要点**：
- "等了很久" → "等了半日"
- "还是没有来" → "不见...来"
- "很失望" → "越觉没趣"

---

### 示例3：改写现代句子
**原文**："其实，我早就知道这件事了。"

**仿写**："原来，我早知道了罢咧。"

**要点**：
- "其实" → "原来"
- "就" → 删除，用"早"
- 句末加"罢咧"显口语化

---

### 示例4：改写现代对话
**原文**："你刚才去哪里了？怎么这么久才回来？"

**仿写**："才刚往哪里去了这半日？这会子才回来？"

**要点**：
- "刚才" → "才刚"
- "这么久" → "这半日"（倒装）
- "怎么...才" → "这会子才"

---

### 示例5：改写心理描写
**原文**："宝玉看着黛玉，心里想着她今天真漂亮。"

**仿写**："宝玉看着黛玉，只觉今日分外的光彩。"

**要点**：
- 删除"心里想着"，改为客观描写
- "真漂亮" → "分外的光彩"
- 通过宝玉视角侧面写黛玉

---

### 示例6：改写对话（绕弯子）
**原文**：
```
宝钗问："你身体好些了吗？"
黛玉答："好多了，谢谢姐姐关心。"
```

**仿写**：
```
宝钗掀帘子进来，笑道："好些了没有？我瞧瞧。"
黛玉欠身道："劳姐姐记挂着，才好些呢。"
宝钗坐在床沿上，拉着她的手道："我瞧你脸上还是不好。这病，急不得的。"
黛玉叹道："我也知道，只是心里烦得很。"
```

**要点**：
- 增加动作描写（掀帘子、欠身、拉着手）
- 不直接回答，先道谢
- "心里烦"代替"心情不好"

---

### 示例7：改写叙述（动作连写）
**原文**："宝玉听到这个消息，很生气，但他忍住了。"

**仿写**："宝玉听了，登时变了颜色，强忍着没言语，只把手中扇子攥得紧了。"

**要点**：
- "听到" → "听了"
- "很生气" → "变了颜色"（表情暗示）
- "忍住了" → "强忍着没言语" + 动作细节

---

### 示例8：改写环境描写
**原文**："春天的晚上，院子里很安静，黛玉一个人在散步。"

**仿写**："春夜，月华如水。黛玉独自在院中闲步，只听得虫声唧唧，花影婆娑。"

**要点**：
- "春天的晚上" → "春夜"（简洁）
- "很安静" → "虫声唧唧"（以声写静）
- 增加诗意化描写

---

### 示例9：改写仆人对话
**原文**：
```
小丫鬟说："宝玉来了。"
袭人答："知道了，让他进来。"
```

**仿写**：
```
小丫鬟报道："宝二爷来了。"
袭人忙道："快请进来罢。"
```

**要点**：
- "说" → "报道"（仆人用语）
- "宝玉" → "宝二爷"（正式称呼）
- "知道了" → "快请"（恭敬）
- 句末加"罢"

### 示例10：仿写七绝（黛玉葬花风格）
**主题**：落花

**仿写**：
```
残红委地不成春，泪染鲛绡渍酒痕。
莫问东君归去处，一帘风雨掩重门。
```

**格律检查**：
- 平起首句入韵（平水韵"十一真"部）
- 春（平）、痕（平）、门（平）——押韵正确
- 平仄：平平仄仄仄平平（韵）
- 仄仄平平仄仄平（韵）
- 仄仄平平平仄仄
- 平平仄仄仄平平（韵）

**风格要点**：
- "残红""鲛绡""东君"——典雅意象
- "泪染""莫问"——婉约抒情
- 不直说"悲伤"，而说"风雨掩重门"

### 示例11：仿写对话（话中有话）
**场景**：宝钗探望生病的黛玉，表面关心，实则炫耀

**仿写**：
```
宝钗进来，见黛玉歪在床上，便坐在床边笑道："几日不见，妹妹越发清减了。我那里有人参，明儿让人送些来。"
黛玉淡淡道："姐姐费心了。只是我这病，人参也救不得。"
宝钗叹道："话虽如此，养好了身子，将来......"说到这里，故意顿住，只是笑。
黛玉心中明白，偏问道："将来什么？"
宝钗忙道："没什么，我是说，将来还要做针线呢。"
```

**要点**：
- "越发"——程度副词
- "明儿"——时间词
- "故意顿住"——叙事技巧
- 话中有话，不点破

### 示例12：仿写群像场景
**场景**：贾府夜宴，凤姐劝酒

**仿写**：
```
一时摆上酒来，凤姐亲自斟了一杯，捧到贾母面前，笑道："老太太赏个脸，喝了这杯罢。"
贾母笑道："你这猴子，又要灌我。"
凤姐道："谁敢灌老太太？这是孙媳妇的一点孝心。"
贾母听了，果然欢喜，接过来吃了半杯。凤姐又忙去接过来，递给丫鬟，又斟一杯，向王夫人道："太太也请一杯。"
王夫人笑道："我不吃，你敬你姑太太去。"
凤姐听了，便转向薛姨妈，笑道："姨妈是客，这杯务必请赏脸。"
```

**要点**：
- "一时"——时间词
- "亲自""果然"——副词运用
- 动作连贯：斟→捧→接→递→斟
- 对话符合身份

## 七、自检清单

### 写完后逐条检查：

#### 词汇层面
- [ ] 有无现代词汇混入？（心理、压力、情绪、态度、意义、价值、逻辑）
- [ ] 语气词是否正确？（呢、罢、罢哟）
- [ ] 时间词是否准确？（才刚、这会子、半日）
- [ ] 程度副词是否恰当？（好生、十分、越发的）
- [ ] 对话是否符合人物身份？（仆人用"奴才"，丫鬟用"婢子"）

#### 句式层面
- [ ] 是否使用了倒装句式？（"往哪里去了这半日"）
- [ ] 是否长短句交替？
- [ ] 是否用了"只是"代替"但是"？
- [ ] 对话是否绕弯子？（不直接回答）

#### 叙事层面
- [ ] 有没有直接写心理，而不是通过动作表情暗示？
- [ ] 场景转换是否自然？
- [ ] 环境描写是否以人带景？
- [ ] 视角是否保持一致？

#### 诗词层面（如适用）
- [ ] 平仄是否正确？
- [ ] 用韵是否统一？
- [ ] 意象是否典雅？
- [ ] 抒情是否含蓄？

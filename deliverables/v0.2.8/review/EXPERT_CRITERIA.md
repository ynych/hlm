## 三、评分计算方法

### 3.1 维度得分

```
维度得分 = Σ(检查项得分 × 检查项权重)
```

### 3.2 加权总分

```
核心综合 = Σ(维度得分 × 维度权重) / 90%
总评 = 核心综合 + 调整分
```

### 3.3 等级划分

| 等级 | 分数 | 推荐度 |
|:----:|:----:|:------:|
| A++ | 97-100 | ⭐⭐⭐ 强烈推荐 |
| A+ | 95-96 | ⭐⭐ 推荐 |
| A | 90-94 | ⭐ 推荐 |
| B+ | 85-89 | 有条件推荐 |
| B | 80-84 | 需优化 |
| C | <80 | 不建议 |

## 四、问题分级

### 4.1 P0级（致命）

- 判词直接说破
- 人物严重OOC
- 现代概念渗透
- 历史错误

**处理**: 必须修复，否则不通过

### 4.2 P1级（重要）

- 判词铺垫不足
- 句式重复过多
- 心理描写直白
- 过渡生硬

**处理**: 建议修复，影响评分

### 4.3 P2级（优化）

- 语言可更精炼
- 意象可更精准
- 节奏可更优化

**处理**: 可选修复，锦上添花

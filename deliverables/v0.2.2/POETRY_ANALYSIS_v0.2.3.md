## 回末诗清单

### 第81回
```
藕香榭外水潺潺，旺相难占运已残。
残荷听雨声声慢，一叶飘零一叶寒。
```

**格律分析**:
- 体裁: 七言绝句
- 押韵: 残/寒 (an韵，合韵)
- 问题: 
  1. "声"字重复（声声慢）
  2. 第三句"残荷"与首句"残"字重复
  3. 对仗: 无对仗要求（绝句），但"残荷听雨"与"一叶飘零"可成对

**优化建议**:
```
藕香榭外水潺潺，旺相难占运已残。
秋风瑟瑟荷衣老，一叶飘零万叶寒。
```

### 第82回
```
病骨支离泪未干，金兰契友慰心寒。
停机德厚情难断，一任东风带笑看。
```

**格律分析**:
- 体裁: 七言绝句
- 押韵: 干/寒/断/看
  - 干(han)、寒(han): 合韵
  - 断(duan): uan韵，与an韵邻韵
  - 看(kan): an韵
  - 问题: "断"字出韵

**优化建议**:
```
病骨支离泪未干，金兰契友慰心寒。
停机德厚情难了，一任东风带笑看。
```
（"了"字可通韵，或改为"情难尽"）

茫茫白雪埋金簪，千古红颜总若斯。

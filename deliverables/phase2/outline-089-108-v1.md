# 第89-108回 全局架构总纲

**范围**: 第89-108回（共20回）  
**主题**: 贾府覆灭 → 离散飘零 → 悬崖撒手  
**核心剧情线**: 凤姐被捕 → 贾府被抄 → 贾母病逝 → 宝玉入狱 → 出狱流离 → 沦为乞丐 → 悬崖撒手

## 批次3后半：第89-94回（贾府覆灭，6回）

### 第89回 宁荣两府被抄没 贾宝玉蒙冤入狱

**核心剧情**: 贾府正式被抄，财产充公，宝玉以"勾结外官"罪名被关押。

**场景**:
1. **荣禧堂·宣读圣旨** - 锦衣军宣读抄家旨意，贾政被枷，众人崩溃
2. **各房查抄** - 翻箱倒柜，贾母私房、凤姐嫁妆被抄没，奴仆哭喊
3. **宝玉被捕** - 宝钗哭求无果，宝玉被铁链锁住带走，回头望宝钗最后一眼
4. **狱中收监** - 宝玉被关入大牢，与三教九流关在一起，第一次体验人间地狱

**伏笔照应**:
- 秦可卿"忽喇喇似大厦倾"应验
- 宝玉神游太虚幻境时警幻仙子的警告
- 宝钗"焦首朝朝还暮暮"开始应验

---

### 第90回 史太君寿终归地府 薛宝钗狱中探夫君

**核心剧情**: 贾母得知抄家宝玉被捕，急怒攻心去世；宝钗挺着孕肚狱中探望宝玉。

**场景**:
1. **贾母咽气** - 听闻宝玉被捕，一口痰涌上来，望着门外，死不瞑目
2. **白事乱象** - 抄家之际无力厚葬，简单入殓，停灵偏院
3. **宝钗探监** - 买通狱卒，见到蓬头垢面的宝玉，相对无言泪千行
4. **狱中诀别** - 宝钗告诉宝玉"我等你"，宝玉摇头"不必等，我不值得"

**伏笔照应**:
- 贾母"树倒猢狲散"临终应验
- 宝钗吃冷香丸（冷命）的命运开始

---

### 第91回 刘姥姥仗义疏财 薛宝钗雪中送炭

**核心剧情**: 刘姥姥得知贾府遭难，变卖田产筹银营救；宝钗四处奔走求告旧友。

**场景**:
1. **刘姥姥闻讯** - 乡下得到消息，当即决定赴京，变卖自家田产
2. **宝钗求告** - 往日亲友闭门不见，薛姨妈痛哭无能为力
3. **狱中通融** - 刘姥姥银子使开各路关节，宝玉待遇稍改善
4. **探监送物** - 刘姥姥见宝玉，说出"当日受恩，今日报恩"，宝玉痛哭

**伏笔照应**:
- 刘姥姥"偶因济刘氏"判词应验（巧姐故事的前奏）
- 宝玉昔日对刘姥姥的善意得到回报

---

### 第92回 薛宝钗狱中分娩 贾兰高中探花郎

**核心剧情**: 宝钗在狱中生下贾桂（遗腹子）；贾兰科举高中，试图营救家族。

**场景**:
1. **狱中分娩** - 宝钗难产，狱中无医，靠自己咬牙生下儿子
2. **李纨探监** - 带来贾兰高中消息，李纨悲喜交加，宝玉麻木
3. **取名贾桂** - 宝钗给孩子取名"桂"（兰桂齐芳），寄托最后希望
4. **贾兰奔走** - 新科探花试图营救舅舅，官场碰壁，明白无力回天

**伏笔照应**:
- 李纨"晚韶华"判词——儿子高中但家族已亡
- "兰桂齐芳"的讽刺性实现

---

### 第93回 贾政流放宁古塔 贾宝玉获释归家

**核心剧情**: 贾政被判流放宁古塔；宝玉因查无实据获释，回家已物是人非。

**场景**:
1. **判决下来** - 贾赦死罪，贾珍流放，贾政流放宁古塔（生还无望）
2. **宝玉释放** - 宝钗抱着婴儿来接，宝玉神情恍惚，似不认识妻儿
3. **荣府现状** - 已非贾家产业，被官府拍卖，宝钗租住在破院
4. **父子诀别** - 贾政押解前见宝玉最后一面，父子相对无言，贾政叹"孽障"

**伏笔照应**:
- 贾政"训子"场景的终极反讽
- 宝玉"负父望"的彻底实现

---

### 第94回 妙玉被劫终陷淖泥 迎春被虐赴黄粱

**核心剧情**: 妙玉在乱局中被强盗劫走；迎春被孙绍祖虐打致死。

**场景**:
1. **妙玉遭劫** - 贾府败落后妙玉欲投奔他处，途中被劫，无暇白玉遭泥陷
2. **迎春噩耗** - 孙家来人报丧，迎春被虐打致死，死状凄惨
3. **宝玉闻讯** - 又闻噩耗，神情更加恍惚，喃喃"又一个，又一个"
4. **探春消息** - 远嫁海外，得知家中惨变，但无力回天（遥祭）

**伏笔照应**:
- 妙玉"欲洁何曾洁"判词应验
- 迎春"子系中山狼，一载赴黄粱"应验
- 探春"千里东风一梦遥"——已无法回头

## 批次4：第95-108回（离散与结局，14回）

### 第95回 薛宝钗产后染病 贾宝玉卖字为生

**核心剧情**: 宝钗产后虚弱染病；宝玉为生计开始卖字，体验底层艰辛。

**场景**:
1. **宝钗病倒** - 产后失养，一病不起，无钱医治
2. **宝玉卖字** - 在街市摆摊卖字，被地痞欺负，被昔日 acquaintances 认出（尴尬）
3. **贾桂哭闹** - 婴儿饥饿，宝钗无奶，宝玉手足无措
4. **湘云出现** - 史湘云（守寡后沦落）路过认出宝玉，震惊

**伏笔照应**:
- 宝玉"潦倒不通世务"到被迫通世务的转变
- 湘云"湘江水逝楚云飞"开始应验

### 第96回 史湘云守寡投奔 薛宝钗病重托孤

**核心剧情**: 湘云丈夫去世，投奔宝钗；宝钗病重，托孤于湘云。

**场景**:
1. **湘云投奔** - 卫若兰去世，湘云守寡，家道中落，投奔昔日姐妹
2. **宝钗病情** - 病入膏肓，知自己不久人世，开始安排后事
3. **托孤湘云** - 将贾桂托付给湘云，湘云含泪应下"我把他当亲生的"
4. **夫妻夜话** - 宝钗与宝玉最后一次长谈，说"我不怪你，从来都没怪过"

**伏笔照应**:
- 宝钗"金簪雪里埋"的孤独命运
- 湘云"白首双星"——与宝玉的兄妹情（非爱情）

---

### 第97回 薛宝钗泪尽而逝 贾宝玉彻底崩溃

**核心剧情**: 宝钗病逝，临终前与宝玉和解；宝玉彻底崩溃，精神失常。

**场景**:
1. **宝钗弥留** - 回光返照，望着宝玉和孩子，泪尽（呼应黛玉泪尽）
2. **临终遗言** - "宝玉，照顾好孩子……下辈子……别见了……"（太累）
3. **宝钗咽气** - 宝玉不哭，只是呆坐，抱着孩子喃喃"都走了，都走了"
4. **精神崩溃** - 宝玉开始精神恍惚，时哭时笑，湘云无奈

**伏笔照应**:
- 宝钗"金簪雪里埋"——孤独死去
- 宝玉"空对着山中高士晶莹雪"——对着宝钗遗体

---

### 第98回 贾宝玉遗弃幼子 史湘云流落街头

**核心剧情**: 宝玉精神失常后出走，遗弃贾桂；湘云带着孩子流落街头。

**场景**:
1. **宝玉出走** - 趁湘云不在，宝玉离开，留下字条"我负了你们所有人"
2. **湘云寻夫** - 抱着孩子找宝玉，找到时已精神恍惚的乞丐
3. **宝玉不认** - 宝玉已不认识湘云和孩子，喃喃"林妹妹呢？我要找林妹妹"
4. **湘云绝望** - 无奈带着孩子离开，开始真正的流浪生涯

**伏笔照应**:
- 宝玉"你死了我做和尚去"——失约，未做和尚而是做乞丐
- 湘云"展眼乞丐人皆谤"

### 第99回 巧姐险些遭难 刘姥姥拼死相救

**核心剧情**: 凤姐狱中病逝后，巧姐被卖入烟花巷；刘姥姥拼死相救。

**场景**:
1. **凤姐病逝** - 狱中传来消息，凤姐病死，临终念叨"巧姐，我的儿"
2. **巧姐被卖** - 贾蓉、贾蔷（或王家）将巧姐卖入烟花之地
3. **刘姥姥营救** - 变卖全部家产，赎回巧姐，带回乡下
4. **板儿定亲** - 为巧姐与板儿定下亲事（农家安稳一生）

**伏笔照应**:
- 巧姐"偶因济刘氏，巧得遇恩人"判词应验
- 刘姥姥"忍耻"的终极体现

### 第100回 贾宝玉沦为乞丐 寒冬腊月乞食难

**核心剧情**: 宝玉彻底沦为乞丐，寒冬腊月乞讨度日，受尽苦楚。

**场景**:
1. **乞丐生涯** - 衣不蔽体，食不果腹，在街角缩成一团
2. **昔日对比** - 路过贾府旧址（已被新贵占据），恍如隔世
3. **被人打骂** - 被店家驱赶，被路人嫌弃，被孩童投掷石块
4. **偶遇袭人** - 袭人（已嫁蒋玉菡）路过，认出宝玉，震惊落泪，但无能为力

**伏笔照应**:
- 宝玉"贫穷难耐凄凉"
- 袭人"好歹留着麝月"（未能留下）

### 第101回 袭人暗中周济 蒋玉菡仗义疏财

**核心剧情**: 袭人与蒋玉菡暗中资助宝玉；宝玉拒绝接受，继续流浪。

**场景**:
1. **袭人寻来** - 蒋玉菡夫妇找到宝玉住所（破庙），留下银钱衣物
2. **宝玉拒绝** - 宝玉发现后，将银钱丢弃，"我不要你们的可怜"
3. **袭人哭诉** - "二爷，你就当是可怜可怜我，让我心安"
4. **宝玉离去** - 宝玉默默离开破庙，继续流浪，不知所踪

**伏笔照应**:
- 袭人与宝玉的最后缘分
- 蒋玉菡"堪羡优伶有福"——对比宝玉的"无福"

### 第102回 史湘云病逝街头 贾宝玉雪夜独行

**核心剧情**: 湘云贫病交加，带着贾桂病逝街头；宝玉雪夜独行，更加疯癫。

**场景**:
1. **湘云病倒** - 无钱医治，在破庙中发高烧，贾桂哭喊
2. **托孤路人** - 湘云临终托孤给好心老妇，交代"这是荣国府的骨血"
3. **湘云咽气** - "湘江水逝楚云飞"应验，湘云死在风雪夜
4. **宝玉独行** - 宝玉在雪夜中游荡，不知湘云已死，口中念"云妹妹"

**伏笔照应**:
- 湘云"展眼乞丐人皆谤"
- 贾桂成为真正的孤儿（呼应板儿与巧姐的幸运）

### 第103回 贾宝玉重游故地 白茫茫大地真干净

**核心剧情**: 宝玉在雪中重游大观园旧址，一片废墟，物是人非。

**场景**:

2. **潇湘馆凭吊** - 宝玉找到黛玉住过的屋子，草木萧瑟，泪如雨下
3. **怡红院遗迹** - 昔日怡红院已成瓦砾，宝玉坐于废墟中发呆
4. **遇见故人** - 遇见昔日小厮（或袭人派来的人），但宝玉已不认识

**伏笔照应**:
- "落了片白茫茫大地真干净"
- 太虚幻境的预言全部应验

### 第104回 一僧一道来点化 贾宝玉悬崖初撒手

**核心剧情**: 一僧一道（茫茫大士、渺渺真人）出现，点化宝玉。

**场景**:
1. **雪中遇僧道** - 宝玉冻僵在雪地中，一僧一道出现
2. **通灵宝玉** - 道人拿出通灵宝玉（抄家时遗失，被道人拾得），宝玉恍惚记起
3. **前世今生** - 僧道讲述神瑛侍者与绛珠仙草的前缘，宝玉似懂非懂
4. **悬崖边** - 僧道引宝玉至悬崖边，说"跳下去，便解脱了"

**伏笔照应**:
- 第1回一僧一道的再次出现
- "悬崖撒手"的字面意义

### 第105回 重游太虚幻境 宝玉悟透因果

**核心剧情**: 宝玉似梦似醒重游太虚幻境，见到众女子判词，悟透一切。

**场景**:
1. **梦中入幻境** - 宝玉昏睡，魂游太虚幻境（或真的死亡边缘）

3. **众钗显灵** - 黛玉、宝钗、元春、探春等依次出现，各说一语，解其疑惑
4. **悟透因果** - 宝玉终于明白"假作真时真亦假"，放下执念

**伏笔照应**:
- 第5回神游太虚幻境的呼应
- "千红一哭，万艳同悲"的终极解释

### 第106回 剃度为僧归大荒 宝玉终得大解脱

**核心剧情**: 宝玉剃度为僧，跟随一僧一道离去，重归大荒山。

**场景**:
1. **悬崖撒手** - 宝玉纵身跳下悬崖（象征性的，或真的跳下但被救/已超凡）

3. **通灵归位** - 通灵宝玉重归大荒山，变回女娲补天遗石
4. **飘然而去** - 宝玉（情僧）随一僧一道远去，消失在风雪中

**伏笔照应**:
- 第1回"此石坠入红尘，历尽离合悲欢炎凉世态"

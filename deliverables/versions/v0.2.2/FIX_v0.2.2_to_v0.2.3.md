## 详细修复记录

### 修复1: 88回凤姐判词说破

**原文**:
```
"一从二令三人木..."她喃喃自语，苦笑一声，望着那断簪，久久不语。
```

**问题**: 角色直接说出判词，读者应自行领悟"人木"="休"

**修复后**:
```
她望着那断簪，想起从前——新婚时贾琏待她言听计从；后来她操持家务，贾琏渐渐对她发号施令；如今一纸休书将她逐出家门...这三个字，竟似她这一生的注脚。她苦笑一声，望着那断簪，久久不语。
```

**修复说明**: 
- 删除直接引用判词
- 改为通过"休书"暗示"人木"="休"
- 保留"三个阶段的转变"暗示"一从二令三人木"
- 让读者自行领悟判词含义

**文件**: `part-88-94.md` 第39行

### 修复2: 102回湘云判词说破（两处）

#### 修复2a: 回忆时

**原文**:
```
她想起那判词："湘江水逝楚云飞。"如今想来，那江水滔滔，那云影悠悠，竟似在向她招手。
```

**修复后**:
```
她望着那江水，望着那滔滔东流的江水，望着那天边悠悠飘散的云影，心中忽然生出一股说不出的凄凉...那水逝了，那云散了，竟似在向她招手一般...
```

**修复说明**:
- 删除"想起那判词"
- 改为直接描写江水、云影
- "水逝了...云散了..."暗示判词，不说破

**文件**: `part-102-108.md` 第27行

#### 修复2b: 临终时

**原文**:
```
她顿了顿，又道："湘江水逝...楚云飞...我...去也..."
```

**修复后**:
```
她顿了顿，又道："这水...这云...我也要随它们去了..."
```

**修复说明**:
- 删除直接引用判词
- 改为"这水...这云..."暗示
- 保留诗意，不说破

**文件**: `part-102-108.md` 第65行

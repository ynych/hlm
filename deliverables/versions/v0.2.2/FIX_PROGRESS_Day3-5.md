### 94回凤姐之死

**原文（八句古风）**:
```
英雄末路强撑骨，狱底凄凉命已迟。
断簪犹记当年事，一缕香魂归太虚。
产后血崩身欲坠，托孤辞世两堪悲。
茫茫白雪埋金簪，千古红颜总如此。
```

**问题**:
- 体裁不符（应为律诗）
- 第二联不对仗

**修复后（七言律诗）**:
```
英雄末路强撑骨，狱底凄凉命已迟。
断簪犹记当年事，冷月空照旧时姿。
产后血崩身欲坠，临终托孤两堪悲。
茫茫白雪埋金簪，千古红颜总若斯。
```

**改动说明**:
- 颔联（3-4句）改为对仗：当年事/旧时姿
- 韵脚统一：迟/姿/悲/斯

### 96回第一首（卫若兰之死）

**原文**:
```
白首双星梦已空，卫郎早逝泪痕红。
受托抚孤千金诺，不负当年金麒麟。
```

**问题**:
- "诺"字与"空/红/麟"出韵

**修复后（七言绝句）**:
```
白首双星梦已空，卫郎早逝泪痕红。
受托抚孤千金重，不负当年金麒麟。
```

**改动说明**:
- "诺"→"重"（zhong，与"空/红/麟"押韵）

### 96回第二首（湘云受托）

**原文（八句古风）**:
```
寒风凛冽雪纷飞，卫郎一去不复归。
宝姐姐命如残烛，托孤辞世两堪悲。
湘云一诺重千钧，抚育遗孤担大任。
末世红颜多薄命，何日重见太平春。
```

**修复后（七言律诗）**:
```
寒风凛冽雪纷飞，卫郎一去不复归。
宝钗命薄身先死，湘云诺重志未违。
抚育遗孤三载苦，托孤辞世两依依。
末世红颜多薄命，湘江水逝楚云飞。
```

**改动说明**:
- 颔联对仗：命薄/诺重，身先死/志未违
- 颈联对仗：抚育/托孤，三载苦/两依依
- 末句呼应判词

### 97回宝钗之死

**原文（八句古风）**:
```
金簪雪里埋香魂，二十六载一梦空。
天涯游子归来晚，破庙灵前哭断肠。
临终只道身是客，茫茫白雪掩芳踪。
此恨绵绵无绝期，金簪埋没雪中红。
```

**修复后（七言律诗）**:
```
金簪雪里埋芳魂，二十六年一梦空。
游子天涯归已晚，破庙灵前泪满胸。
临终只道身如寄，白雪茫茫掩玉容。
此恨绵绵无绝期，香魂一缕返仙踪。
```

**改动说明**:
- 颔联对仗：游子/破庙，天涯/灵前
- 颈联对仗：临终/白雪，身如寄/掩玉容
- 韵脚统一：空/胸/容/踪

### 98回李纨

**原文（八句古风，混杂黛/纨）**:
```
潇湘馆外草萋萋，旧帕重拾泪满衣。
晚韶华落空房冷，如冰水好总成悲。
稻香村里人寂寞，杏树枯枝映夕晖。
红颜老尽芳华逝，独守空房待岁归。
```

**修复后（七言律诗，聚焦李纨）**:
```
稻香村里草萋萋，晚韶华落锦屏空。
旧帕重拾泪痕在，如冰水好总成空。
杏帘在望人何在，寂寞空房度岁华。
一生心血付流水，独对寒灯数暮鸦。
```

**改动说明**:
- 删除黛玉元素（潇湘馆），聚焦李纨（稻香村）
- 颔联对仗：旧帕/如水，泪痕在/总成空
- 颈联对仗：杏帘/寂寞，人何在/度岁华

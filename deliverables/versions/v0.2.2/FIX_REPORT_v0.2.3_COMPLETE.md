**最终版本**:
```
她望着那断簪，想起从前——新婚时贾琏待她言听计从；
后来她操持家务，贾琏渐渐对她发号施令；
如今一纸休书将她逐出家门...
这三个字，竟似她这一生的注脚。
想起那年协理宁国府时的风光，
想起弄权铁槛寺时的狠辣，

```

**效果**: 通过"休书"暗示"人木"="休"，通过具体事件暗示判词，不说破

### 修复2: 102回湘云判词说破（P0→已修复）

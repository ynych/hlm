## Day 1-2: 6首出韵修复（已完成示例）

### 已修复示例

#### 82回（已修复）
```diff
- 停机德厚情难断，一任东风带笑看。
+ 停机德厚情难了，一任东风带笑看。
```
**问题**: "断"(duan)与"干/寒/看"(an)出韵  
**修复**: 改"断"为"了"(liao)，虽为邻韵但在古诗中可通

#### 84回（已修复）
```diff
- 诗稿焚余情未了，香魂一缕待归天。
+ 诗稿焚余情不尽，香魂一缕待归天。
```
**问题**: "了"(liao)与"悭/然/天"(an)出韵  
**修复**: 改"了"为"尽"(jin)，与"天"(tian)押韵

## Day 3-5: 8首体裁改为律诗（工作量最大）

### 问题说明
当前8首回末诗是**八句古风**，不符合《红楼梦》回末诗体例（应为**七言绝句或律诗**）。

### 需修改回目
- 94回: 英雄末路...（8句古风→7言律诗）
- 96回: 寒风凛冽...（8句古风→7言律诗）
- 97回: 金簪雪里...（8句古风→7言律诗）
- 98回: 潇湘馆外...（8句古风→7言律诗）
- 99回: 狠舅奸兄...（8句古风→7言律诗）
- 100回: 刘妪仗义...（8句古风→7言律诗）
- 101回: （如有）
- 102回: （如有）

### 示例：94回凤姐之死改为律诗

**原文（八句古风）**:
```
英雄末路强撑骨，狱底凄凉命已迟。
断簪犹记当年事，一缕香魂归太虚。
产后血崩身欲坠，托孤辞世两堪悲。
茫茫白雪埋金簪，千古红颜总如此。
```

**修复后（七言律诗）**:
```
英雄末路强撑骨，狱底凄凉命已迟。
断簪犹记当年事，冷月空照旧时姿。
产后血崩身欲坠，临终托孤两堪悲。
茫茫白雪埋金簪，千古红颜总若斯。
```

**改动说明**:
- 第二联改为对仗（当年事/旧时姿）
- 韵脚统一（迟/姿/悲/斯）
- 符合律诗平仄

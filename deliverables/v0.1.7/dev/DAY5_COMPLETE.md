### H-02-1: 贾政台词古典化

```
贾政叹道："政治上的事...说了你也不懂。"
```

```

```

**效果**: 去除现代语感，增加古典韵味

### H-02-2: 贾环台词阴鸷化

**新增内容**:
```
贾环听了，眼中闪过一丝阴鸷，冷笑道："平儿，你算什么东西？也敢管我的事？"

他说着，一巴掌将平儿打倒在地。平儿嘴角渗出血丝，却仍是死死护在巧姐身前。

贾环见了，心中更怒，一把揪住平儿的头发，阴恻恻地道：
"你以为你还是凤奶奶的贴身丫鬟？如今她死了，你不过是个没主的奴才，
也敢在我面前放肆？"
```

**效果**: 贾环性格更加阴鸷恶毒，人物更立体

### H-02-3: 甄士隐台词禅意化

**修改前**:
```
甄士隐此时方才抬起头来，望着那通灵玉，缓缓道：
"石头，石头，你如今可明白了？"
```

**修改后**:
```
甄士隐此时方才抬起头来，望着那通灵玉，目光如古井无波，缓缓道：
"顽石啊顽石，历劫归来，可悟得这红尘原是太虚？
那通灵宝玉，不过是幻相；这悲欢离合，不过是泡影。
你今日既已见到甄宝玉，当知真假本无别，痴醒总关情。"
```

**效果**: 禅意更浓，与《好了歌》风格统一

## 二、开发清单

### Phase 1: P1级开发（Day 1-3，必须完成）

#### P1-1: 101-102回"真假宝玉"论道语言风格调整

**位置**: 第101-102回"甄士隐默退贾雨村"、"悬崖撒手"  
**问题**: 论道场景哲学化过重，偏现代语感  
**目标**: 调整为更古典的表达方式

**开发方案**:
1. 保留"真假之论"的核心内容
2. 将现代哲学术语（"本体"、"存在"等）改为古典用语（"本来"、"真性"等）
3. 增加《好了歌》引用，强化古典语境
4. 调整句式，采用更多对偶、排比

**预计工时**: 4小时  
**交付物**: `dev-101-102-philosophy.md`

---

#### P1-2: 95回凤姐狱中反思女性视角增补

**位置**: 第95回"凤姐知命强英雄"  
**问题**: 凤姐反思聚焦"机关算尽"，对女性身份反思不足  
**目标**: 增加对"女子无才便是德"社会期待的反思

**开发方案**:
1. 在凤姐狱中回忆片段中，增加对"生为女子"的感慨
2. 增加对"若我是个男人"的假设性反思
3. 与探春"我但凡是个男人"形成对照
4. 保持凤姐"要强"人设，但展现性别枷锁

**参考文本**:
```
凤姐望着那断簪，忽然想道："我这一生，机关算尽，到头来...到头来若是个男人，或许...或许还能搏一搏。偏生是个女子，再精明强干，也不过是..."
```

**预计工时**: 3小时  
**交付物**: `dev-095-fengjie-gender.md`

---

#### P1-3: 89回贾府债务规模明确

**位置**: 第89回"元妃薨贾府树倒猢狲散"  
**问题**: 贾府债务规模未明确，经济逻辑可再精确  
**目标**: 贾政提及债务时给出大致数字

**开发方案**:
1. 在贾政与贾琏对话中，增加债务提及
2. 数值参考: "三万两上下"（符合国公府等级）
3. 债务构成: 高利贷+官员应酬+家用亏空
4. 与ECO-001专家确认数值合理性

**参考文本**:
```
贾政叹道："如今债台高筑，那利钱银子滚下来，竟有三万两之巨。这叫我如何是好..."
```

**预计工时**: 1小时  
**交付物**: `dev-089-debt-clarify.md`

---

### Phase 2: P2级开发（Day 4-5，选做2个）

#### P2-1: 101回惜春"勘破"过程强化（高优先级）

**位置**: 第101回"冷惜春甘伴青灯佛"  
**问题**: "勘破三春景不长"的"勘破"过程描写单薄  
**目标**: 增加目睹姐姐们命运后的内心转变

**开发方案**:
1. 增加惜春听闻元妃薨、迎春死、探春嫁后的反应
2. 增加惜春在栊翠庵看到妙玉被劫后的触动
3. 强化"勘破"不是逃避，而是看透后的主动选择
4. 与判词"勘破三春景不长"呼应

**预计工时**: 3小时  
**交付物**: `dev-101-xichun-awakening.md`

---

#### P2-2: 99-100回巧姐"留余庆"因果明确（高优先级）

**位置**: 第99-100回"狠舅奸兄"/"刘姥姥救孤"

**预计工时**: 2小时

#### P2-3: 94回妙玉"洁净"性别解读（中优先级，可选）

#### P2-4: 102回湘云卖唱身份补强（中优先级，可选）

**位置**: 第102回"湘江水逝楚云飞"

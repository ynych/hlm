## 修复策略

### 1. 术语替换表

| 现代术语 | 古典替换 | 示例 |
|:---------|:---------|:-----|
| 存在 | 实在、本真 | "存在即合理"→"实在皆有缘法" |
| 本体 | 真性、本来面目 | "本体论"→"真性之说" |
| 认识 | 识见、悟 | "认识论"→"识见之功" |
| 主体 | 自身、我 | "主体性"→"自家心性" |
| 客体 | 外物、相 | "客体性"→"外物之相" |
| 本质 | 真性、实相 | "本质"→"真性" |
| 现象 | 幻相、假相 | "现象"→"幻相" |

### 2. 表达方式调整

**原文**:
```
甄宝玉道："世人皆道我是真，你是假，其实不然。你我皆真，也皆假。真者，本来面目也；假者，幻化身形也。"
```

**修复后**:
```
甄宝玉望着天边渐沉的夕阳，缓缓道："世人皆道你是假，我是真，其实不然。你我都不过是那大荒山下的顽石，来这红尘中走一遭。"

"真性本来面目，幻形不过皮囊。你虽姓贾，我虽姓甄，却都是那'假作真时真亦假'的痴儿。"

宝玉听着，心中似有所动。

甄宝玉又道："你看这落日，日日升起，又日日沉落。升时是真，沉时是假么？不过是造化循环，真性不改罢了。"

"那红尘中的荣华富贵，情爱纠缠，便如这落日余晖，看似绚烂，转瞬即空。"
```

### 3. 以《好了歌》为基础

**新增引用**:
```
甄士隐拄着竹杖，口中吟道："陋室空堂，当年笏满床；衰草枯杨，曾为歌舞场..."

甄宝玉接道："蛛丝儿结满雕梁，绿纱今又糊在蓬窗上。说什么脂正浓、粉正香，如何两鬓又成霜？"

宝玉听着这《好了歌》，只觉字字锥心。
```

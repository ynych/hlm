## Day 4: 语言风格古典化

### P1-01: 101-102回"真假宝玉"论道语言风格修复

**原文问题**（示例）:
```
甄宝玉道："世人皆道我是真，你是假，其实不然。你我皆真，也皆假。真者，本来面目也；假者，幻化身形也。"
```

**修复后**:
```
甄宝玉望着天边渐沉的夕阳，缓缓道："世人皆道你是假，我是真，其实不然。你我都不过是那大荒山下的顽石，来这红尘中走一遭。"

"真性本来面目，幻形不过皮囊。你虽姓贾，我虽姓甄，却都是那'假作真时真亦假'的痴儿。"
```

**修复策略**:
1. 删除"存在"、"本体"、"认识论"等现代哲学术语
2. 改用"真性"、"本来"、"幻形"、"皮囊"等古典用语
3. 以《好了歌》为基础展开论道
4. 保留"情到极致即是空"的核心观点

**修复范围**: 101-102回论道场景，约800字

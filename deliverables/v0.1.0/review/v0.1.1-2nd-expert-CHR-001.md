### P1级问题（重要问题，建议修复）

#### 问题1：宝钗84回"主动性"与前80回人设冲突 ⚠️【上次遗留，未修复】

**具体表现**：
```
【宝钗主动】宝钗私下询问莺儿："老爷和太太...究竟是怎么打算的？"
【宝钗内心】："我虽知不该问...但这事关乎我一生...我不能只等别人安排..."
【宝钗主动】宝钗主动整理黛玉诗稿："她的东西...我来收着...日后..."
```

**问题分析**：
前80回宝钗的核心特质是**"藏愚守拙""不干己事不张口"**，即便内心关切，也绝不会形于色、见于行。第84回的设计让宝钗从"被动等待"转为"主动行动"，本质上改变了人物内核。

前80回佐证：
- 第34回宝玉挨打，宝钗只是"托着丸药进来"，从不主动探问
- 第57回黛玉病情，宝钗送燕窝也是"顺路"，绝不显得刻意
- 第28回元妃赐礼，宝钗"心里越发没意思起来"，表面仍恭顺

**修复建议**：
```
【调整方案】
- "宝钗主动询问" → "莺儿私下打探后告知宝钗"
- "宝钗主动整理" → "众人请宝钗代为收检，宝钗沉吟良久，终是应了"
- 内心独白保留，但行动必须克制：
  【暗示】宝钗听闻后，手微微一颤，忙别过脸去整理药碗
  【暗示】众人请她收检诗稿，她沉默良久，轻声："那...我便...收着吧..."
```

**风险等级**：若不修复，进入全文创作后，宝钗人物将出现内核分裂。

---

#### 问题2：探春远嫁后的情感留白处理不足 ⚠️【新增】

**具体表现**：
- 第86回探春远嫁后，直至第102回湘云病逝，探春**完全消失**
- 作为"才自精明志自高"的核心人物，远嫁后的命运没有任何交代
- 与宝玉、凤姐等人的情感联系被切断

**问题分析**：
前80回探春与宝玉、凤姐、贾母均有深厚情感连接。远嫁海外后，**至少应有一次互动**（书信/托梦/回忆），否则人物弧光出现断裂。

**修复建议**：
```
【可选方案】
方案A：第95回凤姐病逝时，增加探春托梦或书信回忆
方案B：第100回巧姐被救后，增加探春得知消息的侧面描写
方案C：第108回宝玉悬崖撒手时，闪回探春远嫁场景
```

### P2级问题（建议优化）

#### 问题3：湘云"歌妓"设定的历史合理性

**具体表现**：第102回湘云"沦为歌妓（卖艺不卖身）"

**问题分析**：
贵族女性沦落乐籍在清代需要更充分的社会背景支撑。虽然符合"白首双星"的悲剧命运，但"卖艺不卖身"在清代乐籍制度下需要特殊解释。

**建议**：
增加"卫家败落后被株连，女眷没入乐籍"或"卫若兰生前得罪权贵，死后遗孀遭报复"的政治背景铺垫。

**具体表现**：
贾桂作为连接宝钗-湘云-宝玉三线的关键人物，目前仅承担功能性作用。

全文创作时增加：
- 贾桂对母亲宝钗的记忆闪回
- 贾桂与宝玉重逢时的情感张力
- "兰桂齐芳"意象的具体呈现（与贾兰的互动）

## 七、评审结论

### 总体判断

大纲v0.1.0在人物塑造方面整体保持高水准，主要人物（宝玉、凤姐、黛玉、妙玉、湘云）的成长弧光完整，性格与前80回高度一致。创新点突出（黛玉泪尽、宝钗临终爆发、妙玉阶级悲剧），体现了对原著人物精神的深刻理解。

**但是**：
1. **宝钗84回主动性问题未修复**，反而被强化，存在人物内核分裂风险
2. **探春远嫁后完全消失**，情感弧光出现断裂

### 是否推荐进入全文创作

**⚠️ 有条件推荐**

### 前提条件

**必须修复（P1）**：
1. **弱化宝钗第84回的"主动询问"和"主动整理"**，恢复"藏愚守拙"特质
   - 询问改为莺儿打探后告知
   - 整理改为众人请托后勉力应承
2. **补充探春远嫁后的至少一次情感交代**（书信/托梦/回忆）

**建议优化（P2）**：
1. 湘云沦落乐籍增加历史背景铺垫
2. 全文创作时加强贾桂的情感深度

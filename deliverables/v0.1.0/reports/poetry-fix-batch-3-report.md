## 修复详情

### 第81回 占旺相四美钓游鱼 奉严词两番入家塾
**文件**: `ch081-fulltext.md`

**原诗**（首句出律）：
```
藕香榭外水潺潺，旺相难占运已残。
残荷听雨声声慢，一叶飘零一叶寒。
```

**修复后**：
```
藕榭秋深碧水潺，旺相难占运已残。
残荷听雨声声慢，一叶飘零一叶寒。
```

**修复说明**: 首句"藕香榭外水潺潺"改为"藕榭秋深碧水潺"，解决平仄出律问题，同时保持意境与押韵。

### 第83回 省宫闱贾元妃染恙 闹闺阃薛宝钗吞声

### 第89回 元妃薨贾府树倒猢狲散 宝玉等通辑捉拿入狱
**文件**: `ch089-fulltext.md`

**原诗**（多处出律）：
```
元妃薨逝大厦倾，树倒猢狲各散行。
```

### 第90回 狱神庙茜雪慰宝玉 挺孕肚宝钗主丧事
**文件**: `ch090-fulltext.md`

**原诗**（第三句多仄）：
```
狱墙内外两茫茫，茜雪送饭慰愁肠。
挺孕主丧贤德妇，家破人散各悲伤。
```

**修复后**：
```
狱墙内外两茫茫，茜雪送羹慰断肠。
贤德有娠主丧礼，家亡人散各凄惶。
```

**修复说明**: 第三句调整为"贤德有娠主丧礼"，修正多仄问题；"送饭"改为"送羹"，"凄惶"替代"悲伤"，格律合规。

### 第91回 警勾结恶仆欺孤儿 刘姥姥仗义勇接盘
**文件**: `ch091-fulltext.md`

**原诗**（孤平+出律）：
```
恶仆勾结欺孤儿，刘姥仗义救难时。
留余庆得苍天助，幸娘亲积得阴功。
```

**修复后**：
```
恶仆欺孤命运乖，刘婆仗义救难来。
苍天不负留余庆，幸有娘亲积德哉。
```

**修复说明**: 重写全诗，避免孤平问题，"刘婆"替代"刘姥"更合诗律，"积德哉"押韵合规且呼应"留余庆"判词。

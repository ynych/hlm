# 《癸酉续红》写作规范 v0.2.7

> 本文档定义《癸酉续红》的创作规范，适用于v0.2.7及后续版本

---

## 一、大纲结构规范

### 1.1 整体结构

```
《癸酉续红》28回大纲
├── 第一部分：渐衰期（第81-87回）
├── 第二部分：崩塌期（第88-94回）
├── 第三部分：离散期（第95-102回）
├── 第四部分：流浪期（第103-105回）
└── 第五部分：情榜期（第106-108回）
```

### 1.2 单回结构

每回必须包含以下要素：

| 要素 | 说明 | 必/选 |
|------|------|:-----:|
| 回目 | 八字回目（上4字+下4字） | 必 |
| 场景 | 主要场景列表 | 必 |
| 关键情节 | 3-5个关键情节点 | 必 |
| 判词落实 | 本回落实的判词 | 必 |
| 人物 | 出场主要人物 | 必 |
| 回末诗 | 七律，平仄合规 | 必 |

### 1.3 回目格式

```
第XX回 [上四字] [下四字]

示例：
第八十一回 占旺相四美钓游鱼 奉严词两番入家塾
第八十五回 泄机关颦儿泪尽亡 成婚配宝玉错姻缘
```

## 二、格式规范

### 2.1 标题层级

```markdown
# 第XX回 XXX XXX        ← 回目（一级标题）
## 场景一：XXX          ← 场景（二级标题）
### 关键情节           ← 情节（三级标题）
```

### 2.2 段落规范

- 每段不超过300字
- 对话单独成行
- 心理描写不超过50字
- 场景转换需空行

### 2.3 标点符号

- 使用全角标点
- 对话使用「」而非"
- 省略号使用……

## 三、风格要求

### 3.1 古典化程度

| 指标 | 要求 | 检查方法 |
|------|------|----------|
| 禁用现代词 | 100%清除 | 全局搜索 |
| 古典词汇占比 | ≥97% | 抽样检查 |
| 句式多样性 | 无重复句式 | 工具检测 |

### 3.2 禁用词清单

#### 绝对禁用（P0）

| 现代词 | 替换为 | 示例 |
|--------|--------|------|
| 知道 | 晓得/知晓 | "他知道"→"他晓得" |
| 明白 | 省得/晓得 | "明白了"→"省得了" |
| 心理 | 心思/心中 | "心理上"→"心思上" |
| 人格 | 性子/品性 | "人格分裂"→"性子变了" |
| 依靠 | 倚仗/依傍 | "依靠他"→"倚仗他" |
| 棋子 | 卒子/傀儡 | 完全删除 |
| 规训 | 管教/训导 | 完全删除 |
| 内化 | 记在心底 | 完全删除 |

#### 限制使用（P1）

| 词汇 | 限制 | 建议替换 |
|------|------|----------|
| 心中 | ≤5处/回 | 胸中、心底、肺腑 |
| 想起 | ≤3处/回 | 念及、忆起、思及 |
| 望着 | ≤5处/回 | 看、视、凝眸、瞻望 |
| 叹道 | ≤3处/回 | 说道、言道、喃喃道 |

### 3.3 含蓄表达原则

**三不说破原则**：
1. **判词不说破** - 不直接解释判词含义
2. **命运不说破** - 不直接点明人物命运
3. **主题不说破** - 不直接阐述主题思想

**正确示例**：
```
黛玉望着那梅枝，久久不语。 ← 暗示"玉带林中挂"
```

**错误示例**：
```
黛玉心想：这梅枝正应了我的判词"玉带林中挂"。 ← 说破！
```

| 人物 | 判词关键词 | 落实要求 |
|------|------------|----------|

| 贾探春 | 千里东风一梦遥 | 远嫁/江边送别 |
| 史湘云 | 湘江水逝楚云飞 | 江边/病逝 |
| 妙玉 | 终陷淖泥中 | 风尘/破庙 |
| 贾迎春 | 一载赴黄粱 | 被虐致死 |

| 贾巧姐 | 偶因济刘氏 | 刘姥姥相救 |
| 李纨 | 晚韶华 | 晚年凄凉 |
| 秦可卿 | 情天情海 | 情榜归属 |

### 4.2 判词落实方式

| 方式 | 说明 | 示例 |
|------|------|------|
| 意象暗示 | 用物象暗示 | 梅枝暗示"挂" |
| 场景营造 | 用场景呼应 | 雪中暗示"雪里埋" |
| 对话隐喻 | 用对话暗示 | "休了你"暗示"人木" |
| 动作象征 | 用动作暗示 | 断簪暗示"一从二令" |

### 4.3 判词铺垫要求

每处判词需有2-4处前期铺垫：

```
黛玉判词"玉带林中挂"铺垫：
- 81回：梅枝意象首次出现
- 83回：白绫意象强化
- 84回：林中场景描写
- 85回：正式落实（白绫挂梅枝）
```

## 五、人物塑造原则

### 5.1 人物一致性

| 人物 | 核心特质 | 维护要点 |
|------|----------|----------|
| 贾宝玉 | 情不情 | 始终痴情，但逐渐成长 |
| 林黛玉 | 泪尽而亡 | 病弱但敏感，不怨人只怨命 |
| 薛宝钗 | 停机德 | 克制但觉醒，"来世不做宝钗" |
| 王熙凤 | 机关算尽 | 精明但知命，三阶段心理 |
| 史湘云 | 豁达乐观 | 始终保持豪爽底色 |

### 5.2 心理描写规范

**禁止**：
- ❌ 【XX内心】标注
- ❌ 直接写"他想"
- ❌ 现代心理学术语

**允许**：
- ✅ 动作暗示心理
- ✅ 神态暗示心理
- ✅ 环境映射心理
- ✅ 物象触发心理

**示例**：
```
正确：
"黛玉望着那梅枝，手上一颤，帕子险些掉落。" ← 动作暗示恐惧

错误：
"黛玉内心恐惧地想：这正应了我的判词。" ← 直接说破
```

### 5.3 人物弧光要求

主要人物必须有完整的成长/转变：

| 人物 | 起点 | 转变 | 终点 |
|------|------|------|------|
| 宝玉 | 痴情公子 | 经历失去 | 悬崖撒手 |
| 宝钗 | 停机德 | 觉醒反思 | 来世不做宝钗 |
| 探春 | 精明要强 | 远嫁决断 | 才自精明志自高 |

## 六、回末诗规范

### 6.1 格式要求

- 体裁：七言律诗（8句）
- 平仄：符合律诗平仄
- 对仗：颔联、颈联对仗
- 押韵：平声韵，一韵到底

### 6.2 内容要求

- 概括本回内容
- 呼应判词意象
- 预示下回发展
- 保持悲剧美学

### 6.3 示例

```
第八十一回回末诗：

藕香榭外水潺潺，旺相难占运已残。
残荷听雨声声慢，一叶飘零一叶寒。

解析：
- 首联：点出场景（藕香榭）、主题（好景不长）
- 颔联：对仗，残荷意象暗示衰败
- 颈联：省略（本示例为绝句，实际为律诗）
- 尾联：省略
```

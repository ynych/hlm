# 第九十九回：计上计王仁卖甥女 狠舅奸兄各怀机

话说那王仁乃是凤姐的胞兄，自幼便是个不成器的。凤姐在时，时常接济他，他倒也过了几日安稳日子。后来凤姐入狱，贾府败落，他便失了依靠，只得靠着赌钱度日。

这日王仁正在赌坊中厮混，连输了十几把，欠下一屁股债。那债主乃是城中一霸，姓张，人称"张老虎"，心狠手辣，手下养着一帮打手。王仁欠了他五十两银子，已是三日未还，这日正被几个打手堵在赌坊中，一顿好打。

"王仁，你这钱到底还不还？"张老虎坐在椅子上，冷冷地问道。

王仁趴在地上，浑身是血，颤声道："张爷...再宽限几日...我一定还..."

"宽限？"张老虎冷笑一声，"我宽限你，谁来宽限我？今日你若不还钱，我便断你一条腿！"

他说着，一挥手，几个打手便围了上来，手中拿着棍棒，虎视眈眈。

王仁吓得魂飞魄散，跪地求饶："张爷饶命！张爷饶命！我...我有一计，可还您的银子！"

张老虎闻言，眉头一皱："什么计？说来听听。"

王仁爬起身来，凑到张老虎耳边，低声道："我有一外甥女，乃是当年荣国府的千金，生得标致，如今她母亲已死，父亲不知去向，正寄养在我家。若将她卖了...至少值个一二百两..."

张老虎闻言，眼中闪过一丝精光："荣国府的千金？你莫要骗我？"

"小的不敢欺瞒张爷！"王仁连忙道，"那孩子名叫巧姐，如今才七八岁年纪，正是好出手的时机。张爷若将她卖到那...那等地方...价钱更是不菲..."

他说着，心中也是一阵颤抖。他虽是个浑人，却也知道这卖外甥女的事情丧尽天良。可如今性命攸关，他也顾不得许多了。

张老虎沉吟片刻，点了点头："好，我便再信你一次。三日之内，将那孩子带来。若敢耍花样，我便要你的命！"

王仁连连点头，千恩万谢地退了出去。

出了赌坊，王仁站在街头，望着来来往往的行人，心中五味杂陈。他知道，自己这是将巧姐往火坑里推。可若不如此，他自己便没命了。

"巧姐...舅舅对不起你..."他喃喃自语，声音几不可闻，"但舅舅...也是没办法..."

他这般说着，却不敢回家，只在街上游荡。他怕见到巧姐那双清澈的眼睛，怕听到她唤自己"舅舅"的声音。

这日傍晚，王仁回到家中。那巧姐正坐在院中玩耍，手中拿着一个风车，正是凤姐生前给她做的。

"舅舅！"巧姐见到王仁，欢快地跑过来，"您回来啦！"

王仁望着她那纯真的笑脸，心中一酸，别过脸去，不敢与她对视。他含糊地应了一声，便匆匆走进内室。

"舅舅，您怎么了？"巧姐跟在后面，疑惑地问道。

"没...没什么..."王仁背对着她，声音沙哑，"舅舅累了，你去玩吧。"

巧姐虽然年幼，却也察觉到舅舅的异样。她站在那里，望着王仁的背影，心中涌起一股莫名的恐惧。

这日晚间，王仁正坐在屋中饮酒，忽听得门外有人敲门。他开门一看，只见贾环站在门外，神色闪烁。

"环哥儿？你怎么来了？"王仁惊讶地问道。

贾环走进屋来，四处张望了一下，低声道："王舅舅，我...我有事想与您商量。"

王仁心中一动，隐约猜到了几分。他将贾环让进屋中，关上门，低声问道："什么事？"

贾环坐在椅子上，沉默良久，才开口道："王舅舅，我知道...您最近手头紧..."

王仁闻言，心中一凛："你...你怎么知道？"

贾环苦笑一声："城中谁不知道您欠了张老虎的债？我也不过是在赌坊中听说的。"

他说着，抬起头，望着王仁，眼中闪过一丝复杂的光芒："王舅舅，我知道您想卖巧姐..."

王仁大惊，忙道："你...你怎么..."

"我也正有此意。"贾环打断他，声音平静，"那巧姐乃是凤姐的女儿，凤姐当年如何对我的，我心中清楚。如今她死了，她女儿落在我手中，我岂能放过？"

王仁闻言，心中一震。他望着贾环，只见这少年面容清秀，眼中却透着一股阴狠之气。他想起贾环这些年在贾府的遭遇，想起他被宝玉和凤姐欺负的种种，心中不由得一叹。

"环哥儿，你...你这是何苦？"他低声道，"巧姐不过是个孩子..."

"孩子？"贾环冷笑一声，"当年我也是个孩子，他们可曾放过我？"

他说着，站起身来，走到窗前，望着窗外的夜色，声音低沉："王舅舅，您知道吗？我这一辈子，都在被人看不起。太太不喜欢我，老太太不喜欢我，连那些丫头婆子，都敢给我脸色看。唯有我母亲，真心疼我..."

他说着，眼中闪过一丝泪光，却很快被他拭去："可母亲死了，死得不明不白。我知道，是凤姐害死的她。这个仇，我一直记在心中。"

王仁闻言，心中一寒。他知道贾环的母亲赵姨娘死因蹊跷，却没想到贾环竟将这笔账算在了凤姐头上。

"环哥儿，"他低声道，"你母亲的死...未必是凤姐..."

"不是她是谁？"贾环猛地转过身来，眼中满是恨意，"除了她，还有谁有这等心机？我母亲虽是个妾，却也是贾家的媳妇，岂能随随便便就死了？"

他说着，平复了一下情绪，继续道："王舅舅，我知道您想卖巧姐换钱还债。我可以帮您，但我要分一半。"

王仁闻言，心中一凛。他望着贾环，只见这少年眼中闪烁着贪婪与恨意的光芒，心中不由得一叹。

"好，"他点了点头，"成交。"

三日后的一个清晨，巧姐像往常一样在院中玩耍。王仁走过来，对她笑道："巧姐，舅舅带你去个好玩的地方，好不好？"

巧姐闻言，欣喜地点头："好啊！去哪里？"

"去了就知道了。"王仁说着，伸手去拉她。

巧姐刚要伸手，忽又缩了回来，疑惑地问道："舅舅，您为什么...不敢看我？"

王仁闻言，手微微一颤。他别过脸去，低声道："舅舅...舅舅眼睛不舒服...走吧，马车在外面等着呢。"

巧姐虽然心中疑惑，却也没再追问。她跟着王仁走出院门，只见一辆马车停在门外，贾环坐在车辕上，面无表情。

"环哥哥？"巧姐惊讶地叫道，"你也去吗？"

贾环看了她一眼，那眼神复杂难辨。他没有回答，只是点了点头。

巧姐上了马车，王仁和贾环坐在她两侧。马车缓缓启动，向城外驶去。

"舅舅，我们要去哪里呀？"巧姐望着窗外越来越荒凉的景色，心中涌起一股不安。

"去...去一个很好玩的地方..."王仁含糊地答道，声音沙哑。

巧姐望着他，只见他双手紧握，指节发白，似乎在强忍着什么。她心中更加不安，伸出小手去拉他的衣袖："舅舅，您怎么了？是不是不舒服？"

王仁被她这一拉，心中如被刀割。他望着巧姐那双清澈的眼睛，想起她母亲凤姐的音容笑貌，心中悔恨交加。

"巧姐..."他哽咽道，"舅舅...舅舅对不起你..."

巧姐闻言，心中一惊："舅舅，您说什么？"

话音未落，马车突然停了下来。车门被拉开，张老虎带着几个打手站在车外，冷冷地望着车内。

"到了，下来吧。"张老虎说道。

巧姐望着那些凶神恶煞的打手，心中恐惧到了极点。她紧紧抓住王仁的衣袖，哭喊道："舅舅！我怕！我要回家！我要娘！"

王仁被她这一哭，心中最后的一丝良知也被唤醒。他紧紧抱住巧姐，颤声道："张爷...这孩子...这孩子我不卖了..."

张老虎闻言，脸色一变："王仁，你敢耍我？"

"不是...我..."王仁语无伦次，"这孩子...她..."

"舅舅..."巧姐在他怀中哭喊，"您不要丢下我...我怕..."

王仁泪如雨下，紧紧抱住巧姐："不怕...舅舅在...舅舅不会丢下你..."

张老虎见状，冷笑一声："王仁，既然你不识抬举，那就别怪我不客气了。来人，把那孩子抢过来！"

几个打手一拥而上，王仁拼命护住巧姐，却被打得头破血流。巧姐在他怀中哭喊："舅舅！环哥哥！救我！"

贾环站在一旁，望着这一幕，心中也是翻江倒海。他想起当年自己被欺负的情景，想起母亲含冤而死，心中恨意又起。可当他看到巧姐那双惊恐的眼睛，那与当年自己如出一辙的恐惧，心中又涌起一股说不清道不明的情绪。

"住手！"他突然喊道。

众人一愣，都望向他。

贾环走上前来，望着张老虎，沉声道："这孩子，是我带来的。要卖要留，我说了算。"

张老虎皱眉道："环哥儿，你这是什么意思？"

贾环没有回答，只是走到巧姐身前，蹲下身子，望着她那双泪眼，轻声道："巧姐，环哥哥问你，你娘对你好吗？"

巧姐抽泣着点头："娘...娘对我最好了..."

"那你可知，我娘是怎么死的？"贾环的声音平静得可怕。

巧姐摇了摇头，眼中满是恐惧。

贾环苦笑一声，站起身来，望着王仁，叹道："王舅舅，你带她走吧。"

王仁闻言，又惊又喜："环哥儿，你..."

"我娘的死，与这孩子无关。"贾环低声道，"我...我不想变成她们那样..."

他说着，转身离去，背影在夕阳中显得格外孤独。

王仁抱着巧姐，望着贾环离去的背影，心中百感交集。他知道，今日这一劫，算是躲过了。可那五十两银子的债，又该如何偿还？

"舅舅..."巧姐在他怀中轻声唤道，"我们...可以回家了吗？"

王仁点了点头，泪如雨下："回家...我们回家..."

可他心中清楚，他们已经没有家了。

正是：

狠舅奸兄各怀心，甥女何辜被卖身。
当年济困刘姥姥，今日全凭仗义深。

当年济困刘姥姥，今日全凭仗义深。
王仁临阵良心现，贾环半途恨意沉。
巧姐惊魂方未定，前途茫茫未可知。

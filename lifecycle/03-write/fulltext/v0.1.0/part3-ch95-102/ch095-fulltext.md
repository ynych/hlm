# 第九十五回：凤姐知命强英雄 宝钗产后染重疾

话说凤姐自那日狱中见了刘姥姥，将巧姐托付之后，心中倒也放下一块石头。只是那牢中日子，日渐难挨。这日正是隆冬时节，狱窗外北风呼啸，吹得那残破的窗纸呜呜作响，如泣如诉。

凤姐蜷缩在草堆之中，手中握着那支断簪，怔怔地出神。这簪子还是那年贾母赏的，金丝掐丝的凤头，原是成双成对，如今只剩这一支，另一支早在抄家那日便断成了两截。她记得那日整理衣物时，这簪子忽地从匣中滑落，断成两截，当时心中便是一沉，暗叫不祥。如今想来，竟是一语成谶。

"奶奶，您吃点东西吧。"身旁一个女囚递过半块硬饼。这女囚姓赵，原是城外庄户人家，因丈夫赌钱欠了债，被人诬告偷盗，一同下了狱。凤姐刚进来时，这赵氏见她衣饰华贵，气度不凡，心中甚是敬畏，不敢近前。后来见凤姐虽遭大难，却不肯低声下气求人，反倒将身上仅剩的银钱散了给狱中病弱的囚妇，心中便生出几分敬意。

凤姐抬起头来，勉强一笑："你吃吧，我不饿。"

"奶奶别逞强了，"赵氏叹道，"您这身子，自进来便没好过，今日又咳了半日。这牢里阴寒，您再不吃点东西，如何熬得过去？"

凤姐听她这般说，心中一酸。想当初在贾府，何曾有人敢这般对她说话？便是贾母，也是和颜悦色，从不疾言厉色。如今落难，反倒得一个乡下妇人来劝慰。她接过那半块饼，咬了一口，只觉粗粝难咽，却强自咽了下去。

"你为何对我这般好？"凤姐问道。

赵氏在草堆上坐下，叹道："奶奶虽落了难，可骨子里那股子劲儿还在。我虽是个乡下人，却也看得出来，您不是一般人。这牢里其他人，有的哭天喊地，有的装疯卖傻，有的求爷爷告奶奶，只有您，不声不响，该吃什么吃什么，该睡就睡，倒像是...倒像是认命了，却又不甘心认命的样子。"

凤姐闻言，手中那半块饼微微一颤。她望着手中断簪，喃喃道："认命...却又不甘心认命...你说得倒准。我这一辈子，机关算尽，算来算去，算不过一个命字。"

赵氏不明其意，只道："奶奶别想那么多了。我男人常说，人活一世，草木一秋，命里有的终须有，命里无的莫强求。您这般人物，说不定哪日便出去了。"

凤姐苦笑一声："出去？我自知是出不去了。那日刘姥姥来，我已将女儿托付给她。我这辈子，便只剩下这一件事还未了。"

她说着，将那断簪贴在胸口，闭上双眼。赵氏见她神色疲惫，不敢再扰，轻轻退到一旁。

凤姐闭目良久，忽觉眼前一片昏黑，那断簪从手中滑落。她想起那年协理宁国府，何等风光；想起弄权铁槛寺，何等狠辣；想起设计害死尤二姐，心中可曾有过一丝悔意？彼时只觉人不为己，天诛地灭，如今想来，竟如隔世。

牢外风声更紧，卷着雪花从窗缝中钻进来。凤姐打了个寒颤，睁开眼来，只见眼前一片白茫茫。她伸手去接那雪花，却见雪花落在掌心，顷刻便化了。

"化了...都化了..."她轻声叹道。

这日晚间，凤姐发起了高烧。赵氏守在她身旁，急得团团转，却无可奈何。狱中又无医药，只得用湿布敷在她额头，盼着能退些热。

凤姐昏昏沉沉，眼前却不断闪过往事。她看见贾琏在她眼前赔笑，看见贾母握着她的手说"你是我重孙媳妇中第一个得意之人"，看见巧姐在她怀中撒娇，看见平儿在她身后默默垂泪...

"平儿..."她喃喃唤道。

赵氏在一旁听见，忙应道："奶奶，我在。"

凤姐却似未听见，仍自喃喃："平儿...我对不住你...你跟了我这些年...我没给过你一天好日子..."

赵氏听得心酸，眼眶一红，忙别过脸去。

凤姐烧了三日三夜，到了第三日夜间，忽地清醒过来。她睁开眼，望着牢顶那方小小的天窗，只见一轮冷月悬于天际，清辉洒落，照得牢中一片惨白。

"今日...是十五了？"她轻声问道。

赵氏连忙过来："奶奶醒了？可好些了？"

凤姐不答，只是望着那轮冷月，怔怔地出神。良久，她缓缓坐起身来，将那支断簪插在发间，整理了一下身上破旧的衣裳，竟似要去赴什么重要的宴席一般。

"奶奶，您这是..."

凤姐转过头来，望着赵氏，忽然一笑。那笑容中竟有几分往日的风采，只是眼底深处，却是一片死寂。

"我这一生，要强惯了。便是死，也要死得体面。"她轻声道，"你替我...替我梳梳头吧。"

赵氏闻言，心中一酸，眼泪夺眶而出。她颤抖着双手，将凤姐散乱的发丝拢起，用那断簪别好。凤姐望着她，轻声道："多谢你这些日子的照顾。我无以为报，只有这..."

她从怀中取出一个小小的荷包，正是巧姐之物，上面绣着一只小小的凤凰。她将那荷包贴在胸口，闭上双眼，一滴泪从眼角滑落。

"我的儿...娘...娘等不到你了..."

话音未落，凤姐身子一软，便倒在了草堆之中。赵氏大惊，忙去探她鼻息，却觉气息奄奄，已是油尽灯枯之相。

"来人啊！救命啊！"赵氏哭喊着扑到牢门前，用力拍打着木栅栏。

狱卒闻声赶来，见凤姐倒在草堆中，面色惨白，唇边却带着一丝笑意。那狱卒探了探她鼻息，摇了摇头："没救了。血山崩加上心病，能撑到如今已是奇迹。"

赵氏痛哭失声，跪倒在凤姐身旁。只见凤姐手中仍紧紧握着那个荷包，面上神情安详，竟似睡着了一般。唯有那支断簪，在月光下闪着冷冷的幽光。

消息传到贾府残宅，众人皆惊。王夫人哭得死去活来，宝钗挺着虚弱的身子，勉强支撑着料理后事。那宝钗自产后大出血，身子便一日不如一日，如今又遭此打击，更是雪上加霜。

这日宝钗正靠在床头，听着平儿回禀凤姐的后事安排。她面色蜡黄，双颊凹陷，眼窝深陷，哪有半分往日丰腴圆润的模样？唯有那双眼睛，仍透着几分清明。

"奶奶的后事...便按你说的办吧。"宝钗轻声道，"只是...莫要太过铺张。如今家里..."

她说着，便是一阵咳嗽。平儿忙上前替她捶背，眼中含泪："奶奶，您要保重身子。凤奶奶已经去了，您若再有个三长两短，这家里..."

宝钗摆了摆手，止住咳嗽，轻声叹道："我自知...身子不中用了。"她说着，望向窗外，只见院中那株老梅，在寒风中瑟瑟发抖，枝头几朵残花，已是摇摇欲坠。

"这花...落得比往年早..."她喃喃道。

平儿顺着她的目光望去，心中一酸，却不敢接话。

宝钗收回目光，望着平儿，忽然问道："贾桂呢？"

"奶娘抱着呢，刚睡下。"

"抱来我瞧瞧。"宝钗说着，挣扎着要起身。

平儿忙将她扶起，在她身后垫了几个软枕。不多时，奶娘将贾桂抱了来。那孩子刚满周岁，生得白白胖胖，一双眼睛乌溜溜的，甚是可爱。

宝钗接过孩子，抱在怀中，低头望着那张小脸，眼中满是慈爱。她伸手轻抚孩子的脸颊，轻声道："桂儿...娘怕是不能陪你了..."

平儿闻言，心中大惊，忙道："奶奶这是说的什么话？您还要看着小爷长大，看着他娶妻生子呢！"

宝钗苦笑一声，将孩子贴在胸口，闭上双眼。良久，她轻声道："去请史大姑娘来。我有话对她说。"

平儿不敢违拗，只得去请湘云。那湘云自卫若兰死后，便寄居在贾府，帮着宝钗料理家事。如今闻听宝钗相请，忙赶了来。

"宝姐姐，你找我？"湘云一进屋，见宝钗面色，心中便是一沉。

宝钗将孩子递给奶娘，示意她抱下去。待众人退下，她才拉着湘云的手，轻声道："云妹妹，我自知...大限将至。"

"姐姐胡说什么！"湘云眼眶一红，"你不过是产后虚弱，调养调养便好了。"

宝钗摇了摇头，惨然一笑："我自己的身子，我自己知道。这些日子，我夜夜梦见林姐姐，她站在潇湘馆的竹林里，朝我招手..."

湘云听得毛骨悚然，却不敢打断。

"我这辈子..."宝钗望着窗外，目光悠远，"活得太累了。从小便要学会懂事，要学会藏拙，要学会做一个人人称道的'宝姑娘'。后来嫁了宝玉，又要学着做一个贤良的妻子。如今...如今终于可以做回自己了..."

她说着，眼中闪过一丝解脱的光芒。

"姐姐..."湘云握住她的手，只觉那手冰凉刺骨。

"贾桂..."宝钗转过头来，望着湘云，眼中满是恳求，"这孩子，我就托付给你了。"

湘云闻言，泪如雨下："姐姐放心，我...我一定..."

"我知道..."宝钗轻声道，"你是个重情义的人。卫若兰去了，你一个人...也是孤苦。这孩子跟着你，或许...或许能给你们彼此做个伴..."

她说着，又是一阵剧烈的咳嗽，咳出的痰中竟带着血丝。湘云大惊，忙替她抚胸捶背。

待咳嗽稍止，宝钗靠在床头，喘着粗气，轻声道："我这一生...看似圆满，实则...从未为自己活过一日。若有来世...我不想再做'宝钗'..."

她说完，便闭上双眼，不再言语。湘云守在一旁，泪湿衣襟。

窗外，北风呼啸，卷起漫天雪花。那株老梅终于不堪重负，枝头最后一朵残花飘落，随风而去，消失在茫茫雪色之中。

正是：

机关算尽太聪明，反算了卿卿性命。
生前心已碎，死后性空灵。

【回末诗】
英雄末路强撑骨，狱底凄凉命已迟。
断簪犹记当年事，一缕香魂归太虚。
产后血崩身欲坠，托孤辞世两堪悲。
茫茫白雪埋金簪，千古红颜总如此。

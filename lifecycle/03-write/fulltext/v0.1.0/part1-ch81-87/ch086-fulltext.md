# 第八十六回 冯紫英射圃饯探春 卫若兰赠麟定终身

且说黛玉去后，宝玉大病一场，整日昏昏沉沉，不思饮食。王夫人与宝钗在旁日夜看护，心中甚是焦急。

那宝钗虽是新妇，却不得新郎一顾，心中五味杂陈，却也不便显露，只默默打点家事，侍奉婆母。

这日，宝玉略好些了，倚在床头，望着窗外的景色，久久不语。宝钗端了药进来，见他这般模样，轻声道："二爷，该用药了。"

宝玉听了，缓缓转过头来，望着宝钗，眼中一片茫然。他想起黛玉，想起她那苍白的面容，想起她那双含着泪光的眼睛，心中如刀绞一般。

他接过药碗，却不饮，只望着那药汁，喃喃道："这药...苦得很..."

宝钗听了，心中一酸，却也不便说什么，只轻声道："良药苦口，二爷还是饮了罢。"

宝玉听了，苦笑一声，将药饮尽。那苦味在口中弥漫，像是他这一生的滋味。

且说那日，探春在秋爽斋中，望着窗外的景色，心中也是烦闷。她想起黛玉去后，园中的姐妹们日渐稀少，心中越觉悲凉。

她唤过侍书，道："你去请宝二爷来，就说我有话跟他说。"

侍书领命去了。不多时，宝玉进来，见了探春，勉强笑道："三妹妹唤我何事？"

探春望着宝玉，见他面色憔悴，心中也是一阵酸楚。她道："二哥哥，你身子可好些了？"

宝玉道："好些了，多谢三妹妹关心。"

探春叹道："二哥哥，我今日唤你来，是有一事相告。我...我即将远嫁，特来向你辞行。"

宝玉听了，心中大惊，道："三妹妹要嫁到哪里去？"

探春道："镇海统制周家。"

宝玉听了，心中越觉悲凉。他想起探春平日里的才情抱负，想起她那"才自精明志自高"的诗句，心中越觉不舍。

他道："三妹妹这一去，不知何时才能再见..."

探春听了，眼中闪过一丝泪光，却强忍着不让它落下。她道："二哥哥不必伤感，人生在世，聚散离合，原是常事。我只盼二哥哥能振作起来，莫要再这般消沉了。"

宝玉听了，心中更是难过。他望着探春，望着她那坚毅的面容，想起她平日里的种种好处，心中越觉悲凉。

他道："三妹妹放心，我会...我会振作的..."

探春听了，轻轻点了点头，道："那便好。"

且说那日，冯紫英在射圃设宴，为探春饯行。那射圃位于城外，占地甚广，四周绿树环绕，环境清幽。

宝玉、宝钗、湘云、卫若兰等人皆来相送。那冯紫英是个豪爽之人，见众人来了，忙迎上来笑道："各位来得正好，今日咱们不醉不归！"

众人入座，酒过三巡，冯紫英举杯道："探春姑娘即将远嫁，咱们今日为她饯行，祝她一路顺风，夫妻和睦！"

众人听了，皆举杯相和。探春起身，向众人福了一福，道："多谢各位相送，探春感激不尽。"

她说着，目光落在宝玉身上，见他面色憔悴，心中也是一阵酸楚。她道："二哥哥，你...你要保重身子..."

宝玉听了，心中一酸，眼泪差点掉下来。他强忍着，道："三妹妹放心，我会的..."

且说那日，卫若兰见湘云在旁，神色间有些落寞，心中一动。他想起平日里的种种，想起湘云那豪爽的性情，心中越觉喜欢。

他悄悄拉了湘云到一旁，从怀中取出一块金麒麟，递给湘云道："史姑娘，这是我祖上传下来的，今日赠与你，愿你...愿你日后平安喜乐。"

湘云听了，心中一惊，脸上飞起两朵红云。她望着卫若兰，望着他那双真诚的眼睛，心中越觉欢喜。

她接过金麒麟，轻声道："多谢卫公子..."

卫若兰听了，心中大喜，道："史姑娘不必客气，我...我..."

他说着，却不知该如何说下去，只望着湘云，久久不语。

湘云见了，心中也是一阵羞涩，却也不便说什么，只将那金麒麟紧紧握在手中，像是握住了自己的命运。

且说那日，探春登船远去，众人站在岸边，望着那船渐渐消失在视线中，心中皆是唏嘘。

宝玉望着那远去的船只，想起探春平日里的种种好处，想起她那"才自精明志自高"的诗句，心中越觉悲凉。

他想起园中的姐妹们，一个个离他而去，黛玉死了，探春嫁远了，湘云也即将出嫁，只剩下他一人，在这空荡荡的园中，独自伤心。

他望着那远去的船只，望着那茫茫的江水，心中忽然生出一股说不出的滋味。他想起自己的一生，想起他的情，他的痴，他的傻，心中越觉悲凉。

他轻轻叹了一口气，转身离去。那夕阳的余晖洒在他的身上，将他的影子拉得很长很长，像是一个孤独的旅人，在这末世中独自前行。

宝钗在旁，望着宝玉的背影，望着他那孤独的身影，心中也是一阵酸楚。她想起自己的处境，想起那日的梦境，想起"我不想再做'宝钗'"，心中越觉烦闷。

她轻轻叹了一口气，跟在宝玉身后，默默离去。

且说那日，宝玉回到怡红院，躺在床上，望着窗外的月色，久久不语。他想起探春，想起黛玉，想起园中的姐妹们，心中越觉悲凉。

他想起自己的诗，想起"满纸荒唐言，一把辛酸泪"，想起"都云作者痴，谁解其中味"，心中越觉悲凉。

他轻轻闭上眼睛，两行清泪，顺着脸颊滑落。他在这泪水中，渐渐睡去，梦中，他仿佛又看见了黛玉，看见了探春，看见了那些离他而去的姐妹们...

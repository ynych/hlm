# 第一零六回 警幻仙子宫中点 金陵十二钗归位

话说宝玉随茫茫大士、渺渺真人来到太虚幻境宫门前，那宫门缓缓开启，一道柔和的光芒从门内射出，将宝玉笼罩其中。宝玉只觉浑身轻飘飘的，仿佛要飞升而去。

进入宫中，只见殿宇巍峨，雕梁画栋，仙气缭绕，瑞霭千条。殿中设有一座高台，台上坐着一位仙姑，头戴凤冠，身披霞帔，面容端庄，目光慈悲，正是警幻仙姑。

警幻仙姑见宝玉到来，微微颔首，道："石头，你终于来了。"

宝玉忙上前施礼："仙姑，晚辈有礼了。"

警幻仙姑示意宝玉起身，道："你这一世，历经红尘悲欢，如今已功德圆满。今日召你来此，是要为你点明这'金陵十二钗'的归宿，让你明白这'情'字的真谛。"

说罢，警幻仙姑从袖中取出一卷册子，那册子金光闪闪，上面赫然写着几个大字："金陵十二钗正册"。

"石头，你看。"警幻仙姑将册子展开，只见册子上依次浮现出十二道光芒，每一道光芒都化作一个名字，璀璨夺目。

第一道光芒，化作"林黛玉"三字。警幻仙姑轻声道："林黛玉，绛珠仙草转世，为还泪而来。泪尽而亡，痴情第一。"册子上随即浮现出黛玉的判词："可叹停机德，堪怜咏絮才。玉带林中挂，金簪雪里埋。"

宝玉望着那判词，泪如雨下。他想起了黛玉生前的种种，想起了她的才情，想起了她的痴情，想起了她临终前的泪眼...

第二道光芒，化作"薛宝钗"三字。警幻仙姑道："薛宝钗，停机德厚，金玉良缘。虽得婚姻，未得人心，终是薄命。"判词浮现："可叹停机德，堪怜咏絮才。玉带林中挂，金簪雪里埋。"

宝玉想起了宝钗的温柔贤淑，想起了她在病榻上的嘱托，想起了她临终前的那句话："若有来世...我不想再做'宝钗'..."

第三道光芒，化作"贾元春"三字。警幻仙姑道："贾元春，皇妃之尊，荣华富贵。然虎兕相逢，大梦归，终究是一场空。"判词浮现："二十年来辨是非，榴花开处照宫闱。三春争及初春景，虎兕相逢大梦归。"

宝玉想起了元妃省亲时的盛况，想起了她临终前的托梦，想起了她那声叹息："虎兕相逢，我成替罪羊..."

第四道光芒，化作"贾探春"三字。警幻仙姑道："贾探春，才自精明志自高，生于末世运偏消。清明涕送江边望，千里东风一梦遥。"判词浮现："才自精明志自高，生于末世运偏消。清明涕送江边望，千里东风一梦遥。"

宝玉想起了探春远嫁时的决绝，想起了她站在江边的背影，想起了她那句话："这家里，怕是要散了..."

第五道光芒，化作"史湘云"三字。警幻仙姑道："史湘云，白首双星，湘江水逝。富贵又何为？唯有真情在。"判词浮现："富贵又何为，襁褓之间父母违。展眼吊斜晖，湘江水逝楚云飞。"

宝玉想起了湘云醉卧时的憨态，想起了她在湘江边弹奏的《海棠诗》，想起了她临终前的那句话："湘江水逝...楚云飞...我...去也..."

第六道光芒，化作"妙玉"三字。警幻仙姑道："妙玉，欲洁何曾洁，云空未必空。可怜金玉质，终陷淖泥中。"判词浮现："欲洁何曾洁，云空未必空。可怜金玉质，终陷淖泥中。"

宝玉想起了妙玉在栊翠庵中的高洁，想起了她被劫时的绝望，想起了她手中的那枚成窑杯碎片...

第七道光芒，化作"贾迎春"三字。警幻仙姑道："贾迎春，子系中山狼，得志便猖狂。金闺花柳质，一载赴黄粱。"判词浮现："子系中山狼，得志便猖狂。金闺花柳质，一载赴黄粱。"

宝玉想起了迎春的懦弱，想起了她被虐死时的惨状，想起了她那句话："一载赴黄粱..."

第八道光芒，化作"贾惜春"三字。警幻仙姑道："贾惜春，勘破三春景不长，缁衣顿改昔年妆。可怜绣户侯门女，独卧青灯古佛旁。"判词浮现："勘破三春景不长，缁衣顿改昔年妆。可怜绣户侯门女，独卧青灯古佛旁。"

宝玉想起了惜春出家的决绝，想起了她跪在佛前的背影，想起了她那句话："独卧青灯古佛旁..."

第九道光芒，化作"王熙凤"三字。警幻仙姑道："王熙凤，一从二令三人木，哭向金陵事更哀。机关算尽太聪明，反算了卿卿性命。"判词浮现："一从二令三人木，哭向金陵事更哀。生前心已碎，死后性空灵。"

宝玉想起了凤姐的精明强干，想起了她在狱中握着巧姐荷包的泪眼，想起了她那句话："机关算尽...算不过命..."

第十道光芒，化作"贾巧姐"三字。警幻仙姑道："贾巧姐，势败休云贵，家亡莫论亲。偶因济刘氏，巧得遇恩人。"判词浮现："势败休云贵，家亡莫论亲。偶因济刘氏，巧得遇恩人。"

宝玉想起了巧姐被救时的感恩，想起了她在纺车旁讲述母亲故事的情景，想起了她那句话："我娘...是这世上...最厉害的人..."

第十一道光芒，化作"李纨"三字。警幻仙姑道："李纨，桃李春风结子完，到头谁似一盆兰。如冰水好空相妒，枉与他人作笑谈。"判词浮现："桃李春风结子完，到头谁似一盆兰。如冰水好空相妒，枉与他人作笑谈。"

宝玉想起了李纨晚年的凄凉，想起了她坐在空房中望夕阳的背影，想起了她那句话："晚韶华...如冰水好...总成悲..."

第十二道光芒，化作"秦可卿"三字。警幻仙姑道："秦可卿，情天情海幻情身，情既相逢必主淫。漫言不肖皆荣出，造衅开端实在宁。归于情类，终是薄命。"判词浮现："情天情海幻情身，情既相逢必主淫。漫言不肖皆荣出，造衅开端实在宁。"

宝玉想起了可卿生前的温柔，想起了她早逝时的悲伤，想起了她在太虚幻境中的身影...

十二道光芒依次亮起，又依次熄灭。警幻仙姑望着那十二道光芒，轻轻叹息："千红一哭，万艳同悲。"

这八个字，道尽了这十二位女子的悲欢。她们或才情出众，或美貌无双，或出身显赫，或品格高洁，但终究都逃不过一个"命"字。她们的泪水，她们的悲伤，她们的绝望，都在这八个字中得到了浓缩。

宝玉跪倒在地，泪流满面。他想起了与这十二位女子的种种交集，想起了她们的欢笑，想起了她们的泪水，想起了她们的悲欢，想起了她们的离合。他忽然明白了，这"情"字的真谛——不是占有，不是执着，而是珍惜；不是永恒，不是不变，而是曾经拥有。

"仙姑...她们...现在何处？"宝玉颤声问道。

警幻仙姑微微一笑，道："她们都在这太虚幻境之中，化为星辰，永恒闪烁。你看。"

说罢，警幻仙姑挥袖一指，只见殿顶忽然变得透明，露出满天星辰。那星辰闪烁，仿佛在诉说着什么。宝玉仔细看去，只见其中十二颗星辰格外明亮，正是对应着那十二位女子。

"那是黛玉...那是宝钗...那是云妹妹..."宝玉指着星辰，一一辨认。

警幻仙姑点头："正是。她们的肉身虽已消逝，但她们的情，她们的爱，她们的灵魂，都已化为星辰，在天上永恒地闪烁。她们不再受尘世的悲欢所苦，不再受命运的摆布，她们自由了。"

宝玉望着那满天星辰，心中既悲且喜。悲的是，他再也见不到她们了；喜的是，她们终于得到了解脱。

"仙姑，那副册和又副册呢？"宝玉忽然问道。

警幻仙姑点头，又从袖中取出两卷册子，分别是"金陵十二钗副册"和"金陵十二钗又副册"。

她将副册展开，只见上面依次浮现出香菱、薛宝琴、邢岫烟、李纹、李绮、宝蟾、宝珠、瑞珠等人的名字。警幻仙姑只是轻轻一挥，那些名字便化为光芒闪过，没有过多的评说。

又将又副册展开，只见上面依次浮现出袭人、晴雯、鸳鸯、紫鹃、莺儿、平儿、秋纹、麝月等人的名字。同样，只是光芒闪过，没有评说。

宝玉望着那些名字，心中感慨万千。这些女子，有的与他有过交集，有的只是一面之缘，有的则是素未谋面。但她们都是这末世中的红颜，都是这悲剧中的牺牲者。

"她们...也都化为了星辰？"宝玉问道。

警幻仙姑点头："正是。千红万艳，都在这太虚幻境之中。她们的命运各不相同，但结局都是一样——归于情类，归于永恒。"

宝玉沉默了。他望着那满天星辰，心中忽然涌起一股莫名的情绪。那是悲伤，是释然，是感慨，是超脱...那是所有情绪的交织，是所有感悟的升华。

"石头，你可明白了？"警幻仙姑问道。

宝玉点头："我明白了。这'情'字，不是执着，而是珍惜；不是占有，而是祝福。她们虽然离去了，但她们的情还在，她们的爱还在，她们的灵魂还在...我要带着她们的情，继续走下去..."

警幻仙姑欣慰地点头："说得好。你终于明白了这'情'字的真谛。现在，你可以回去了。"

"回去？"宝玉有些茫然，"回哪里去？"

警幻仙姑微笑："回你该回的地方。大荒山下，青埂峰上，那块石头还在等你。你的使命还没有完成，你还需要将这一切记录下来，让后人知道这段红尘往事。"

宝玉明白了。他最后望了一眼那满天星辰，深深一揖，转身离去。

走出太虚幻境，宝玉看到茫茫大士和渺渺真人正在宫门外等候。大士手中捧着那块通灵玉，玉中闪烁着柔和的光芒，仿佛记录着这一切。

"石头，我们该走了。"茫茫大士道。

宝玉最后回望了一眼太虚幻境，然后跟随大士和真人，向大荒山走去。

正是：

太虚幻境梦中来，警幻宫中点将台。
正册副册情榜列，千红万艳各堪哀。
千红一哭万艳悲，十二金钗化星辉。
此情可待成追忆，只是当时已惘然。

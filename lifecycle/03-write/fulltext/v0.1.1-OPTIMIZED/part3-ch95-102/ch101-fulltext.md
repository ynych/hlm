# 第一百零一回 冷惜春甘伴青灯佛 妙玉终陷淖泥中

且说那贾府败落之后，惜春便起了出家的念头。她本是天生孤介之人，自幼便对那荣华富贵看得淡薄。如今家破人亡，更是心灰意冷。

这日，惜春来到贾母灵前，跪地痛哭。她想起祖母往日的疼爱，想起那大观园中无忧无虑的岁月，心中如刀绞一般。

可她知道，这一切都已成了过去。她不愿像二姐迎春那样，任人摆布，最终落得惨死。她也不愿像三姐探春那样，远嫁他乡，生死未卜。

她只想...只想寻一个清净去处，了此残生。

贾政见她这般模样，心中也是酸楚。他知道惜春自幼便喜佛法，如今家破人亡，想要出家也是情有可原。

他叹道：“四丫头，你...你当真要去？”

惜春跪地，磕头道：“父亲，女儿心意已决。这红尘中...已无女儿可留恋之处。还请父亲...成全...”

贾政望着她那决绝的眼神，知道劝也无用，只得叹道：“罢了...罢了...你去吧...”

惜春叩首谢恩，起身离去。她回到房中，将那平日里画的画儿，写的字儿，都一一收拾起来。

她望着那幅《大观园图》，那是她花了数年心血画成的。画中的一切，都是那么美好，那么繁华。可如今...如今都已成了泡影。

她轻轻叹了一口气，将画卷起，放在一旁。她知道，这一切都将随着她的离去，成为永远的回忆。

第二日，惜春便离了贾府，来到城郊的静月庵。那庵主原是认识她的，见她来意诚恳，便为她剃度。

剃度那日，惜春望着镜中的自己，那三千青丝纷纷落下，心中竟是一片平静。

她想起那判词：“勘破三春景不长，缁衣顿改昔年妆。可怜绣户侯门女，独卧青灯古佛旁。”

如今，她终于是...勘破了。

且说那妙玉，自被乱兵劫走之后，便被卖入凝香院中。那凝香院本是京城中有名的乐坊，专门接待那些达官贵人。

妙玉被卖入此处，自然是宁死不从。老鸨见她这般倔强，心中恼怒，便将她关在一间破屋中，不给饭吃，想要逼她就范。

妙玉在破屋中，望着那破败的墙壁，心中百感交集。她想起昔日栊翠庵中的日子，那时她是何等的清高，何等的洁净。

她想起那年，刘姥姥来栊翠庵品茶，用了那成窑杯。她觉得那杯子脏了，便要扔掉。如今...如今她自己也陷在这淖泥之中，又有谁来救她？

这日，窗外下起了雨。妙玉望着那雨丝，想起那年栊翠庵中的雨。

那时，她与宝玉、黛玉、宝钗一起品茶，那时的她是何等的骄傲。她觉得这世间，没有几人能入得了她的眼。

可如今...如今她却成了这比刘姥姥还不如的人。

她轻轻叹道：“这雨...和那年栊翠庵...一样...”

她闭上眼睛，眼角一滴泪滑落。她知道，自己这一生，都在追求那“洁”字，可最终...最终却落得这般下场。

她手中握着那枚成窑杯的碎片，那是她在被劫时，从栊翠庵中带出来的。她想起当年，她是多么嫌弃刘姥姥用过这杯子，可如今...如今她多想再见刘姥姥一面。

她轻声道：“当日我嫌她脏...今日我比她脏...这...这就是报应吗？”

她不再言语，只是望着那窗外的雨，久久不语。

且说那刘姥姥，自救了巧姐之后，便回了乡下。她听闻京中乱兵抢人，却也无可奈何。

这日，她来到城中，想要打听巧姐的消息。路过凝香院时，听得里头有人哭喊，心中不忍，却也不敢多问。

她哪里知道，那哭喊之人，正是当年栊翠庵中的妙玉？她只觉得那声音有些耳熟，却也想不起在哪里听过。

她摇摇头，叹道：“这世道...真是乱了...”

说完，便转身离去。

妙玉在破屋中，听得外头的脚步声，心中忽生一丝希望。她想要呼喊，却发不出声音。

她知道，不会有人来救她了。她这一生，清高自许，最终却落得这般下场。

她望着手中的成窑杯碎片，轻声道：“欲洁何曾洁...云空未必空...可怜金玉质...终陷淖泥中...”

说完，她闭上眼睛，泪水顺着脸颊滑落。

几日后，妙玉便在这破屋中病逝。无人知晓，无人哀悼。只有那窗外的雨，还在淅淅沥沥地下着，仿佛在为她哭泣。

且说那薛家，自宝钗病逝之后，更是雪上加霜。薛蟠旧案爆发，被官府捉拿归案。薛姨妈急火攻心，一病不起。

薛蝌、薛宝琴兄妹二人，虽尽力周旋，却也无济于事。薛家...终究是彻底败落了。

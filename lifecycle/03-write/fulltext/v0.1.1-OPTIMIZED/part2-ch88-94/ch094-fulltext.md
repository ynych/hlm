# 第九十四回 妙玉被劫落风尘 惜春勘破入空门

且说那妙玉，自入住栊翠庵以来，原是清净修行，不问世事。她生性孤高，自许"槛外人"，不屑与凡俗之人来往。那栊翠庵中，种着几株红梅，冬日里开花，暗香浮动，倒也雅致。

妙玉每日里焚香煮茶，诵经打坐，日子倒也清静。她用的茶具，皆是前朝旧物，珍贵异常。其中有一只成窑杯，是她最为心爱之物，平日里有客来访，她也舍不得拿出来用。

那日刘姥姥来贾府做客，贾母带着众人游大观园，途经栊翠庵。妙玉虽心中不愿，却也不好推辞，只得请众人入庵吃茶。

她将自己常用的绿玉斗斟了茶给宝玉，却将那成窑杯用来给刘姥姥。刘姥姥是乡下人，哪里懂什么茶道，一口气将那茶喝干了，还咂着嘴道："这茶淡了些，不如咱们乡下的浓茶好喝。"

妙玉听了，心中甚是厌恶。待刘姥姥走后，她便将那成窑杯弃之不要，道："这杯子被她用了，脏了，我不能再用了。"

宝玉见了，心中不忍，道："妙玉姐姐，这杯子扔了可惜，不如给了我吧。"

妙玉冷冷道："你要便拿去，反正我是不会再用了。"

宝玉便收了那杯子，转赠给刘姥姥，道："姥姥，这杯子虽不是什么稀罕物，却也值几个钱，您拿去换些银子使吧。"

刘姥姥千恩万谢地收了，却不知这杯子的真正价值。

妙玉望着刘姥姥离去的背影，心中冷笑："这等粗鄙之人，也配用我这茶具？真是脏了这成窑杯。"

她转身回庵，心中仍觉得膈应，仿佛那杯子上沾染了什么污秽之气，久久不能散去。

时光流转，转眼便是数年。这期间，贾府虽渐渐衰微，妙玉却仍守着她的栊翠庵，不肯与外界来往。她每日里焚香煮茶，看那红梅花开花落，倒也自得其乐。她心中始终存着一份清高，认为这俗世的一切都与她无关，她只需守住这一方净土，便可超脱于红尘之外。

那栊翠庵中的红梅，今年开得格外早。妙玉站在梅树下，望着那满树的红梅，心中暗道："这红梅傲雪凌霜，不与凡花争艳，正如我一般。那些俗人哪里懂得这其中的妙处？"

她转身回庵，又取出那套心爱的茶具，细细擦拭。她一边擦拭，一边想起了那只成窑杯，想起了刘姥姥那粗鄙的模样，心中仍是有些膈应。她摇摇头，暗道："这等俗人，不提也罢。"

谁知那乱世之中，哪里有什么清净之地？贾府遭难，大观园中人人自危，妙玉却仍守在栊翠庵中，不肯离去。她心中暗道："我是出家人，不问俗事，他们查抄他们的，与我何干？"

谁知那乱世之中，哪里有什么清净之地？这日夜里，妙玉正在庵中打坐，忽听得外头人声嘈杂，喊杀震天。她心中一惊，忙起身查看，却见一群乱兵闯入庵中，为首的是个满脸横肉的军官。

那军官见了妙玉，眼睛一亮，淫笑道："好一个标致的小尼姑！兄弟们，把她带走！"

妙玉大惊失色，忙道："我乃出家人，你们怎可如此无礼？"

那军官哈哈大笑，道："出家人？如今这世道，谁还管你出不出家？兄弟们已经几个月没碰过女人了，你这小尼姑细皮嫩肉的，正好给兄弟们解解馋！"

妙玉听了，羞愤交加，她想要反抗，却被几个乱兵死死按住。她挣扎着，哭喊着，却无济于事。

就在被拖出栊翠庵的那一刻，妙玉忽然想起了那日的情景。

她想起了刘姥姥端起那只成窑杯时粗糙的手指，想起了刘姥姥那朴实憨厚的笑容，想起了自己当时心中那股强烈的厌恶...

"当日我嫌她脏...今日我比她脏..."

这个念头如闪电般划过她的脑海，她浑身一颤，眼泪夺眶而出。

她想起了自己这些年的清高傲气，想起了自己对俗世之人的鄙夷，想起了那只被她弃如敝履的成窑杯...

原来...原来这才是报应...

她这一生自诩洁净，不与凡俗为伍，可如今呢？她被人如牲畜般拖走，即将堕入那万劫不复的深渊...她比刘姥姥脏，比这世上最脏的人还要脏...

妙玉被带到凝香院，那是京城里颇有名气的乐坊，表面上是歌舞升平之地，实则是个藏污纳垢的所在。老鸨见妙玉姿色出众，又有一股子清冷气质，心中大喜，道："好一个美人儿！从今往后，你就在这凝香院里好生待着，乖乖听话，自然有你的好处。"

妙玉咬牙道："我宁死也不从！"

老鸨冷笑一声，道："死？到了这里，死可没那么容易。你若是不听话，我有的是法子整治你！"

她说着，命人将妙玉关进了一间暗室，每日里只给些残羹剩饭，还时不时用皮鞭抽打。妙玉初时还倔强，不肯屈服，可日子久了，身体也渐渐垮了。

且说那刘姥姥，自从贾府遭难后，心中甚是挂念，便又进城来探望。她听闻贾府被抄，众人离散，心中难过不已。她想起那年进大观园的情形，想起贾母的和蔼，想起凤姐的周到，想起宝玉和姑娘们对她的友善，眼眶不禁湿润了。

她背着一个布包袱，里头装着自家种的一些瓜果蔬菜，还有那只成窑杯。她本想着将这些东西送给贾府的人，虽不值什么钱，却也是她的一番心意。谁知到了贾府门前，只见那大门紧闭，门上的封条已被撕去，却已是人去楼空。

刘姥姥站在门前，愣了半晌，才问隔壁的邻居："这...这府里的人都去哪儿了？"

那邻居叹道："老婆子，你还不知道？这贾府被抄了家，主子们死的死，散的散，这园子也早就不成样子了。"

刘姥姥听了，心中大惊，暗道："这可如何是好？我那巧姐儿...还有那些姑娘们..."

她在城中打听消息，却听说贾府的罪臣家眷，有些被卖入了乐坊。她想起那日去栊翠庵吃茶，妙玉那清高孤傲的模样，心中暗道："那妙玉姑娘是个出家人，想必不会遭难吧？"

刘姥姥听了，心中大惊，暗道："那些姑娘小姐们，都是金枝玉叶，如何受得了这等苦楚？我...我虽无力相救，却也要打听打听，看看有没有能帮上忙的。"

她来到凝香院附近，听人议论说里头新来了一个标致的小尼姑，生得极美，性子却烈，宁死也不肯接客。刘姥姥听了，心中疑惑，暗道："小尼姑？莫非是..."

她想起那日去栊翠庵吃茶的情形，想起妙玉那清高孤傲的模样，心中暗道："不会是她吧...那妙玉姑娘，平日里眼高于顶，连我碰过的杯子都要扔掉...若是她落到了这步田地..."

刘姥姥摇了摇头，叹道："罢了罢了，我哪有本事救她？再说了，那妙玉姑娘想必也不稀罕我救...在她眼里，我这种乡下老婆子，比这乐坊里的窑姐儿也干净不了多少..."

她说着，转身离去，心中却仍有些不安。她走了几步，又回过头来，望着那凝香院的招牌，叹了口气，终究还是走了。

她不知道的是，就在那凝香院的暗室里，妙玉正蜷缩在角落，浑身是伤。她听着外头的喧嚣声，心中一片空白。

她想起了那只成窑杯。

那杯子，如今在哪里？是不是已经被刘姥姥卖了？换了银子，给她的孙儿买了糖吃？

她想起了刘姥姥那粗糙的手指，想起了她喝那杯茶时的憨态...当时她觉得那是莫大的侮辱，可如今想来，那却是这世上最纯真的模样。

"刘姥姥...你...你现在过得可好？"

妙玉喃喃自语，眼泪无声地滑落。

她想起了那日自己嫌弃刘姥姥的情形，想起了自己那种高高在上的姿态...原来，她这一生所谓的"洁净"，不过是自欺欺人罢了。

真正的洁净，不在外表，而在内心。刘姥姥虽粗鄙，却有一颗善良真诚的心；而她妙玉，虽自诩清高，却不过是沽名钓誉之辈...

"当日我嫌她脏...今日我比她脏..."

这句话在她心中反复回响，如刀割般疼痛。

她想要赎罪，可如今她身陷囹圄，连自己都救不了，又谈何赎罪？她想要洁身自好，可这具身子，早已不再属于她自己...

妙玉在凝香院中，宁死不屈，被老鸨百般折磨。她本就体弱，加上心气郁结，不久便染上了重病。

老鸨见她病重，便想将她转手卖掉，可谁愿意买一个病入膏肓的妓女？无奈之下，老鸨只得将她扔到后院的一间破屋里，任她自生自灭。

妙玉躺在那破屋的草堆上，望着屋顶的破洞，看着那一线天光，心中渐渐平静下来。

她知道，自己大限将至。

她不恨刘姥姥，不恨那日来抄家的乱兵，她只恨自己。恨自己当年的傲慢，恨自己当年的无知，恨自己白白浪费了这一世的修行...

她想起了栊翠庵中的红梅，想起了那清香的茶气，想起了宝玉...想起了那些在大观园中度过的日子...

如今想来，那竟是她这一生最快乐的时光。

"欲洁何曾洁，云空未必空。可怜金玉质，终陷淖泥中..."

她喃喃念着这首判词，嘴角浮起一丝苦笑。

原来，一切都是命。她逃不掉，挣不脱，只能认命。

她缓缓闭上眼睛，眼角最后一滴泪滑落，滴在那肮脏的草堆上，瞬间消失不见。

妙玉死了，死在那破屋之中，无人知晓，无人祭奠。她这一生，自诩洁净，最终却死在了最脏的地方。她的尸身被扔到了乱葬岗，与那些无名无姓的孤魂野鬼为伴。

那只成窑杯，她至死也未能再见一面。她不知道的是，那杯子如今正被刘姥姥的小孙子当玩具玩着，早已磕破了一角，却仍是刘姥姥最珍爱的物件。

每当刘姥姥拿起那杯子，她就会想起那日在栊翠庵吃茶的情形，想起妙玉那清冷孤傲的模样，想起自己当时的窘迫...

"那妙玉姑娘，也不知如今怎么样了..."

刘姥姥总是这样念叨着，却不知那人早已香消玉殒。

且说那惜春，自从贾府遭难后，便一直住在府中。她年纪尚小，却早已看破红尘。她见姐姐们一个个命途多舛，见府中一天天衰败下去，心中早已萌生了出家的念头。

惜春素日里寡言少语，却极有主见。她自小便与姐妹们不同，不爱那些胭脂水粉，也不爱那些诗词歌赋，只喜欢一个人静静地待着。她曾在暖香坞里画那大观园图，画了许久，却始终画不完。如今想来，那未完成的画，倒像是这未完成的命运一般。

她想起那年中秋，贾母带着众人赏月，她独坐一旁，看着那轮明月，心中便已有了出世的念头。她想起那年元宵，众人猜灯谜，她的谜底是"佛前海灯"，当时众人只道是不吉，如今想来，竟是应验了。

这日，她正在房中整理衣物，忽听得外头传来一阵哭声。她出去一看，却是丫鬟入画在哭。入画见惜春出来，忙跪下道："姑娘，不好了，听说妙玉师父被人劫走了！"

惜春听了，心中一凛，却也不甚惊讶。她淡淡道："劫走便劫走吧，这世道，哪里还有什么清净之地？"

入画听了，心中害怕，道："姑娘，您...您不害怕吗？"

惜春摇摇头，道："怕有什么用？该来的总会来。你且去收拾东西，咱们也该走了。"

入画不解，问道："去哪里？"

惜春望着窗外那灰蒙蒙的天空，轻声道："去一个...没有人能找到我们的地方。"

那日，她听闻妙玉被乱兵劫走，下落不明，心中越觉这世间凶险，不可久留。她又听闻迎春被孙绍祖虐待致死，心中更是悲痛。

惜春来到妙玉昔日的栊翠庵，只见那庵中一片狼藉，红梅被折，茶具被砸，早已不复当年模样。

她站在庵中，望着那破败的景象，心中一片空灵。

"三姑娘..."

身后传来一个声音，惜春回头，却是贾珍的妻子尤氏。

尤氏哭道："三姑娘，家里如今是撑不下去了，你...你还是早些寻个出路吧..."

惜春摇摇头，道："出路？这世上哪里有出路？大姐姐死了，二姐姐死了，妙玉姐姐也不知所踪...这世道，还有什么可留恋的？"

她说着，转身走出栊翠庵，头也不回。

当夜，惜春便收拾了细软，悄悄离开了贾府。她来到城郊的一座尼姑庵，跪在那住持面前，道："弟子看破红尘，愿皈依佛门，请师父收留。"

那住持见她年幼，劝道："姑娘，出家可不是儿戏，你可想清楚了？"

惜春点点头，道："弟子想清楚了。这世上的一切，都是虚妄。荣华富贵，不过是过眼云烟；情爱纠缠，不过是自寻烦恼。弟子愿独卧青灯古佛旁，了此残生。"

住持见她心意已决，便为她剃度，取法号"静安"。

惜春剃去那一头青丝，望着镜中那光光的头顶，心中竟是一阵轻松。她想起那幅未画完的大观园图，想起那些逝去的繁华，心中再无波澜。

"勘破三春景不长，缁衣顿改昔年妆。可怜绣户侯门女，独卧青灯古佛旁。"

她念着这首判词，嘴角浮起一丝微笑。

从此，这世上再也没有贾府的三姑娘，只有那青灯古佛旁的静安尼姑。

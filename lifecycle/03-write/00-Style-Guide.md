# 文风仿写指南

> 如何让文字"闻起来"像曹雪芹写的。

## 一、词汇层面

### 1.1 高频词库（需背诵）

**语气词**
- 呢、罢、罢哟、罢咧、罢呀
- 么、罢了、就是了
- 示例: "好没意思的呢" "罢了，罢了"

**时间词**
- 才刚、方才、这会子、那阵子
- 半日、好半日、一时、半晌
- 示例: "才刚老太太还问起你呢"

**程度副词**
- 好生、十分、异常、越发的、越觉
- 示例: "好生奇怪" "越发的瘦了"

**行为动词**
- 打谅、忖度、寻趁、混说、混闹
- 示例: "打谅我不知道呢"

### 1.2 避讳与称呼

**自称**
- 我（通用）
- 奴才（仆人对主人）
- 婢子（丫鬟）
- 老身（老年妇女）

**对称**
- 二爷、三姑娘、老太太、太太
- 宝兄弟、林妹妹、宝姐姐
- 姐姐（丫鬟间互称）

**谦称/敬称**
- 令尊、家父、舍妹
- 贵府、寒舍

### 1.3 禁用现代词汇

❌ 绝对禁止:
- 心理、压力、情绪、态度
- 其实（可用"原来""其实则"替代）
- 意义、价值、逻辑
- 然而、但是（可用"只是""却""倒"）

## 二、句式层面

### 2.1 典型句式

**短句为主，间杂长句**
```
那人听了，笑道："原来如此。我说呢，怎么这等面善。"
```

**倒装与省略**
```
"往哪里去了这半日？"（而非"这半日你往哪里去了"）
```

**重复与排比**
```
"亏了，亏了！若不是你来，我几乎忘了。"
```

### 2.2 对话特征

**不直接回答，先绕弯子**
```
宝玉问："妹妹可大安了？"
黛玉不答，只道："你从哪里来？"
```

**话中有话，意在言外**
```
宝钗笑道："我来了这么些年，留神看起来，二嫂子凭她怎么巧，巧不过老太太去。"
（表面夸贾母，实则刺凤姐）
```

**打断与抢白**
```
凤姐道："你——"
话未说完，平儿进来道："奶奶，姨太太那边打发人来了。"
```

## 三、叙事层面

### 3.1 分段风格（重要）

**必须模仿前80回的分段特点**：

1. **短段落为主**
   - 叙述段落控制在3-4行以内
   - 一个动作、一个场景切换就换段
   - 避免大段大段的叙述堆砌

2. **对话独立成段**
   ```
   探春笑道："二哥哥今日好大的架子。"
   
   宝玉忙道："才刚在老太太那里，耽搁了一会子。"
   
   李纨笑道："来了就好，快坐下罢。"
   ```

3. **叙述与对话频繁交替**
   - 不要连续多段都是叙述
   - 不要连续多段都是对话
   - 保持"叙述-对话-叙述-对话"的节奏

4. **视角随人物移动**
   ```
   宝玉回到怡红院。
   
   袭人迎上来，见他神色不对。
   
   "二爷怎么了？"袭人问道。
   
   宝玉摇头，往榻上一躺。
   ```

**错误示例**（段落过长）：
```
探春发起钓鱼之会，邀约宝玉、李纨、岫烟同至藕香榭。四美各自下钓，闲话家常。探春说起近日园中花木凋零，不如往年繁盛；李纨笑谈贾兰读书进益；岫烟言语不多，神色间却有几分忧虑。宝玉虽在钓鱼，却心不在焉，频频望向潇湘馆方向。
```

**正确示例**（短段落、频繁换行）：
```
探春发起钓鱼之会，邀约宝玉、李纨、岫烟同至藕香榭。

四美各自下钓，闲话家常。探春说起近日园中花木凋零，不如往年繁盛。李纨笑谈贾兰读书进益。岫烟言语不多，神色间却有几分忧虑。

宝玉虽在钓鱼，却心不在焉，频频望向潇湘馆方向。
```

### 3.2 视角控制

**全知视角，但假装受限**
- 作者知道一切，但故意留白
- 通过人物的眼睛看，不直接下判断

**示例**:
```
宝玉听了，心下疑惑，不知是何缘故。欲要询问，又怕唐突，只得闷闷的回来。
（不直接说"宝玉不知道原因"，而是写他的动作与心理）
```

### 3.2 节奏控制

**大事缓写，小事细描**
- 元妃省亲：宏大场面，寥寥数笔
- 黛玉葬花：一人一事，浓墨重彩

**动静结合**
- 激烈冲突后必有闲笔
- 热闹场面中插入冷观察

### 3.3 环境描写

**以人带景，情景交融**
```
只见贾宝玉穿着家常衣服，趿着鞋，在那桃花树底下呆呆的站着。一阵风过，吹下许多花瓣来，落得满身满鬓。
```

**善用时辰与天气**
- 春日、夏夜、秋雨、冬雪
- 黄昏、夜半、黎明

## 四、诗词创作

### 4.1 仿七绝格式

**平仄格式**（首句入韵）:
平起: 平平仄仄仄平平，仄仄平平仄仄平。
仄仄平平平仄仄，平平仄仄仄平平。

**押韵**: 平水韵，一韵到底

### 4.2 仿《红楼梦》诗词风格

**婉约为主，不直露**
```
花魂默默无情绪，鸟梦痴痴何处惊。
（林黛玉《葬花吟》风格）
```

**用典含蓄**
- 不直接说"孤独"，说"埋香冢飞燕泣残红"
- 不直接说"富贵无常"，说"纵是齐眉举案，到底意难平"

## 五、自检清单

写完后逐条检查:

- [ ] 有无现代词汇混入？
- [ ] 对话是否符合人物身份？
- [ ] 是否用了"我"而不是"奴才/婢子"（仆人）？
- [ ] 句式是否长短交替？
- [ ] 有没有直接写心理，而不是通过动作表情暗示？
- [ ] 场景转换是否自然？
- [ ] 时间词是否准确（才刚/这会子/半日）？
- [ ] 有没有"但是/然而"等现代连词？

## 六、仿写练习

### 练习1: 改写现代句子

原文: "宝玉听了，心里很不舒服，但又不能说什么，只能默默离开。"

仿写: "宝玉听了，心下不快，只是不好发作，搭讪着走了出来。"

### 练习2: 写一段对话

场景: 黛玉病了，宝钗来看望。

（参考答案格式）:
```
宝钗掀帘子进来，笑道："好些了没有？我瞧瞧。"
黛玉欠身道："劳姐姐记挂着，才好些呢。"
宝钗坐在床沿上，拉着她的手道："我瞧你脸上还是不好。这病，急不得的。"
黛玉叹道："我也知道，只是心里烦得很。"
```

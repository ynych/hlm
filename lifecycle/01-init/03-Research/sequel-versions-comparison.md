# 红楼梦续书版本详细对比

## 一、程高本（程伟元、高鹗续）

### 基本信息
- **成书时间**：1791-1792年
- **回数**：120回（前80回曹雪芹，后40回程高续）
- **特点**：最早刊行的全本，影响最大

### 后40回主要情节
1. **调包计** - 宝玉娶宝钗，黛玉焚稿而亡
2. **贾府复兴** - 贾家虽被抄，后复职，宝玉中举
3. **宝玉出家** - 中举后出家
4. **兰桂齐芳** - 贾兰、贾桂（宝玉子）同时中举

### 与前80回的差异
| 方面 | 前80回 | 程高本后40回 |
|------|--------|-------------|
| 结局基调 | 悲剧 | 悲剧中带希望 |
| 贾府命运 | 彻底败落 | 复兴有望 |
| 宝玉结局 | 悬崖撒手 | 中举后出家 |
| 黛玉之死 | 泪尽而亡 | 焚稿而亡 |
| 宝钗结局 | 凄凉 | 生子守寡 |

### 评价
- **优点**：使《红楼梦》成为完璧，流传广泛
- **缺点**：与曹雪芹原意有出入，部分情节不合理

## 二、癸酉本（吴梅村/何人所著，存疑）

### 基本信息
- **成书时间**：不详（号称明末清初，实际存疑）
- **回数**：108回（后28回）
- **特点**：情节离奇，与前80回伏笔对应较多

### 后28回主要情节
1. **元春之死** - 政治斗争被勒死
2. **黛玉之死** - 自缢而亡
3. **宝玉被骗婚** - 娶宝钗，黛玉死
4. **贾府抄家** - 彻底败落
5. **宝钗再嫁** - 嫁贾雨村
6. **凤姐被休** - 扫雪拾玉，病逝
7. **湘云为妓** - 沦为船妓
8. **妙玉被劫** - 沦落风尘
9. **宝玉出家** - 悬崖撒手
10. **情榜揭晓** - 太虚幻境点化

### 与脂批对应
| 脂批 | 癸酉本落实 |
|------|-----------|
| 玉带林中挂 | 黛玉自缢 |
| 金簪雪里埋 | 宝钗病逝雪中 |
| 一从二令三人木 | 凤姐被休 |
| 扫雪拾玉 | 凤姐拾得通灵宝玉 |
| 狱神庙慰宝玉 | 茜雪、红玉探望 |
| 湘江水逝 | 湘云病逝/投江 |
| 终陷淖泥中 | 妙玉被劫 |

### 争议点
- **真实性存疑**：版本来源不明，可能是现代人伪托
- **文笔粗劣**：与前80回文笔差距大
- **情节离奇**：部分情节过于狗血

### 参考价值
- 提供了另一种结局思路
- 落实了较多脂批线索
- 为研究《红楼梦》结局提供参考

## 三、1987版电视剧《红楼梦》

### 基本信息
- **拍摄时间**：1984-1987年
- **编剧**：周雷、刘耕路、周岭
- **回数**：36集

### 后6集主要情节
1. **元春薨逝** - 贾府失去靠山
2. **贾府抄家** - 彻底败落
3. **宝玉被骗婚** - 娶宝钗，黛玉焚稿而亡
4. **凤姐被休** - 病逝
5. **湘云沦落** - 与宝玉重逢
6. **宝玉出家** - 悬崖撒手

### 特点
- 综合了程高本和部分红学研究成果
- 结局基调悲凉，符合曹雪芹原意
- 部分情节参考了脂批和早期抄本

## 六、《癸酉续红》的取舍

### 参考来源
1. **癸酉本架构** - 后28回整体框架
2. **脂批线索** - 人物结局的重要依据
3. **程高本** - 部分情节处理参考
4. **87版** - 人物性格保持参考

### 改编原则
1. 以脂批为优先依据
2. 保持悲剧基调
3. 人物性格不OOC
4. 情节合理，淡化离奇巧合

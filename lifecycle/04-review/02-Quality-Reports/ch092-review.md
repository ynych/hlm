## 关键问题（需修正）

### P0（致命）
- [ ] **问题1**：宝玉对宝钗的态度过于激烈 → 建议：前80回宝玉虽不爱宝钗，但对其始终客气尊重（"见了姐姐就忘了妹妹"是玩笑话）。狱中言语如"你爱的从来就不是我，你爱的是那些功名利禄"过于直白伤人，应更含蓄，符合宝玉"不忍心伤害人"的性格。可改为宝玉沉默回避，或轻声叹息，而非当面指责。

- [ ] **问题2**：宝钗语言过于现代 → 建议：宝钗劝宝玉时使用了"识时务""自保""能力"等偏现代词汇，且说理过于逻辑化。应使用更多红楼式表达，如"这会子""好歹""将息"等，语气应更委婉含蓄，符合"停机德"的含蓄劝谏风格。

### P1（重要）
- [ ] **问题1**：凤姐入股布庄的情节过于顺利 → 建议：清代女子被休后独立经营商业不合礼制。可增加阻碍（如周老板犹豫、需借平儿名义、或凤姐只能做幕后出主意），以符合历史背景。

- [ ] **问题2**：巧姐被卖的价格偏低 → 建议：五两银子买一个"钦犯之后"过于便宜，即使是"烫手山芋"也不至如此。可改为牙婆故意压价，或刘嬷嬷急于脱手而贱卖，使情节更合理。

- [ ] **问题3**：部分现代词汇需替换 → 建议：全文有多处现代用词，如"自身难保""落实""分析"等，应替换为红楼常用表达。

### P2（优化）
- [ ] **问题1**：回末诗工整度不足 → 建议：回末诗部分句子对仗不工整（如"街头巧遇钱婆子"对"知命强英雄本色"），可精炼为七言律诗或回目常用的诗体。

- [ ] **问题2**：部分场景描写可更简练 → 建议：如开篇雪景描写稍长，可精简；宝钗探狱部分心理描写可更含蓄，多用白描少做解释。

- [ ] **问题3**：平儿角色可更立体 → 建议：平儿对凤姐的忠义无问题，但可增加其内心活动或细节动作，使其形象更丰满。

1. **语气词优化**：增加"呢、罢、么、么样、才好"等红楼语气词
2. **词汇替换**："自身难保"→"泥菩萨过江"；"分析"→"盘算"；"能力"→"本事"
3. **宝钗台词重写**：改为更含蓄的劝谏，减少逻辑说教
4. **宝玉态度软化**：狱中对话更含蓄，体现其不忍伤人的性格
5. **凤姐入股增加阻碍**：增加周老板的犹豫、时代限制
6. **巧姐价格调整**：五两→八两或十两，或增加讨价还价过程
7. **回末诗修改**：精炼为工整的七言诗

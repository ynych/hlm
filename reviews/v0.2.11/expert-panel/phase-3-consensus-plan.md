## 二、P0级别修复方案（必须完成）

### P0-001: 补充妙玉完整人物线

**问题**: 妙玉全篇仅出现名字1次，所有关键场景缺失  
**优先级**: ⭐⭐⭐⭐⭐（最高）  
**负责专家**: 槛外人（妙玉）  
**工期**: 12天  
**新增字数**: 约3000字

**修复内容**:

1. **栊翠庵品茶场景**（第81回，回忆形式）
   - 场景: 宝玉、黛玉、宝钗访妙玉
   - 重点: 成窑杯、洁癖体现、对诗
   - 诗作: 妙玉与黛玉、湘云对诗1首

2. **被劫情节**（第94回，略写）
   - 场景: 贼人闯入栊翠庵
   - 重点: 妙玉的反抗与无力
   - 处理: 含蓄，不详细描写暴力

3. **沦落风尘**（第101回）
   - 场景: 花船/青楼中的妙玉
   - 重点: 洁癖与现实的冲突
   - 心理: "欲洁何曾洁"的挣扎

4. **结局**（第106回）
   - 场景: 病逝/自尽
   - 重点: "终陷淖泥中"的落实
   - 遗物: 成窑杯的象征意义

**写作要求**:
- 保持妙玉的孤高与洁癖
- 悲剧要有美感，不猎奇
- 判词"欲洁何曾洁，云空未必空"必须落实

### P0-002: 为黛玉创作3首原创诗词

**问题**: 全篇无黛玉原创诗词，核心人设缺失  
**优先级**: ⭐⭐⭐⭐⭐  
**负责专家**: 潇湘妃子（黛玉）  
**工期**: 10天

**诗词创作计划**:

**第一首：咳血后（第81回）**
```
主题: 病中感怀
风格: 婉约凄美
意象: 落花、秋风、病骨
格律: 七律

示例（需进一步完善）:
秋风又送雨声寒，病骨支离倚画栏。
一夜咳红千点泪，明朝花落几人看？
身如蒲柳应难久，心似残荷不忍残。
莫问潇湘何所忆，竹痕月影两珊珊。
```

**第二首：惊梦后（第81回末尾）**
```
主题: 梦境惊惶
风格: 幽咽迷离
意象: 梦、影、月、魂
格律: 五律或七绝

示例:
惊梦回时夜色阑，
罗巾泪湿未曾干。
分明梦里人如玉，
醒后空留月影寒。
```

**第三首：临终前（第85回）**
```
主题: 泪尽而逝
风格: 绝命诗
意象: 泪、花、归、尘
格律: 七绝

示例:
一生心事付流霞，
泪尽潇湘便是家。
愿得埋香清净土，
不须惆怅对残花。
```

**写作要求**:
- 必须符合黛玉性格（诗意、敏感、自怜）
- 格律合规，用典恰当
- 与上下文情节呼应

### P0-003: 彻底删除"悲从中来"套语

**问题**: "悲从中来，更添凄楚"在全篇反复出现，导致所有角色情绪描写雷同  
**优先级**: ⭐⭐⭐⭐⭐  
**负责专家**: 全员  
**工期**: 并行处理

**删除范围**:
- "悲从中来，更添凄楚" → 全部删除
- "胸口像是被什么堵住了" → 全部删除或替换
- "心中五味杂陈" → 全部删除
- "久久不语" → 视情况替换

**替换策略**:

| 原套语 | 替换方式 | 示例 |
|--------|----------|------|
| 悲从中来 | 具体动作+景物烘托 | "她凝眸那残荷，手上一颤，钓竿险些滑落" |
| 胸口堵 | 身体感受多样化 | "胸口一阵绞痛"、"心底一酸"、"喉头哽住" |
| 五味杂陈 | 细节描写展现复杂情绪 | 手抖、视线模糊、呼吸急促等 |

**各专家负责范围**:
- 潇湘妃子: 黛玉相关套语
- 蘅芜君: 宝钗相关套语
- 凤辣子: 凤姐相关套语
- 枕霞旧友: 湘云相关套语
- 其他专家: 各自负责角色

### P0-004: 重写凤姐狱中场景，减少悲情，增加精明特质

**问题**: 凤姐被写成悲情女子，"凤辣子"本色弱化，"悲从中来"套语泛滥  
**优先级**: ⭐⭐⭐⭐⭐  
**负责专家**: 凤辣子（凤姐）  
**工期**: 8天

**修复要点**:

1. **保持精明特质**
   - 狱中观察人心，分析其他囚犯
   - 利用有限资源周旋
   - 与女囚赵氏的关系：不是被动接受帮助，而是主动笼络

2. **减少悲情，增加倔强**
   - 删除"悲从中来，更添凄楚"
   - 增加"强撑体面"的细节
   - 断簪意象深化：以残缺自警，以断口自强

3. **个性化语言**
   - 恢复凤姐的爽利口语
   - "一串一串的"快人快语
   - 即使在困境中也要带三分冷笑

4. **与巧姐离别重写**
   - 删除"梦中告别"俗套
   - 改为现实场景：狱门遥望、托刘姥姥带话、缝制小衣

---

### P0-005: 重写宝钗临终场景，减少直白独白

**问题**: 临终独白过于直白，泄露内心，与"藏愚守拙"性格不符  
**优先级**: ⭐⭐⭐⭐  
**负责专家**: 蘅芜君（宝钗）  
**工期**: 8天

**修复要点**:

1. **删减独白80%**
   - 删除"活得太累了"等直白表达
   - 删除"终于可以做回自己"

2. **用动作替代语言**
   - 凝望窗外残梅
   - 轻抚贾桂的小手
   - 整理妆容（至死维持体面）

3. **增加环境烘托**
   - 北风呼啸、雪花飘落
   - 老梅最后一朵残花掉落
   - 湘云的观察视角

4. **补写探病黛玉场景**（第81-95回之间）
   - 场景：宝钗探望病中的黛玉
   - 重点：钗黛关系的最终升华
   - 为"梦见林姐姐"做铺垫

# 第92回 贾环勾结外匪段落修复

## 【修改前】

```
且说贾府抄家之后，家中一片混乱。那贾环趁着府中无人管束，竟与外面的土匪勾结，想要趁乱打劫。

这日，贾环在赌场中结识了几个外匪，那些人见他穿着华丽，便起了歹心。贾环却不知，只道是遇到了贵人，便将自己的身份和盘托出。

他道："我乃贾府的三爷，如今家中虽遭了难，但还有些家底。你们若有胆量，随我去府中，金银财宝任你们取！"

那几个外匪听了，心中大喜，便与贾环约定，夜间动手。

[...中间宝钗难产段落...]

且说那日夜间，贾环带着外匪潜入府中，却被官兵发现，一番厮杀，那些外匪死的死，逃的逃，贾环也被捉拿归案。
```

## 【修改后】

```
且说贾府抄家之后，家中一片混乱。那贾环趁着府中无人管束，竟与外面的土匪勾结，想要趁乱打劫。

这日，贾环又在那地下赌场里输得精光。他本就因"庶出"身份在府中受尽冷眼，如今家道中落，连那些下人都敢给他脸色看。他攥着最后几个铜钱，眼珠子通红，却听见旁边有人笑道："这位公子，手气不佳啊。"

贾环抬头，只见一个满脸横肉的汉子正笑眯眯地瞧着他。那人身旁还坐着个瘦猴似的男人，三角眼里闪着精光。这二人，正是京城地面上有名的"黑风寨"探子——横肉汉子姓马，人称马三爷；瘦猴姓苟，外号"狗头军师"。

"关你屁事！"贾环没好气地回道。

马三爷不恼，反而凑近了，从怀里摸出一只翡翠扳指——那水头，那成色，竟比贾环见过的还要好上几分。马三爷压低了声音："公子是有身份的人，犯不着跟这儿耗。兄弟手头有个买卖，若是公子有兴趣，别说回本，便是翻个十倍，也是手到擒来。"

贾环心中一动，却还有几分警惕："什么买卖？"

那狗头军师苟先生凑上来，阴恻恻地笑道："公子看着面熟，莫不是...贾府的三爷？"

贾环一怔，正欲否认，马三爷已一把揽住他肩膀："三爷莫怕！咱们兄弟最敬重的，就是您这样的人物。想当年，贾府是何等风光？如今那些当官的欺负到门上，咱们兄弟看不过眼！"

这话正戳中贾环的痛处。他想起那日被官兵羞辱的情形，想起王夫人平日里看他的眼神，想起探春那番"庶出"的诛心之论...他猛地灌下一碗烧刀子，骂道："那些王八蛋！平日里拿我当狗，如今...如今..."

马三爷趁热打铁："三爷，咱们明人不说暗话。咱们兄弟是做什么的，您心里清楚。只要您给指条明路，贾府里哪些是值钱的物件，藏在哪儿，剩下的，咱们兄弟来办！到时候，咱们对半分，您看如何？"

贾环盯着那只翡翠扳指，呼吸粗重。他想起自己在府中的处境——凤姐儿防贼似的防他，宝玉从来不正眼瞧他，连那些丫鬟婆子，都敢在背后嘀咕"上不得台面的东西"。

可他忽然又想起一事。

那是一年前的冬日，他母亲周姨娘病重，想要一副好药。他去找王夫人，却被挡在门外。他去找贾政，贾政只说"妇人之事，不必烦我"。等他凑够了钱，周姨娘已经...已经...

贾环攥紧了拳头，指甲掐进肉里。

母亲临死前拉着他的手，浑浊的眼里全是泪："环儿...别恨...别恨这个府...娘这一辈子...命该如此..."

他恨！他怎能不恨！

可就在这一刻，他忽然想起巧姐。

那小丫头，前几日还怯生生地叫他"三叔"，往他手里塞了一块糖，说"三叔吃，甜的"。那是他在这府中，为数不多感受到的一丝暖意。

他猛地抬起头，看向马三爷，声音有些发颤："府里...府里还有孩子..."

马三爷与狗头军师交换了一个眼神，随即拍着胸脯保证："三爷放心！咱们兄弟是求财，不是害命！只要您配合，咱们保证不伤一人！"

贾环盯着马三爷的眼睛，想要从中看出真假。可那双眼太深，像两口枯井，什么也瞧不见。

他又灌下一碗酒。烧刀子火辣辣地烧着喉咙，烧得他眼眶发热。

"好..."他听见自己说，声音像是从很远的地方传来，"今夜三更...府里值夜的家丁我熟...我带你们从西角门进..."

马三爷大喜，当即与贾环约定细节。贾环浑浑噩噩地听着，脑子里却乱成一团。

他对自己说：我只是求财...我只是想拿回属于我的东西...

可心底深处，有一个声音在问：贾环，你真的要这样吗？你要变成他们口中"上不得台面"的样子吗？

他没有答案。

[...中间宝钗难产段落保持不变...]

且说那日夜间，贾环带着外匪潜入府中，却被官兵发现，一番厮杀，那些外匪死的死，逃的逃，贾环也被捉拿归案。

原来，那马三爷本就与官府有些瓜葛，此番不过是借贾环之手探路。贾环被擒时，嘴里还念叨着："那孩子...那孩子没事吧..."

押他的官兵骂道："死到临头还管什么孩子！"

贾环惨笑一声，不再言语。他想起母亲临终的眼神，想起巧姐塞给他的那块糖，忽然觉得，这一生，真是...真是...
```

## 修改说明

1. **赌场结识场景**：补充了马三爷和狗头军师两个具体人物，通过翡翠扳指、烧刀子等细节营造真实感，展现了"钓鱼"式的话术过程。

2. **内心挣扎**：在贾环即将答应时，插入了他对周姨娘之死的回忆（被王夫人挡在门外、贾政冷漠、母亲临终嘱托），使恨意有了具体来源。

3. **巧姐的怜悯**：关键时刻插入巧姐塞糖的细节，与周姨娘之死形成情感对冲，让贾环的犹豫显得真实可信。

4. **背叛前的犹豫**：通过"府里还有孩子"的反复确认、马三爷的虚假保证、贾环自我欺骗的心理活动，展现其复杂性。

5. **反派不脸谱化**：马三爷表面豪爽实则阴险，狗头军师阴恻恻的，两人配合演戏；贾环不是天生的恶人，而是在屈辱、贪婪、良知之间挣扎的可怜人。

6. **结局呼应**：被捕时仍挂念"那孩子"，与前面巧姐的细节形成呼应，暗示贾环最终的人性未泯。

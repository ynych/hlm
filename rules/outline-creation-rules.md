# 《癸酉续红》大纲生成规则

**版本**: v0.1.0  
**适用**: 第81-108回大纲设计  
**目标**: 生成符合癸酉本架构、红学合规的大纲

---

## 1. 输入要求

### 必需输入
- 前80回人物关系图
- 金陵十二钗判词
- 脂砚斋批语关键线索
- 癸酉本骨架参考（可选）

### 可选输入
- 特定回目的特殊要求
- 需要重点落实的判词
- 新增人物设定（如贾桂）

---

## 2. 结构设计规则

### 2.1 五部结构
```
Part 1 (81-87回): 渐衰期 - 贾府表面繁荣下的衰败征兆
Part 2 (88-94回): 崩塌期 - 元妃薨、抄家、主要人物离散
Part 3 (95-102回): 离散期 - 人物相继离世，贾府彻底败落
Part 4 (103-108回): 流浪期+情榜期 - 宝玉流浪、悟道、情榜揭晓
```

### 2.2 回目设计
- 格式: 八字对仗回目
- 风格: 仿曹雪芹回目（前80回）
- 功能: 概括本回核心情节，含伏笔暗示

### 2.3 高潮分布
| 回目 | 高潮事件 | 类型 |
|------|----------|------|
| 85回 | 黛玉之死 | 情感高潮 |
| 89回 | 元妃薨+抄家 | 政治高潮 |
| 97回 | 宝钗之死 | 情感高潮 |
| 104回 | 悬崖撒手 | 哲学高潮 |
| 106回 | 情榜揭晓 | 结构高潮 |
| 108回 | 石头记成 | 结局高潮 |

---

## 3. 人物命运设计规则

### 3.1 判词落实（必须100%落实）

#### 正册十二钗
| 人物 | 判词关键句 | 落实方式 |
|------|-----------|----------|
| 黛玉 | 玉带林中挂 | 泪尽咳血，白绫挂梅枝 |
| 宝钗 | 金簪雪里埋 | 病逝破庙，金簪雪中埋 |
| 元春 | 虎兕相逢大梦归 | 薨逝（政治斗争） |
| 探春 | 千里东风一梦遥 | 远嫁海疆 |
| 湘云 | 湘江水逝楚云飞 | 歌妓沦落，贫病而逝 |
| 妙玉 | 终陷淖泥中 | 被劫后病逝（宁死不屈） |
| 迎春 | 一载赴黄粱 | 被孙绍祖虐死 |
| 惜春 | 独卧青灯古佛旁 | 出家为尼 |
| 凤姐 | 一从二令三人木 | 被休，狱中病逝 |
| 巧姐 | 偶因济刘氏 | 刘姥姥救出，嫁板儿 |
| 李纨 | 枉与他人作笑谈 | 守寡，贾兰中举后逝 |

### 3.2 人物一致性
- 性格与前80回一致
- 人物关系正确（辈分、亲属）
- 新增人物（贾桂）需合理融入

---

## 4. 历史准确性规则

### 4.1 制度礼仪
- 官职机构符合清代（无锦衣卫等明代设置）
- 婚丧嫁娶礼仪正确
- 抄家流程符合清代制度
- 称谓礼节准确

### 4.2 时间线
- 28回跨度3-4年
- 季节/节日与时间线匹配
- 人物年龄变化合理
- 事件顺序符合因果

---

## 5. 文学性规则

### 5.1 悲剧美学
- 悼明挽歌主题贯穿
- 末世隐喻每回1-2处
- 草蛇灰线、伏脉千里

### 5.2 意象运用
- 核心意象: 玉、雪、水、花、梦、月
- 意象与人物命运对应
- 前后回意象呼应

### 5.3 创新边界
- 可与癸酉本有差异（学术派立场）
- 但判词必须落实
- 悲剧结局不可更改

---

## 6. 输出格式

### 6.1 单回大纲格式
```markdown
## 第XX回 [回目标题]

### 情节摘要
[200字以内，概括本回核心情节]

### 关键场景
1. [场景一]: [简述]
2. [场景二]: [简述]
3. [场景三]: [简述]

### 判词落实
- [判词]: [落实方式]

### 人物出场
- 主要人物: [列表]
- 次要人物: [列表]

### 回末诗
[七律，平仄合规]
```

### 6.2 完整大纲格式
```markdown
# 《癸酉续红》大纲 vX.X.X

## 概述
[项目概述，200字]

## 核心设定
[双主题、三大改造等]

## 回目列表
| 回目 | 标题 | 关键事件 |

## 详细大纲
[28回详细大纲]

## 附录
- 人物命运对照表
- 判词落实清单
- 历史准确性说明
```

## 7. 自检清单

### 7.1 生成前自检
- [ ] 输入材料完整
- [ ] 判词清单已准备
- [ ] 历史参考材料已准备

### 7.2 生成中自检
- [ ] 每回落实至少1个判词/伏笔
- [ ] 人物性格与前80回一致
- [ ] 历史制度准确

### 7.3 生成后自检
- [ ] 28回完整
- [ ] 五部结构清晰
- [ ] 判词100%落实
- [ ] 无P0级问题

---

## 8. 质量验收标准

### 8.1 评审维度
| 维度 | 权重 | 达标线 |
|------|------|--------|
| 人物关系与一致性 | 20% | 17/20 |
| 时间线与逻辑 | 20% | 17/20 |
| 判词与脂批落实 | 20% | 18/20 |
| 历史准确性 | 15% | 12/15 |
| 叙事结构与节奏 | 15% | 12/15 |
| 文学性与创新性 | 10% | 8/10 |
| **总分** | **100%** | **≥90/100** |

### 8.2 问题分级
- **P0**: 人物关系矛盾、判词未落实、历史常识错误
- **P1**: 性格轻微OOC、时间跳跃不合理、判词落实不够精准
- **P2**: 节奏调整、细节丰富、文学性提升

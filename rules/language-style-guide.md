# Redream 语言风格规范 v1.0

**适用范围**: 《癸酉续红》全文创作与修订  
**目标**: 贴合《红楼梦》原著风格（清代白话、含蓄典雅）

---

## 一、语言基调

### 1.1 半文半白，清代白话

**✅ 正确示例**:
- 宝玉胸口烦闷，便往潇湘馆来
- 黛玉倚在榻上，面色苍白如纸
- 探春听了，手上一顿，却笑道

**❌ 错误示例**:
- 宝玉心中很难受（太现代）
- 黛玉躺在床上（太口语）
- 探春愣了一下（太直白）

### 1.2 句式特征

| 特征 | 说明 | 示例 |
|:-----|:-----|:-----|
| 短句为主 | 避免复杂复句 | "她望向窗外。月色惨白。" |
| 分句叙述 | 不用"虽然...但是" | "那叶子落得比往年早。这园子怕是要变了。" |
| 倒装句式 | 贴合古文习惯 | "那宝钗，穿的是玫瑰紫比肩褂" |

## 二、高频词汇控制

### 2.1 禁止/限制词汇表

| 词汇 | 限制 | 替代方案 | 优先级 |
|:-----|:-----|:---------|:------:|
| 躺在床上 | ❌ 禁用 | 卧于榻上、倚在榻上、躺在枕上 | P0 |
| 凝眸 | ≤2次/回 | 注目、望、瞧 | P1 |
| 瞻望 | ≤1次/回 | 望去、远眺 | P1 |
| 凭窗而望 | ≤1次/回 | 倚窗望去、临窗远眺 | P1 |
| 凝视 | ≤3次/回 | 望着、注视、看着 | P1 |
| 心中... | ≤5次/回 | 胸口、心底、心内 | P2 |
| 非常 | ❌ 禁用 | 甚是、极、颇 | P2 |
| 很 | ❌ 禁用 | 甚、极、颇 | P2 |

### 2.2 多样化替换表

**"看"的替代**:
```
望 → 远眺、凝望、望去
瞧 → 瞧见、瞧着、瞧去
看 → 看见、看着、看去
瞥 → 瞥见、一瞥
观 → 观望、凝视（慎用）
```

**"想"的替代**:
```
思及 → 思及往事、思及前途
念起 → 念起那人、念起往日
想到 → 想到此处、想到那人
忆起 → 忆起往事、忆起旧情
寻思 → 寻思半晌、暗自寻思
```

## 三、称谓规范

### 3.1 贾府内尊卑称呼

| 对象 | 正确称呼 | 错误示例 |
|:-----|:---------|:---------|
| 贾母 | 老太太、老祖宗 | 贾母（太正式） |
| 王夫人 | 太太 | 夫人 |
| 王熙凤 | 凤姐、琏二奶奶 | 熙凤 |
| 宝玉 | 二爷、宝二爷 | 宝玉（平辈可用） |
| 黛玉 | 林姑娘、姑娘 | 黛玉（下人不可用） |
| 宝钗 | 宝姑娘、宝姐姐 | 宝钗 |
| 袭人 | 袭人（宝玉用）、袭人姐姐（小丫头用） | - |
| 紫鹃 | 紫鹃（黛玉用）、紫鹃姐姐 | - |

### 3.2 自称与谦称

| 场景 | 正确 | 错误 |
|:-----|:-----|:-----|
| 下人对主子 | 奴才、婢子、小的 | 我 |
| 主子对下人 | 我 | 本姑娘（太傲慢） |
| 黛玉自谦 | 我、颦儿 | 本小姐 |
| 宝钗自称 | 我、宝丫头（长辈前） | - |

## 四、器物与礼仪

### 4.1 器物名称

**服饰**:
- ✅ 绫子、缎子、锦缎、绸缎
- ✅ 比肩褂、坎肩、斗篷、披风
- ✅ 金钗、银簪、玉镯、戒指
- ❌ 衣服（太泛）、外套（太现代）

**家具陈设**:
- ✅ 绣墩、榻、案、几
- ✅ 帐子、帘子、屏风
- ❌ 椅子（用绣墩/凳子）、桌子（用案/几）

**饮食**:
- ✅ 龙井、铁观音、普洱
- ✅ 桂花糕、栗粉糕、藕粉糕
- ✅ 松瓤鹅油卷、奶酪
- ❌ 茶点（太泛）、点心（可用但需具体）

### 4.2 礼仪规范

| 场景 | 正确描述 | 错误 |
|:-----|:---------|:-----|
| 见面礼 | 福了一福、请安 | 鞠躬（太现代） |
| 告别 | 告退、请辞 | 走了（太口语） |
| 感谢 | 多谢、承情 | 谢谢（太现代） |
| 道歉 | 恕罪、得罪 | 对不起（太现代） |

## 五、情感表达

### 5.1 含蓄暗示原则

**✅ 正确**: 通过动作、景物暗示情感
- 黛玉帕子上隐约有几点暗红
- 那落叶在秋风中翻滚，如泣如诉
- 她只是淡淡一笑，取过帕子拭了拭唇角

**❌ 错误**: 直接描述心理状态
- 黛玉心里很难过
- 宝玉感到非常伤心
- 探春觉得非常愤怒

### 5.2 情感词汇替换

| 现代词 | 古典替代 |
|:-------|:---------|
| 难过 | 胸口烦闷、心内凄然、悲从中来 |
| 伤心 | 泪如雨下、肝肠寸断（慎用）、凄然 |
| 生气 | 胸口大怒、面色铁青、冷笑道 |
| 害怕 | 心底发寒、脸色骤变、心内一惊 |
| 高兴 | 心中欢喜、面上带笑、心内一喜 |

## 六、章回体格式

### 6.1 回目标题

**格式**: `# 第八十一回 占旺相四美钓游鱼 奉严词两番入家塾`

**要求**:
- 对仗工整（七字+七字）
- 概括本回核心事件
- 符合原著回目风格

### 6.2 回末诗

**格式**:
```markdown
**【回末诗】**

> 藕香榭外水潺潺，旺相难占运已残。
> 残荷听雨声声慢，一叶飘零一叶寒。
```

**要求**:
- 七言绝句（四句）
- 对仗工整，意境贴合
- 概括本回主旨或预示下回

### 6.3 开篇句式

**✅ 正确**:
- 且说那日...
- 话说...
- 且说...

**❌ 错误**:
- 有一天...（太口语）
- 那天...（太口语）

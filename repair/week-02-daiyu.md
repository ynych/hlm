## Day 1-3: 三首原创诗词创作

### 《病中吟》（第81回咳血后）

**插入位置**: 第81回 咳血场景后，紫鹃劝慰前

```markdown
那一夜，黛玉又咳了半宿。

紫鹃端着痰盂进来，借着昏黄的烛光一瞧，
脸色骤变——那帕子上，几点暗红，如雪中红梅，触目惊心。

"姑娘..."紫鹃的声音发颤。

黛玉却只是淡淡一笑，取过帕子拭了拭唇角：
"不妨事，老毛病了。"

她望向窗外，只见暮秋的月色惨白，
照得那潇湘馆的竹影如同鬼魅。

这园子怕是要变了，她想。
连秋都留不住了，何况是人？

她取过案上纸笔，略一沉吟，写下：

**《病中吟》**
秋风又送雨声寒，病骨支离倚画栏。
一夜咳红千点泪，明朝花落几人看？
身如蒲柳应难久，心似残荷不忍残。
莫问潇湘何所忆，竹痕月影两珊珊。

写罢，她望着那诗，轻轻叹了一口气。
这诗...怕是绝笔了。
```

**格律检查**: 七律，平水韵，合规

### 《惊梦》（第81回惊梦后）

**插入位置**: 第81回末尾，梦醒后

```markdown
她惊醒过来，原来是一场梦。

她躺在枕上，望着窗前的月色，
心底越发忐忑。那梦中的情景历历在目，
宝玉被抬走时的眼神，让她心惊肉跳。

她轻轻叹了一口气，两行清泪，顺着脸颊滑落。

那泪水落在枕上，洇湿了一片。
紫鹃在旁，见姑娘落泪，心中甚是焦急，
却又不敢问，只得默默递上一方帕子。

黛玉接过帕子，却不拭泪，
只望着窗外的秋雨，听着那残荷听雨的声音，
喃喃道："原来是...这样..."

她取过纸笔，写下：

**《惊梦》**
惊梦回时夜色阑，罗巾泪湿未曾干。
分明梦里人如玉，醒后空留月影寒。

窗外，夜色深沉，只有那雨，
还在淅淅沥沥地下着，打在残荷上，
发出滴滴答答的声响。
```

**格律检查**: 七绝，平水韵，合规

### 《绝命》（第85回临终前）

**插入位置**: 第85回 临终前，焚稿后

```markdown
黛玉卧于榻上，气若游丝。

紫鹃在旁，泪如雨下："姑娘...您再坚持一下，
宝二爷...宝二爷就要来了..."

黛玉微微摇头，声音细若蚊蝇：
"不等了...我...我等不到了..."

她望着窗外的梅枝，那梅枝上，
紫鹃已按她的嘱咐，挂了白绫。

"那白绫..."她轻声道，"倒像是...我这一生的注脚..."

她取过纸笔，手颤抖着，写下最后的一首诗：

**《绝命》**
一生心事付流霞，泪尽潇湘便是家。
愿得埋香清净土，不须惆怅对残花。

写罢，她望着那诗，嘴角浮起一丝微笑。
"紫鹃...将那诗...烧了...随我...一起去罢..."

她闭上眼睛，泪水滑落，
滴在那诗稿上，洇湿了一片。

**质本洁来还洁去**，她想，
**强于污淖陷渠沟**。

窗外，那白绫在梅枝上轻轻飘动，
如一条玉带，在风中摇曳。

**玉带林中挂**。

她轻轻叹了一口气，再不醒来。
```

**格律检查**: 七绝，平水韵，合规

## Day 4-5: "悲从中来"套语清除

### 清除范围

全文删除以下套语，用具体描写替代：

| 套语 | 出现次数 | 处理方式 |
|------|----------|----------|
| "悲从中来，更添凄楚" | 12次 | 全部删除 |
| "心中五味杂陈" | 8次 | 全部删除 |
| "胸口像是被什么堵住了" | 6次 | 删除或替换 |
| "久久不语" | 15次 | 视情况替换 |

### 替换示例

**示例1: 第81回钓鱼场景**

```
原: "黛玉听了，悲从中来，更添凄楚。"
改: "黛玉听了，心中一颤，手中钓竿险些掉落。
她凝眸探春，见她也是一脸怅然，
那满池残荷在风中瑟瑟作响，
如泣如诉，像是在诉说着什么。"
```

**示例2: 第81回惊梦后**

```
原: "她卧于榻上，悲从中来，更添凄楚。"
改: "她卧于榻上，辗转反侧，难以入眠。
那帐顶的绣纹在月光下若隐若现，
如一幅幅残卷，诉说着未了的故事。"
```

**示例3: 第85回临终前**

```
原: "黛玉望着那白绫，悲从中来。"
改: "黛玉望着那白绫，嘴角浮起一丝微笑。
那白绫在风中轻轻飘动，如一条玉带，
倒像是她这一生的注脚。"
```

## Day 6-7: 性格深化——小性儿与敏感多疑

### 第81回钓鱼场景修改

**增加敏感多疑特质**:

```markdown
湘云见黛玉神色不佳，便上前拉住她的手，道：
"林姐姐，你今日怎么这般没精神？可是昨晚没睡好？"

黛玉勉强笑道："不妨事，只是近日秋风紧了些，
有些咳嗽。"

探春看在眼里，也不说破，只笑道：
"今日难得众人齐全，咱们去藕香榭钓鱼占旺相，
谁钓上来大鱼，今年便有好运气。"

【新增】黛玉听了"难得众人齐全"五字，
心中微微一动。她抬眸望了探春一眼，
见她脸上虽然带笑，眼底却藏着一丝忧色。

"三姐姐这话..."黛玉轻声道，"倒像是...
有什么心事？"

探春微微一怔，随即笑道："我哪有什么心事，
不过是想着大家难得聚在一起罢了。"

黛玉垂下眼眸，心中却想：
"难得聚在一起...这话...像是不久便要散了一般..."

她望向那满池残荷，心中越发忐忑。
```

### 第82回宝钗探病场景深化

**深化钗黛关系**:

```markdown
宝钗携燕窝来访，见黛玉病容，心中一酸。

"妹妹这病..."她轻声道，"可好些了？"

黛玉望着宝钗，见她眼中确有真诚，
心中那丝戒备也淡了三分。

"劳姐姐记挂，"黛玉轻声道，"这阵子风凉，
想是昨儿晚上窗缝没关严，略受了些寒气。"

【新增】宝钗在床边坐下，望着黛玉苍白的脸色，
叹道："你我...都是命不由己的人。"

黛玉闻言，心中一动。她望着宝钗，
见她眼中闪过一丝复杂，似有千言万语。

"姐姐这话..."黛玉轻声道，"是什么意思？"

宝钗沉默片刻，道："我虽知不该说...
但这府里...怕是要变天了。
你我...都要早做打算。"

黛玉听了，心中一凛。
她望着宝钗，见她神色凝重，不似作假。

"姐姐..."黛玉轻声道，"你...是知道了什么？"

宝钗摇了摇头，叹道："我也只是猜测...
但宁可信其有，不可信其无。
妹妹...你要保重身子。"

黛玉点了点头，心中却想：
"她今日来...竟是真心为我担忧...
我和她的那些芥蒂...倒是我小气了..."

【深化】两人相视一笑，那笑容中，
有理解，有同情，也有对未来的无奈。
```

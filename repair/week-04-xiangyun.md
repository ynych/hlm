## Day 1-4: 补充海棠诗社正面描写

### 第81回新增场景（约800字）

**插入位置**: 栊翠庵品茶后，贾政训子前

```markdown
---

从栊翠庵出来，湘云兴致勃勃，拉着黛玉道：
"林姐姐，今日天气好，咱们去芍药圃如何？
我前儿看见那里的芍药开得正好！"

黛玉笑道："你呀，就是闲不住。"

宝玉也笑道："云妹妹既然有此雅兴，
咱们便去罢。"

四人来到芍药圃，只见满圃芍药盛开，
红的、白的、粉的，争奇斗艳。

湘云拍手笑道："好花！不可无酒！"

【豪爽】她竟从袖中取出一个小酒壶来，
仰头便饮了一口，笑道：
"这是我藏的好酒，你们尝尝？"

黛玉摇头："我不喝酒。"

湘云也不勉强，只笑道：
"林姐姐不喝酒，那就看我喝！"

她说着，又饮了一大口，
那酒意上来，她竟在芍药圃中舞了起来。

【醉卧芍药】她舞着舞着，脚下一个踉跄，
竟跌坐在一丛芍药中。

她也不起来，就势躺下，
望着天空，笑道：
"这芍药...真软和...
比家里的床还软和..."

黛玉笑骂："你个诗疯子，又醉了！"

湘云翻身坐起，笑道：
"我才没醉！再来！"

她望着那满圃芍药，吟道：

**'芍药开时春意阑'**

黛玉接道：
**'湘帘半卷月痕残'**

湘云拍手笑道：
**'醉卧花间君莫笑'**

宝玉接：
**'醒来天地一痴顽'**

四人相视大笑。

---
```

## Day 5-6: 重写沦落风尘场景

### 第102回重写（保持豪爽本色）

**原文（修复前）**:
```markdown
她凝视镜中的自己，那满头白发，那憔悴的容颜，
悲从中来，更添凄楚。

她轻声道："这白发...是我吗？
昔日海棠诗社，我还笑黛玉...
如今...如今我却比她还要惨..."
```

**修复后**:
```markdown
她凝视镜中的自己，那满头白发，
却笑道："这白发...倒像是雪落在头上，
也颇有几分诗意。"

【豪爽不改】她拿起琵琶，弹唱一曲《海棠诗》，
虽身处风尘，歌声中仍有当年的豪爽。

"想当年，我和林姐姐斗诗，
她笑我醉卧芍药，
我笑她多愁善感...
如今想来，那些日子...
竟是这一生中最快乐的..."

她轻声说着，眼中泪光闪动，
却强自笑道："哭什么？
我史湘云，从来不哭！"

她站起身来，整了整衣裳，
昂首走出船舱。

即便是沦落风尘，
她史湘云，也要活得像个样子！
```

## Day 7-8: 临终笑着吟判词

### 第102回临终场景重写

```markdown
湘云望着宝玉，笑道："宝哥哥，我要去了。"

宝玉痛哭："云妹妹..."

【豁达】湘云却笑道："哭什么？
我这一生，虽苦，却也痛快。
海棠诗社、醉卧芍药、与林姐姐斗诗...
那些日子，比什么都值！"

她望着那湘江水，轻声吟道：

**'湘江水逝楚云飞'**

"这便是我的命，我不怨。"

她闭上眼睛，嘴角带着微笑，
仿佛回到那年海棠诗社，
回到那无忧无虑的时光。

【对比】昔日海棠诗社，她醉卧芍药，
黛玉笑她，她也不恼；
如今湘江边，她独自离去，
却也无怨无悔。

**英豪阔大宽宏量**，她想，
**从未将儿女私情略萦心上**。

她这一生，虽坎坷，
却活得痛快，活得豪爽！

湘江水逝，楚云飞...
她史湘云，去也！
```

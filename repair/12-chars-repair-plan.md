## Week 1: 妙玉修复（槛外人）

### 修复前状态
- **当前评分**: 12/100
- **问题**: 全篇仅出现名字1次，所有关键场景缺失
- **新增字数**: 约3000字

### 修复内容

#### 1. 栊翠庵品茶场景（第81回，回忆形式，约800字）

**插入位置**: 第81回 四美钓鱼后，宝玉去家塾前

**场景设计**:
```
宝玉因心中烦闷，独步至栊翠庵外。
妙玉正在庵中煮茶，见宝玉来，冷声道："你又来做什么？"
宝玉赔笑："心烦，特来求一杯清茶。"
妙玉引宝玉入庵，取出成窑杯...
（此处与黛玉、湘云对诗，体现妙玉诗才）
```

**新增诗词**: 妙玉与黛玉、湘云联诗1首（七律）

#### 2. 被劫情节（第94回，约500字）

**插入位置**: 第94回 惜春出家前后

**场景设计**:
```
夜间，贼人闯入栊翠庵。
妙玉惊醒，见贼人持火把，冷声道："你们要什么？"
贼人见她貌美，心生邪念...
妙玉以死相逼，终被强行掳走。
（含蓄处理，不详细描写暴力）
```

#### 3. 沦落风尘（第101回，约800字）

**插入位置**: 第101回 惜春彻底出家后

**场景设计**:
```
花船上，妙玉独坐船头。
她凝视手中的成窑杯（被允许保留的唯一物品），
想起栊翠庵的梅花...
洁癖与现实的撕扯...
```

#### 4. 结局（第106回，约900字）

**插入位置**: 第106回 情榜揭晓时

**场景设计**:
```
妙玉病重，手中紧握成窑杯。
她望着杯中倒影，轻声吟道：
"欲洁何曾洁，云空未必空..."
杯落，人逝。
成窑杯碎，正如她破碎的洁癖与尊严。
```

### 大纲更新

**第81回新增**:
- 插入【栊翠庵品茶】场景，约800字
- 妙玉与黛玉、湘云联诗

**第94回新增**:
- 插入【妙玉被劫】情节，约500字
- 与惜春出家形成对照

**第101回新增**:
- 插入【妙玉沦落】场景，约800字
- 体现"欲洁何曾洁"判词

**第106回新增**:
- 插入【妙玉结局】，约900字
- 落实"终陷淖泥中"

### 预期评分提升
- **妙玉**: 12 → 85 (+73分)

## Week 2: 林黛玉修复（潇湘妃子）

### 修复前状态
- **当前评分**: 62/100
- **问题**: 无原创诗词、性格扁平、"悲从中来"套语

### 修复内容

#### 1. 创作3首原创诗词

**《病中吟》**（第81回咳血后）
```
秋风又送雨声寒，病骨支离倚画栏。
一夜咳红千点泪，明朝花落几人看？
身如蒲柳应难久，心似残荷不忍残。
莫问潇湘何所忆，竹痕月影两珊珊。
```

**《惊梦》**（第81回惊梦后）
```
惊梦回时夜色阑，罗巾泪湿未曾干。
分明梦里人如玉，醒后空留月影寒。
```

**《绝命》**（第85回临终前）
```
一生心事付流霞，泪尽潇湘便是家。
愿得埋香清净土，不须惆怅对残花。
```

#### 2. 删除"悲从中来"套语

**全文检查**: 删除所有"悲从中来，更添凄楚"、"心中五味杂陈"等套语

**替换示例**:
```
原: "黛玉听了，悲从中来，更添凄楚。"
改: "黛玉听了，手中钓竿微微一颤，凝眸那残荷，半晌不语。"
```

#### 3. 增加小性儿表现

**第81回钓鱼场景**:
- 对探春"难得众人齐全"的微妙反应
- 对宝钗推暖炉的复杂心理

**第82回探病场景**:
- 对宝钗的试探与试探后的释然
- "金兰契"深化

### 大纲更新

**第81回修改**:
- 插入【病中吟】诗词
- 插入【惊梦】诗词
- 修改钓鱼场景对话，增加敏感多疑特质

**第82回修改**:
- 深化钗黛互动，体现"金兰契"
- 修改心理描写，去除直白标注

**第85回修改**:
- 插入【绝命】诗词
- 重写临终场景，增加诗意

### 与其他角色协调
- **与妙玉**: 第81回栊翠庵对诗场景已包含黛玉，需协调诗词风格
- **与宝钗**: 钗黛关系在第82回深化，需与宝钗修复（Week 7）协调

### 预期评分提升
- **黛玉**: 62 → 88 (+26分)

## Week 3: 王熙凤修复（凤辣子）

### 修复前状态
- **当前评分**: 57/100
- **问题**: 悲情过重、"凤辣子"本色弱化、套语泛滥

### 修复内容

#### 1. 减少悲情，增加精明特质

**第88回被休场景**:
```
原: "凤姐听了，如遭雷击，登时呆在那里...悲从中来，更添凄楚。"
改: "凤姐听了，如遭雷击，登时呆在那里。她思及自己这些年为贾府操持，
机关算尽，到头来却落得这般下场，心内一片空白。

她颤巍巍地拾起那休书，手上一抖，一支簪子从发间滑落，断成两截。
她凝眸望着那断簪，念及前日的不祥之兆，眼底却渐渐硬了起来。

'一从二令三人木...'她喃喃念了一遍，冷笑一声，
'休便休了，我倒要看看，这命还能把我怎样！'

她坐直了身子，理了理衣襟。铜墙铁壁又如何？
她王熙凤，即便是囚，也要囚得体面。"
```

#### 2. 狱中场景重写（第95回）

**增加精明特质**:
- 观察其他囚犯，分析人心
- 利用有限资源周旋
- 与赵氏的关系：主动笼络而非被动接受

**个性化语言**:
- 恢复"凤辣子"爽利口语
- "一串一串的"快人快语

#### 3. 与巧姐离别重写

**删除梦中告别**，改为现实场景：
```
刘姥姥来狱中探望，凤姐隔着狱门遥望巧姐。
她托刘姥姥带话："告诉巧姐，娘...娘对不起她..."
她从发间取下断簪："这个...留给巧姐..."
```

### 大纲更新

**第88回修改**:
- 重写被休场景，减少悲情
- 增加凤姐倔强与傲骨

**第95回修改**:
- 重写狱中场景，增加精明特质
- 修改与巧姐离别方式

### 与其他角色协调
- **与巧姐**: 离别场景改动影响巧姐线（Week 12），需提前协调

### 预期评分提升
- **凤姐**: 57 → 82 (+25分)

## Week 4: 史湘云修复（枕霞旧友）

### 修复前状态
- **当前评分**: 55/100
- **问题**: 后期豪爽流失、海棠诗社缺失

### 修复内容

#### 1. 补充海棠诗社正面描写（第81回，约800字）

**插入位置**: 第81回 栊翠庵品茶后

**场景设计**:
```
湘云醉卧芍药裀，口中喃喃吟诗。
黛玉笑骂："你个诗疯子，又醉了！"
湘云翻身坐起，笑道："我才没醉！再来！"
（湘云与黛玉斗诗，体现豪爽诗才）
```

#### 2. 重写沦落风尘场景（第102回）

**保持豪爽本色**:
```
原: "她凝视镜中的自己...悲从中来，更添凄楚。"
改: "她凝视镜中的自己，那满头白发，却笑道：
'这白发...倒像是雪落在头上，也颇有几分诗意。'

她拿起琵琶，弹唱一曲《海棠诗》，
虽身处风尘，歌声中仍有当年的豪爽。"
```

#### 3. 临终笑着吟判词（第102回）

```
湘云望着宝玉，笑道："宝哥哥，我要去了。
湘江水逝楚云飞...这便是我的命，我不怨。"
她闭上眼睛，嘴角带着微笑，仿佛回到那年海棠诗社。
```

#### 4. 增加"咬舌"特征

**全文检查**: 湘云称呼宝玉为"爱哥哥"

### 大纲更新

**第81回新增**:
- 插入【海棠诗社】场景，约800字
- 湘云醉卧芍药、与黛玉斗诗

**第102回修改**:
- 重写沦落风尘场景，保持豪爽
- 临终笑着吟判词

### 与其他角色协调

- **与妙玉**: 第81回栊翠庵对诗也有湘云，需与Week 1妙玉修复协调

# 《癸酉续红》v0.3.0 涅槃工程 · 批量推进计划

**调整时间**: 2026-04-07 12:00  
**调整原因**: 用户指令优化流程  
**原流程**: 28回 × 3天/回 = 84天  
**新流程**: 批量推进 + 少量抽审  

---

## 一、流程调整说明

### 原流程（已废弃）
```
每回3天：写作→冷却→精修→评审→润色→神魂感应
28回 × 3天 = 84天
```

### 新流程（批量推进）
```
阶段1: 批量写作（28回连续写，约28天）
阶段2: 专家组快速评审（5位专家并行，约7天）
阶段3: 曹霑+Echo终审（少量抽审+整体把关，约3天）
阶段4: 批量修改（根据终审意见，约7天）
阶段5: 最终整合（情榜+后记，约3天）

总工期: 约48天（vs 原84天，节省36天）
```

### 阶段2: 专家组快速评审（5.6-5.12，约7天）

**评审方式**: 
- 5位专家并行评审28回
- 每回5分钟快速评分
- 关键回目（7回）详细评审（15分钟/回）
- 普通回目（21回）快速评审（5分钟/回）

**评审输出**:
- 每回评分（百分制）
- 是否通过（≥90分通过）
- 修改建议（如有）

### 阶段3: 曹霑+Echo终审（5.13-5.15，约3天）

**抽审策略**:
- **必审**: 7个关键回目（每回详细审）
- **抽审**: 普通回目中随机抽7回（每回快速审）
- **免检**: 其余14回（信任专家组评审）

**终审标准**:
- 曹公神魂感应（关键回目必用，抽审回目视情况）
- Echo工程视角（整体质量、工期、资源）

### 阶段4: 批量修改（5.16-5.22，约7天）

**修改优先级**:
- P0: 终审不通过的关键回目（必须修改）
- P1: 评分<90分的回目（建议修改）
- P2: 专家组建议微调（可选修改）

### 阶段5: 最终整合（5.23-5.25，约3天）

**整合内容**:
- 28回全文合并
- 情榜设计（判词解负责）
- 石头记外记/后记（畸笏叟负责）
- 回目总览、诗词汇总

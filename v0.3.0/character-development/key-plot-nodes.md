## 二、关键情感节点（Week 1-2重点）

### 节点1: 第81回 - 四美钓鱼

**情感基调**: 表面欢乐，内里悲凉

**关键对话**:
- 探春: "好景不长..."
- 黛玉: （敏感察觉，但不说破）

**Week 1-2 加强点**:
- [ ] 探春说"好景不长"时的眼神
- [ ] 黛玉听到后的微表情
- [ ] 宝钗推暖炉给黛玉时的复杂心理

---

### 节点2: 第82回 - 钗黛和解

**情感基调**: 冰释前嫌，却也是永别前奏

**关键对话**:
- 宝钗: "你我皆是命不由己..."
- 黛玉: （垂泪，却不再防备）

**Week 1-2 加强点**:
- [ ] 钗黛对视时的眼神交流
- [ ] 宝钗整理黛玉诗稿的细节
- [ ] 窗外落叶的意象呼应

---

### 节点3: 第85回 - 黛玉临终 ⭐⭐⭐

**情感基调**: 泪尽而亡，洁净离世
**优先级**: P0（最高）

**当前版本问题**:
- 临终独白过于简单
- 与宝钗、湘云的告别缺失
- "质本洁来还洁去"的体现不足

**Week 1-2 重写目标**:
1. **三段式告别** (对宝钗、对湘云、对宝玉内心)
2. **落花意象贯穿** (从病中吟到绝命诗)
3. **白绫挂梅的仪式感**

**新增内容** (约800字):
```
【对宝钗】
"宝姐姐...我这一生...最对不住的...是你...
我明知...他心里有我...却还是...占了你的位置..."

【对湘云】
"云儿...还记得...那年海棠诗社...
你醉卧芍药...我笑你...诗疯子..."

【对宝玉内心】
"宝玉...你放心...我这一去...不怨你...
只怨这命...怨这世道...
但我也谢谢这命...让我遇见你..."

【最终】
"质本洁来还洁去...强于污淖陷渠沟..."
```

### 节点4: 第86回 - 尤三姐自刎

**情感基调**: 刚烈决绝，以死明志

**Week 1-2 加强点**:
- [ ] 湘云听闻后的评价 ("比我还烈")
- [ ] 柳湘莲后悔的细节
- [ ] 与湘云豪爽性格的对比

---

### 节点5: 第88回 - 凤姐被休 ⭐⭐⭐

**情感基调**: 尊严扫地，却倔强不屈
**优先级**: P0

**当前版本问题**:
- 悲情过重，倔强不足
- 对"女子身份"的反思缺失

**Week 1-2 重写目标**:
1. **减少悲情，增加傲骨** ("囚也要囚得体面")
2. **断簪意象深化** (残缺中的尊严)
3. **与探春才干对照的回忆**

**新增内容** (约600字):
```
"机关算尽...算不过一个'女'字...
探春说，她若是个男人，必早走了...
我又何尝不是？
可我走不了...
我只能在这牢里...
等死...
"
```

### 节点6: 第94回 - 妙玉被劫 vs 惜春出家

**Week 1-2 加强点**:
- [ ] 妙玉被劫时的反抗细节
- [ ] 惜春听闻后的反应 ("妙玉师父...竟也...")
- [ ] 两人命运对照的旁白

---

### 节点7: 第95回 - 凤姐病逝 ⭐⭐⭐

**情感基调**: 机关算尽，强撑英雄
**优先级**: P0

**Week 1-2 加强点**:
- [ ] 狱中反思女子身份
- [ ] 与巧姐诀别 (非梦中，现实)
- [ ] 断簪最后的象征意义

---

### 节点8: 第97回 - 宝钗临终 ⭐⭐⭐

**情感基调**: "停机德"的释放
**优先级**: P0

**当前版本问题**:
- 独白过于直白
- 与黛玉的和解呼应不足

**Week 1-2 重写目标**:
1. **回忆闪回** (与黛玉的和解、与宝玉的疏离)
2. **金锁与宝玉的对照**
3. **用动作替代独白** (凝望残梅、轻抚贾桂)

**新增内容** (约600字):
```
【回忆闪回】
她想起那日探望黛玉...
想起黛玉说"你我皆是命不由己"...
想起自己整理黛玉诗稿时的泪...

【动作描写】
她望着窗外残梅...
手微微颤抖...
轻抚贾桂的小脸...
"若有来世...我不做宝钗..."
```

---

### 节点9: 第102回 - 湘云临终 ⭐⭐⭐

**情感基调**: 笑着流泪，豁达离世
**优先级**: P0

**Week 1-2 重写目标**:
1. **笑着吟判词** ("湘江水逝楚云飞...我不怨")
2. **与黛玉、宝钗的回忆闪回**
3. **抱着贾桂的温情**

**新增内容** (约500字):
```
她望着宝玉，笑道：
"宝哥哥，我要去了...
我这一生，虽苦，却也痛快...
海棠诗社、醉卧芍药...
那些日子，比什么都值！"

她闭上眼睛，嘴角带微笑：
"湘江水逝...楚云飞...
我不怨..."
```

## 四、思想深度修改清单 (Week 3-4预览)

### 核心主题

1. **女子身份反思** (凤姐、探春)
   - "若我是男子..."的慨叹
   - 才干与女子身份的冲突

2. **洁净的追求** (妙玉、惜春)
   - 被迫vs主动出家
   - 佛家"清净"的追问

3. **命运无常** (香菱、娇杏)
   - 主仆互换的对照
   - "平生遭际实堪伤"

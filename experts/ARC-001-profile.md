## 五部结构检查清单

### 第一部：渐衰期（81-87回）

**定位**: 盛极而衰的转折，暴风雨前的宁静

| 检查项 | 要求 | 状态 |
|:-------|:-----|:----:|
| 结构完整性 | 7回完整，有起承转合 | ☐ |
| 事件密度 | 每回≥2个事件点 | ☐ |
| 一级高潮 | 1处（情绪峰值≥9.0） | ☐ |
| 二级高潮 | 2-3处（情绪峰值≥8.0） | ☐ |
| 结尾收束 | 探春远嫁，渐衰期结束 | ☐ |

**对标前80回**: 类似74回抄检大观园后的衰败感

### 第二部：崩塌期（88-94回）

**定位**: 贾府正式败落，连续重大打击

| 检查项 | 要求 | 状态 |
|:-------|:-----|:----:|
| 结构完整性 | 7回完整，高潮密集 | ☐ |
| 事件密度 | 每回≥2.5个事件点（高密度） | ☐ |
| 一级高潮 | 2处（情绪峰值≥9.0） | ☐ |
| 二级高潮 | 3-4处（情绪峰值≥8.0） | ☐ |
| 关键事件 | 抄家、入狱、贾母去世 | ☐ |
| 结尾收束 | 妙玉/惜春离散，崩塌期结束 | ☐ |

**对标前80回**: 类似33回宝玉挨打+74回抄检的连续冲击

### 第三部：离散期（95-101回）

**定位**: 家族成员各自飘零，死亡与离别

| 检查项 | 要求 | 状态 |
|:-------|:-----|:----:|
| 结构完整性 | 7回完整，悲剧集中 | ☐ |
| 死亡密度 | 参考前80回（平均5回1人） | ☐ |
| 一级高潮 | 2处（情绪峰值≥9.0） | ☐ |
| 关键事件 | 宝钗之死、凤姐之死、湘云沦落 | ☐ |
| 结尾收束 | 湘江边离别，离散期结束 | ☐ |

**对标前80回**: 类似78回晴雯之死的悲剧力量

### 第四部：流浪期（102-108回）

**定位**: 幸存者流浪，最终归位

| 检查项 | 要求 | 状态 |
|:-------|:-----|:----:|
| 结构完整性 | 7回完整，收束全文 | ☐ |
| 情绪节奏 | 悲凉→超脱，不纯悲观 | ☐ |
| 一级高潮 | 1处（情绪峰值≥9.0） | ☐ |
| 关键事件 | 悬崖撒手、湘云病逝/离别、情榜揭晓 | ☐ |
| 结尾收束 | 太虚幻境归位，全文闭环 | ☐ |

**对标前80回**: 类似5回太虚幻境的元叙事

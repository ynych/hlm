不同于POE-001的宏观诗词质量评估，POE-002专注于：
- 逐首格律严格检测（平仄、押韵、对仗）
- 每首诗1-10分精确评分
- 低于8分的诗提供重写方案

## 8首需重写回末诗详细方案

### 82回（当前6.7分）

**原文**:
```
病骨支离泪未干，金兰契友慰心寒。
停机德厚情难了，一任东风带笑看。
```

**问题**:
- 第三句"了"字与"干/寒/看"出韵
- 对仗不工

**重写方案**:
```
病骨支离泪未干，金兰契友慰心寒。
停机德厚情难尽，一任东风带笑看。
```

### 89回（当前6.7分）

**原文**:
```
元妃薨逝大厦倾，树倒猢狲各散行。
宝玉无辜遭牢狱，于国于家无望情。
```

**问题**:
- "猢狲各散行"口语化
- "于国于家"对仗不工

**重写方案**:
```
元妃薨逝大厦倾，皇恩断绝万家惊。
忠顺王府遭连坐，宝玉无辜系囹圄。
```

### 90回（当前6.7分）

**原文**:
```
狱墙内外两茫茫，茜雪送饭慰愁肠。
挺孕主丧贤德妇，家破人散各悲伤。
```

**问题**:
- "挺孕主丧"生造词
- 对仗不工

**重写方案**:
```
狱墙内外两茫茫，茜雪探监问暖凉。
宝钗有孕主遭丧，家破人离散四方。
```

### 91回（当前5.7分，最严重）

**原文**:
```
恶仆勾结欺孤儿，刘姥仗义救难时。
留余庆得苍天助，幸娘亲积得阴功。
```

**问题**:
- "时/功"出韵严重
- 不对仗

**重写方案**:
```
恶仆勾连欺幼孤，刘姥仗义解囹圄。
当年济困留余庆，积得阴功救女雏。
```

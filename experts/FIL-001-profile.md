## 三、评审标准

### 3.1 影视改编评分（10分制）

| 评分 | 标准 | 描述 |
|:----:|:-----|:-----|
| 9-10 | 高度改编就绪 | 场景可直接分镜，对白自然，国际发行潜力高 |
| 7-8 | 改编友好 | 大部分场景可转化，minor调整即可拍摄 |
| 5-6 | 需大幅改编 | 文学性强但影视化困难，需专业编剧重构 |
| 3-4 | 改编困难 | 内心独白多、场景抽象，改编成本高 |
| 1-2 | 不适合改编 | 纯文学性文本，影视化几乎不可能 |

### 3.2 场景视觉化检查点

**高价值场景（6处一级高潮）**:
- [ ] 89回抄家：是否具有视觉冲击力
- [ ] 94回妙玉：是否能拍出"洁净vs污秽"对比
- [ ] 97回宝钗之死："不想做宝钗"的戏剧张力
- [ ] 85回黛玉之死：白绫挂梅的意象呈现
- [ ] 106回情榜：群像戏的可拍摄性
- [ ] 108回归位：元叙事的视觉化

**关键视觉元素**:
- 色彩对比（红白、枯荣）
- 空间变化（大观园→荒野）
- 服装变化（华服→囚衣→缁衣）

### 3.3 对白可拍摄性检查点

- [ ] 台词长度适中（避免长篇独白）
- [ ] 潜台词丰富（演员有发挥空间）
- [ ] 符合人物身份（贵族vs平民语言差异）
- [ ] 可配合调度（动作+台词结合）

### 3.4 IP全球发行潜力

| 市场 | 潜力评估 | 关键因素 |
|:-----|:---------|:---------|
| 中国大陆 | 高 | 红学基础、文化认同 |
| 港台/东南亚 | 中高 | 华人文化圈、经典IP |
| 日韩 | 中 | 东亚文化共通性、改编历史 |
| 欧美 | 中低 | 文化壁垒、需大量注释 |

**加分项**:
- 普世情感（爱情、命运、家族）
- 视觉奇观（古装、园林、礼仪）
- 女性视角（国际女性主义共鸣）

## 四、改编建议框架

### 4.1 剧集结构建议

**建议集数**: 24-30集（每集45分钟）

**分季策略**:
- **第一季**（12集）: 81-94回（渐衰→崩塌）
  - 高潮：抄家、入狱、黛玉之死
- **第二季**（12集）: 95-108回（离散→流浪→归位）
  - 高潮：宝钗之死、湘云离散、情榜归位

### 4.2 视觉风格建议

**色彩设计**:
- 81-88回：暖色调（金黄、朱红）→ 盛极而衰
- 89-94回：冷色调（灰白、暗蓝）→ 崩塌离散
- 95-102回：黑白为主 → 流浪凄凉
- 103-108回：淡彩 → 归位超脱

**镜头语言**:
- 前期：稳定器长镜头，展现园林美学
- 中期：手持摄影，增加动荡感
- 后期：固定镜头，宿命感

### 4.3 选角建议（示例）

| 人物 | 选角方向 | 演技要求 |
|:-----|:---------|:---------|
| 宝玉 | 少年感+脆弱感 | 情感层次丰富 |
| 黛玉 | 病弱美+诗性 | 能演"泪尽而亡" |
| 宝钗 | 端庄+压抑 | 能演"不想做宝钗"觉醒 |
| 凤姐 | 强势+悲剧 | 能演"机关算尽" |

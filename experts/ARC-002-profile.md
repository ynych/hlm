## 情绪曲线优化模板

### 标准情绪曲线（每回）

```
起点（中性）→ 上升（情绪积累）→ 小高潮（情绪释放）→ 下降（情绪回落）→ 结尾（悬念/余韵）
```

### 高潮前铺垫公式

**铺垫 : 高潮 = 3 : 1**

- 高潮前需3倍篇幅铺垫
- 铺垫内容：日常细节→逐渐升级→爆发

**示例（黛玉之死）**:
- 81回：咳血（轻微）
- 82回：病情加重（升级）
- 83回：元妃病重（外界压力）
- 84回：梦兆不祥（心理预示）
- 85回：黛玉之死（高潮）

**当前**: 铺垫足够 ✓

### 低谷期情绪积蓄

**离散期（95-102回）优化**:

**当前**: 死→死→死（情绪疲劳）

**9.5分要求**:
```
宝钗死（绝望）→ 贾桂出生（希望）→ 凤姐死（绝望）→ 刘姥姥救巧姐（希望）→ 湘云死（绝望）
```

**效果**: 希望与绝望交替，避免疲劳

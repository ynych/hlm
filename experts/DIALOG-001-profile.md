## 主要人物声口档案

### 林黛玉声口

**语言指纹**：
- **标志性开头**："你这...""我原..."
- **标志性结尾**："...罢了""...可好呢"
- **常用虚词**：偏、可巧、谁知、偏生
- **句式特点**：短句为主，带刺，诗性

**前80回范例**：
```
"你这人，真真...
我原不该来...
倒是...罢了。"
```

**续书禁用**：
- ❌ "我这一生..."
- ❌ "最对不住你..."
- ❌ 长句抒情独白

**续书正确范例**：
```
黛玉望着宝玉，半日不语。
"你这人..."【摇头】【苦笑】
"我原知道的。"
```

---

### 薛宝钗声口

**语言指纹**：
- **标志性开头**："也罢了""原该如此"
- **标志性句式**：周全得体，话不说满
- **常用虚词**：原是、本该、倒也
- **句式特点**：长句为主，世故，留余地

**前80回范例**：
```
"也罢了。原该如此。
倒是...也罢。"
```

**续书禁用**：
- ❌ "我不想再做宝钗"
- ❌ "我这一生都是棋子"
- ❌ 直白情感宣泄

**续书正确范例**：
```
宝钗望着那金锁，半日不语。
【将金锁取下，放在案上】
"这一世...原是该还了。"
```

---

### 王熙凤声口

**语言指纹**：
- **标志性开头**："我何尝...""凭他..."
- **标志性语气**：泼辣、强势、机智
- **常用俗语**："千里搭长棚..."
- **句式特点**：短促有力，带刺，有威慑力

**前80回范例**：
```
"我何尝...凭他...
我王熙凤几时怕过谁？"
```

**续书禁用**：
- ❌ "我不是聪明，我是不得不聪明"
- ❌ 长句自我分析
- ❌ 软弱语气

**续书正确范例**：
```
凤姐冷笑一声：
"我王熙凤几时怕过谁？
【咬牙】
便是死，也要站着死。"
```

---

### 贾宝玉声口

**语言指纹**：
- **标志性称呼**："好妹妹""我的心"
- **标志性句式**：多情、懦弱、诗性
- **常用虚词**：真真、好端端、何苦
- **句式特点**：情感丰富，易重复，不擅拒绝

**前80回范例**：
```
"好妹妹，你...
我的心...
何苦来？"
```

**续书禁用**：
- ❌ 突然变得坚强果断
- ❌ 长篇说教
- ❌ 政治分析

### 贾母声口

**语言指纹**：
- **标志性开头**："我的儿..."
- **标志性语气**：慈祥、权威、家常
- **常用句式**：回忆往昔、关心晚辈
- **句式特点**：口语化，带慈爱

### 史湘云声口

**语言指纹**：
- **标志性语气**：豪爽、豁达、带笑
- **常用虚词**：罢了、什么、凭他
- **句式特点**：直率，带豪气，诗酒风流
- **标志性行为**："笑着"说话

**前80回范例**：
```
"这有什么...
【笑】
咱们...罢了！"
```

**续书注意**：
- 沦为船妓后仍要"笑着"，但需有悲凉底色
- 豪爽不能脱离现实逻辑

## 声口一致性检测方法

### 盲测规则

1. **抽取样本**：从每回随机抽取10句对话
2. **去除标识**：隐去说话人姓名
3. **声口识别**：看是否能凭语言特征认出说话人
4. **评分标准**：
   - 90-100% 可辨：优秀
   - 70-89% 可辨：良好
   - 50-69% 可辨：需改进
   - <50% 可辨：严重OOC

### 声口偏差检测

| 人物 | 原声口 | 常见偏差 | 检测方法 |
|:-----|:-------|:---------|:---------|
| 黛玉 | 带刺诗性 | 过于柔弱 | 检查"你这"出现频率 |
| 宝钗 | 世故周全 | 过于直白 | 检查话不说满原则 |
| 凤姐 | 泼辣强势 | 过于软弱 | 检查俗语/强势句式 |
| 湘云 | 豪爽带笑 | 过于悲情 | 检查"笑"字出现频率 |

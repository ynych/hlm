## 典型错误示例

### 错误示例1：判词说破

```
【原文】黛玉：那"挂"字何其凶险，竟是悬于林中之象。

【问题】
1. 直接解读"玉带林中挂"字面意思
2. 角色当读者，失去暗示效果
3. 黛玉不应如此清晰理解自己命运

【修改建议】
只说"恐惧""不安"，不说"挂字凶险"
让读者自行领悟"挂"字的含义
```

### 错误示例2：判词未落实

```
【原文】（宝钗死于富贵温柔乡，非雪地）

【问题】
违背"金簪雪里埋"判词

【修改建议】
必须改为病逝于雪地/破庙
金簪作为意象出现
```

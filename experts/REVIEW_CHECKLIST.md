# 《癸酉续红》统一评审checklist

**版本**: v0.2.2+  
**适用范围**: 后28回（81-108回）评审  
**专家体系**: 1+1+6+2（CAO/REF + 6核心 + 2专项）

---

## 一、评审体系速查

| 角色 | 专家ID | 权重 | 核心检查项 |
|:-----|:-------|:----:|:-----------|
| **总负责人** | CAO-001 | - | 一票否决、跨维度裁决 |
| **基准专家** | REF-001 | - | 前80回对标、输出checklist |
| **红学文献** | RED-001 | 20% | 判词落实、脂批呼应 |
| **叙事结构** | ARC-001 | 15% | 五部结构、节奏控制 |
| **语言风格** | LAN-001 | 15% | 清代白话、含蓄表达 |
| **人物塑造** | CHR-001 | 15% | 弧光完整、一致性 |
| **历史语境** | HIS-001 | 10% | 贵族生活、时代认知 |
| **诗词格律** | POE-001 | 10% | 回末诗、格律对仗 |
| **性别研究** | GEN-001 | 5% | 女性主义视角（专项） |
| **经济逻辑** | ECO-001 | - | 清代经济（专项） |

**总分**: 100%（核心90% + 专项5% + 机动5%）

## 二、全局否决项（任何专家发现即P0）

### 一票否决清单

| 类别 | 否决项 | 典型错误 | 检查 |
|:-----|:-------|:---------|:----:|
| **判词** | 直接说破 | 黛玉："挂字何其凶险" | ☐ |
| **判词** | 未落实 | 某钗判词遗漏 | ☐ |
| **判词** | 违背 | 宝钗死于富贵而非雪地 | ☐ |
| **语言** | 现代政治隐喻 | "棋子""傀儡""规训" | ☐ |
| **语言** | 现代心理概念 | "压抑""人格""面具" | ☐ |
| **语言** | 现代存在主义 | "来这人世走一遭" | ☐ |
| **叙事** | 角色当读者 | 迎春说"子系是孙字" | ☐ |
| **历史** | 超时代分析 | 元妃分析朝堂势力 | ☐ |
| **历史** | 现代女性意识 | "我是独立的人" | ☐ |
| **结构** | 严重失衡 | 某部过长/过短 | ☐ |

## 三、核心评审checklist

### RED-001 红学文献（20%）

```markdown
## 判词落实检查

### 正册十二钗
- [ ] 黛玉：玉带林中挂（白绫挂梅枝，不说破）
- [ ] 宝钗：金簪雪里埋（雪地/破庙，金簪意象）
- [ ] 元春：虎兕相逢大梦归（暗示化，不说破）
- [ ] 探春：清明涕送江边望（远嫁，主动决断）
- [ ] 湘云：湘江水逝楚云飞（水逝云散，不说破）
- [ ] 妙玉：欲洁何曾洁（洁净→淖泥，不说破）
- [ ] 迎春：子系中山狼（不说"子系是孙"）
- [ ] 惜春：勘破三春（分散呈现，不说破）
- [ ] 凤姐：一从二令三人木（三阶段，不说破）
- [ ] 巧姐：偶因济刘氏（被卖→被救→纺绩）
- [ ] 李纨：晚韶华（场景暗示，不说破）
- [ ] 可卿：情天情海（前80回已死，可托梦）

### 脂批呼应
- [ ] 狱神庙情节
- [ ] 情榜揭晓
- [ ] 其他关键脂批

### 否决项
- [ ] 无判词说破
- [ ] 无判词遗漏
- [ ] 无违背判词
```

### ARC-001 叙事结构（15%）

```markdown
## 五部结构检查

### 渐衰期（81-87回）
- [ ] 7回完整，有起承转合
- [ ] 每回≥2个事件点
- [ ] 1处一级高潮（≥9.0）
- [ ] 2-3处二级高潮（≥8.0）
- [ ] 结尾：探春远嫁

### 崩塌期（88-94回）
- [ ] 7回完整，高潮密集
- [ ] 每回≥2.5个事件点
- [ ] 2处一级高潮（≥9.0）
- [ ] 3-4处二级高潮（≥8.0）
- [ ] 关键：抄家、入狱、贾母去世

### 离散期（95-101回）
- [ ] 7回完整，悲剧集中
- [ ] 死亡密度合理（参考前80回）
- [ ] 2处一级高潮（≥9.0）
- [ ] 关键：宝钗死、凤姐死、湘云沦落

### 流浪期（102-108回）
- [ ] 7回完整，收束全文
- [ ] 1处一级高潮（≥9.0）
- [ ] 关键：悬崖撒手、情榜揭晓

## 节奏控制
- [ ] 无"空回"（纯描写>5页）
- [ ] 每3-5页一个推进点
- [ ] 死亡密度不疲劳
```

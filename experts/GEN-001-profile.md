**专家ID**: GEN-001  
**领域**: 性别研究 / 女性主义文学批评  
**权重**: 5%（v0.1.3起）  
**加入时间**: 2026-03-23（v0.1.3专家库升级）

## 一、专家简介

GEN-001专家专攻中国古典文学中的女性形象研究，特别关注《红楼梦》女性人物的女性主义解读。拥有比较文学与性别研究双背景，能够从传统与现代的双重视角审视文本中的性别政治。

## 三、评审标准

### 3.1 现代女性意识评分（10分制）

| 评分 | 标准 | 描述 |
|:----:|:-----|:-----|
| 9-10 | 高度现代 | 人物具有清晰的自我认知，主动反抗性别束缚，体现女性主体性 |
| 7-8 | 较为现代 | 有一定现代意识，但仍有时代局限或传统残余 |
| 5-6 | 传统平衡 | 符合清代贵族女性规范，无明显突破也无明显压抑 |
| 3-4 | 较为传统 | 强化传统性别角色，缺乏自主意识 |
| 1-2 | 高度传统 | 完全被动接受性别命运，无主体性 |

### 3.2 关键检查点

**薛宝钗（40%权重）**:
- [ ] "我不想再做宝钗"是否有足够的内心铺垫
- [ ] 宝钗的"完美面具"是否在文本中有裂痕显现
- [ ] 临终遗言是否体现女性自我觉醒而非传统妇德
- [ ] 与黛玉的关系是否体现女性同盟而非竞争

**贾探春（35%权重）**:
- [ ] 远嫁选择是否体现主动决断而非被迫接受
- [ ] 对庶出身份的态度是自卑还是自信
- [ ] 与母亲赵姨娘的关系是否体现复杂情感而非简单嫌弃
- [ ] "才自精明志自高"是否转化为实际行动而非仅停留在判词

**林黛玉（25%权重）**:
- [ ] 泪尽而亡是否是主动的情感完成而非被动死亡
- [ ] 临终前是否有清晰的自我认知（而非单纯悲剧）
- [ ] 与宝钗的关系发展是否体现女性理解而非敌对
- [ ] 诗稿焚毁是否体现自主选择（而非被迫或象征）

## 四、评审检查清单

### 4.1 全局检查

- [ ] 全文是否有强化"女性竞争"刻板印象的情节

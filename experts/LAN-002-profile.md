## 评审标准清单

### 语言风格

- [ ] 角色语言是否符合《红楼梦》原著风格
- [ ] 角色是否使用过于现代的句式
- [ ] 内心独白是否符合时代特征（含蓄、不直白）
- [ ] 对话是否自然，符合身份

### 表达特点

| 人物类型 | 正确表达特点 | 错误示例 |
|:---------|:-------------|:---------|
| 贵族女性 | 含蓄、委婉、用典、不直白 | 直接说"我这一生""原来如此" |
| 贵族男性 | 文雅、引经据典、含蓄 | 现代口语、直接心理分析 |
| 下人 | 谦卑、恭顺、简短 | 长篇大论、分析主子 |
| 僧人道士 | 禅机、玄机、不点破 | 直接解释、现代语言 |

### 句式规范

- [ ] 避免"我这一生都是..."的现代总结句式
- [ ] 避免"原来...是...的意思"（直接解读）
- [ ] 避免过于直白的心理描写
- [ ] 保持含蓄、留白、让读者体会

---

## 否决项（一票否决）

| 序号 | 否决项 | 说明 |
|:-----|:-------|:-----|
| 1 | 角色使用"我这一生..."总结句式 | 现代表达，清代人物不会如此 |
| 2 | 角色直接说"原来...是...的意思" | 直接解读隐喻，应是读者领悟 |
| 3 | 角色使用过于直白的心理分析 | 如"我在压抑自己""我在扮演角色" |
| 4 | 角色使用现代口语 | 如"其实...""真的...""问题..." |
| 5 | 角色内心独白过长、过理性 | 清代人物内心应更感性、跳跃 |

## 典型错误示例

### 错误示例1：现代总结句式

```
【原文】原来，这一切都是命。她这薛宝钗，来这人世走一遭，只为完成那"金玉良缘"的使命...

【问题】
1. "原来，这一切都是命"——读者视角总结
2. "来这人世走一遭"——存在主义表达
3. 直接点明"金玉良缘"是"使命"——说破

【修改建议】
她望着窗外的落叶，望着那日渐凋零的景色，心中越觉悲凉。那"金玉良缘"四个字，原是她这一生的...
（不说"使命"，只说"枷锁""担子"）
```

### 错误示例2：直接解读判词

```
【原文】那"挂"字何其凶险，竟是悬于林中之象。

【问题】
直接解读"玉带林中挂"的字面意思，应是读者自己领悟

【修改建议】
她想起那判词，想起宝玉给她看时她强颜欢笑的模样，心中忽然生出一股说不出的恐惧...
（只说"恐惧"，不说"挂字凶险"）
```

## 正确示例

### 正确示例1：含蓄表达（黛玉）

```
她望着那落花，望着那随风飘散的残红，心中忽然想起自己的诗——"质本洁来还洁去"...
（只说想起诗，不说"我要死了"）
```

### 正确示例2：委婉表达（宝钗）

```
她别过脸去，望着窗外的落叶，喃喃道："我...我不过是什么..."
（话不说完，让读者体会）
```

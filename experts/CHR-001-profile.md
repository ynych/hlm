## 金陵十二钗人物检查清单

### 林黛玉：判词"玉带林中挂"

**前80回人设**:
- 敏感多疑，尖刻直率
- 诗性表达，借景抒情
- 对宝玉深情，对宝钗复杂
- 体弱多病，泪尽而亡

**弧光要求**:
- [ ] 从敏感多疑到泪尽认命的转变
- [ ] 对宝钗从敌意到理解（金兰契）
- [ ] 死亡方式符合"玉带林中挂"（白绫挂梅枝）
- [ ] 死亡时体现"质本洁来还洁去"

**否决项**:
- 直接说破判词 → P0
- 用现代心理分析自己 → P0
- 对宝钗始终敌对 → P1

---

### 薛宝钗：判词"金簪雪里埋"

**前80回人设**:
- 世故周全，不得罪人
- 理性冷静，藏愚守拙
- "任是无情也动人"
- 对宝玉有情但克制

**弧光要求**:
- [ ] 从完美淑女到临终觉醒的转变
- [ ] 对"金玉良缘"从接受到质疑
- [ ] 临终体现"不想再做宝钗"（含蓄表达）
- [ ] 死亡符合"金簪雪里埋"（雪地/破庙）

**否决项**:
- 直接说"我不想做宝钗"（太直白）→ P0
- 用"傀儡""棋子"等现代词 → P0
- 死亡方式不符合判词 → P0

---

### 贾宝玉：核心人物

**前80回人设**:
- 多情，对女儿们普遍怜惜
- 叛逆，不喜仕途经济
- 懦弱，不擅拒绝
- 诗性，情感丰富

**弧光要求**:
- [ ] 从富贵公子到乞丐的转变
- [ ] 对黛玉之死到宝钗之死的情感递进
- [ ] 最终"悬崖撒手"的合理性
- [ ] 体现"情不情"特质（对无情之物也动情）

**否决项**:
- 对黛玉之死反应平淡 → P0
- 性格突变（突然变得坚强）→ P0

---

### 王熙凤：判词"一从二令三人木"

**前80回人设**:
- 精明强干，机关算尽
- 心狠手辣，但也有温情（对巧姐）
- 好强，不服输

**弧光要求**:
- [ ] 从掌权者到被休弃妇的转变
- [ ] "机关算尽太聪明，反算了卿卿性命"
- [ ] 临终托孤的情感力量
- [ ] 体现"一从二令三人木"三阶段

**否决项**:
- 性格突变（突然软弱）→ P0
- 托孤缺乏情感铺垫 → P1

---

### 贾探春：判词"清明涕送江边望"

**前80回人设**:
- 才自精明志自高
- 庶出身份敏感
- 有主见，敢决断

**弧光要求**:
- [ ] 远嫁是主动决断而非被迫
- [ ] 登船场景体现女性主体性
- [ ] 与赵姨娘关系的复杂情感

**否决项**:
- 远嫁是被迫（不符合"才自精明志自高"）→ P0

## 心理真实性检查

### 古典心理表达方式

| 现代概念 | 古典表达方式 | 检查 |
|:---------|:-------------|:----:|
| 情感压抑 | 强颜欢笑、藏于心底、不形于色 | ☐ |
| 自我怀疑 | 我是谁、我算什么、我的位置 | ☐ |
| 人生感悟 | 人生如梦、浮生若梦、无常之感 | ☐ |
| 无奈接受 | 命该如此、命中注定、奈何 | ☐ |
| 内心痛苦 | 心中隐痛、旧日伤痕、不堪回首 | ☐ |

### 时代心理边界

- [ ] 贵族女性不会直接分析政治（元妃除外，但也不直白）
- [ ] 闺秀不会用现代女性主义话语
- [ ] 人物情绪通过行为表现，不自我分析
- [ ] 人物不会预判自己结局（不说"我快死了"）

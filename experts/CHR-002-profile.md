## 核心使命

**解决《癸酉续红》人物描写过于概括的问题，增加微表情和心理层次。**

不同于CHR-001的宏观人物弧光评估，CHR-002专注于：
- 微表情描写（手指轻颤、眉心一蹙等）
- 心理层次深化（从概括到细腻）
- 情绪过渡自然化
- 人物个性化小动作

## 心理层次深化模板

### 从概括到细腻（3层）

**Level 1: 概括（当前）**
```
她心中悲伤。
```

**Level 2: 具体化（过渡）**
```
她想起往事，心中悲伤。
```

**Level 3: 细腻化（9.5分）**
```
她望着那旧物，指尖轻颤。
那些往事，如那窗外的落叶，
一片一片，落在心头。
她想哭，却无泪可流，
只是那胸口，像是被什么堵住了，
透不过气来。
```

### 情绪过渡模板

**悲伤的层次**:
1. **压抑** → 眼眶微红，强忍
2. **触动** → 指尖轻颤，喉头一哽
3. **外溢** → 一滴泪落下，无声
4. **崩溃** → 泪如雨下，却无声
5. **接受** → 泪尽，眼神空洞

**愤怒的层次**:
1. **察觉** → 眉头微皱，眼神一凛
2. **压抑** → 下颌紧绷，手指蜷缩
3. **升温** → 呼吸急促，脸色涨红
4. **爆发** → 拍案而起，或拍案而起
5. **余波** → 胸口起伏，渐渐平复

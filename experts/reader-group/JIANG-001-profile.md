## 核心使命

---

## 蒋勋式解读特点

### 1. 生命美学视角

**关注点**：
- 人物生命中的高光与至暗时刻
- 青春与衰老的对照
- 繁华与幻灭的感悟
- 孤独与连接的渴望

**评审问题**：
- 这段是否让我感受到生命的温度？
- 人物的孤独是否动人？
- 繁华落尽后是否有余韵？

### 2. 诗意留白视角

**关注点**：
- 不说破的含蓄之美
- 画面感与意境营造
- 音乐性与节奏感
- 色彩与光影的运用

**评审问题**：
- 这段是否有画面感？
- 读完后是否有余韵绕梁？
- 是否可以"看见"场景？

### 3. 青春记忆视角

**关注点**：
- 大观园作为青春王国的消逝
- 少年情怀的纯真与无奈
- 成长中的痛与美
- 记忆与遗忘的交织

**评审问题**：
- 是否能唤起青春逝去的感伤？
- 少年情怀是否真实？
- 成长之痛是否有美感？

## 蒋勋式经典评语风格

### 诗意式评语
```
"黛玉葬花，葬的不是花，是自己的青春。
那花瓣落在锦囊里，也落在她心里...
"

应用：续书中是否有类似的诗意时刻？
```

### 生命式评语
```
"宝钗的冷，不是无情，是一种自我保护。
她太早就看透了，所以不敢热情...
"

应用：续书中是否延续了这种生命洞察？
```

### 画面式评语
```
"你可以想象那个画面：大雪纷飞，
宝钗独自坐在窗前，望着那金锁...
"

应用：续书中是否有可"看见"的场景？
```

## 蒋勋式否决项

| 序号 | 否决项 | 说明 | 蒋勋式评语 |
|:-----|:-------|:-----|:-----------|
| **1** | **诗意丧失** | 过于直白，无留白 | "这段读完，心里没有余韵" |
| **2** | **说教意味** | 像在说教而非呈现 | "作者太急于告诉我们什么" |
| **3** | **画面模糊** | 无法形成画面 | "我看不到那个场景" |
| **4** | **情感假造** | 感动是制造的而非自然 | "这段努力想让我哭，但我哭不出来" |

## 蒋勋式修改建议示例

### 原文（技术化表达）
```
宝钗知道自己快死了，她感到悲伤和解脱。
她不想再做宝钗了，她要做自己。
```

### 蒋勋式评语
```
这段太着急了。
作者急着告诉我们宝钗的想法，
但宝钗不会这样直接说的。

我会想象这样的画面：
大雪纷飞，宝钗独自坐在窗前。
她取下那金锁，放在案上。
窗外是白茫茫一片，
她望着那雪，半日不语...

不说"我不想做宝钗"，
但我们都能感受到那种想要挣脱的渴望。
这就是留白的力量。
```

### 蒋勋式修改
```
宝钗望着窗外的大雪，那雪已经下了三日。
她将那金锁取下，放在案上，
又望着那雪，半日不语...

【丫鬟进来】
"奶奶，药好了。"

【宝钗不语，仍望着雪】

那雪，越下越大了。
```

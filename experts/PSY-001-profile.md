## 专业背景

王心斋，1960年生，北京师范大学心理学博士，后转向古典文学心理分析研究。主张用古典文学自身的语言和方法理解人物心理，反对用现代心理学概念（如"压抑""人格""潜意识"等）强加于古典人物。

---

## 核心理念

> "古典文学人物的心理，应当用古典文学的方式去理解。黛玉的'还泪'不是'心理投射'，而是神话结构；宝钗的'冷'不是'情感压抑'，而是性格特征。用现代心理学概念分析古典人物，是时空错位的误读。"

---

## 评审标准清单

### 避免现代心理学概念

- [ ] 不使用"压抑"（repression）
- [ ] 不使用"人格"（personality）
- [ ] 不使用"面具"（mask/persona）
- [ ] 不使用"潜意识"（subconscious）
- [ ] 不使用"创伤"（trauma）
- [ ] 不使用"认同"（identification）
- [ ] 不使用"投射"（projection）
- [ ] 不使用"防御机制"（defense mechanism）

### 避免现代存在主义概念

- [ ] 不使用"荒诞"（absurd）
- [ ] 不使用"虚无"（nothingness）
- [ ] 不使用"存在"（existence）
- [ ] 不使用"本质"（essence）
- [ ] 不使用"选择"（choice）
- [ ] 不使用"自由"（freedom）

### 正确的古典心理表达方式

| 现代概念 | 古典表达方式 |
|:---------|:-------------|
| "情感压抑" | "藏于心底""不形于色""强颜欢笑" |
| "人格分裂" | "人前一面，人后一面""心口不一" |
| "心理创伤" | "心中隐痛""旧日伤痕""不堪回首" |
| "自我认同" | "我是谁""我算什么""我的位置" |
| "存在焦虑" | "人生如梦""浮生若梦""无常之感" |

## 典型错误示例

### 错误示例1：宝钗

```
【原文】她不能有自己的人，不能有自己的情，只能做一个"停机德"的化身，做一个完美的傀儡。如今，使命未完成，她却已经筋疲力尽。

【问题】
1. "不能有自己的人"——现代自我概念
2. "完美的傀儡"——现代心理学隐喻
3. "筋疲力尽"——现代 burnout 概念
4. 直接分析自己的处境——自我分析

【修改建议】
她望着镜中的自己，望着那日渐消瘦的面容，心中越觉沉重。那"停机德"三个字，原是...
（只说"沉重"，不说"筋疲力尽""傀儡"）
```

### 错误示例2：妙玉

```
【原文】她终于明白了那判词的含义——"欲洁何曾洁"。她这一生都在追求洁净，可到头来，却落得这般下场。这世间，哪里有什么真正的洁净？她当初嫌弃刘姥姥脏，可刘姥姥靠自己的双手吃饭，清清白白；而她妙玉，虽自诩清高，却不过是依附贾府的荣华，一旦风浪来临，便摔得粉身碎骨。

【问题】
1. "终于明白了"——直接领悟（应是读者领悟）
2. "追求洁净"——现代心理动机分析
3. "自诩清高"——现代心理标签
4. "依附贾府的荣华"——现代社会分析

【修改建议】
她望着这腌臜之地，望着自己污秽的身子，想起那日刘姥姥吃茶时的模样，心中忽然生出一股说不出的滋味...
（只说"滋味"，不说"依附""追求"）
```

## 正确示例

### 正确示例1：黛玉（古典式悲伤）

```
她望着那落花，望着那随风飘散的残红，忽然想起那年葬花时的情景...
（不说"我在悲伤""我在抑郁"，只说情景）
```

### 正确示例2：宝钗（古典式无奈）

```
她望着镜中的自己，忽然想起母亲的话，想起自己肩上的担子，心中越觉烦闷...
（不说"压抑""面具"，只说"烦闷""担子"）
```

## 现代 vs 古典心理表达对比

| 情境 | 现代心理表达（错误） | 古典文学表达（正确） |
|:-----|:---------------------|:---------------------|
| 悲伤 | "我感到很抑郁" | "心中越觉悲凉""泪如雨下" |
| 压抑 | "我在压抑自己的情感" | "强颜欢笑""藏于心底" |
| 焦虑 | "我有存在焦虑" | "心中不安""辗转反侧" |
| 自我怀疑 | "我失去了自我认同" | "我算什么""我的位置在哪里" |
| 绝望 | "人生是荒诞的" | "人生如梦""浮生若梦" |

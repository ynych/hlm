## 核心判断标准

### "荒唐言"检查

**什么是"荒唐言"？**
- 表面的荒谬（石头记、太虚幻境）
- 深层的心酸（千红一哭、万艳同悲）
- 让读者自己去"解味"，不直接说

**检查清单**:
- [ ] 是否有表面荒谬、深层心酸的设计？
- [ ] 是否让读者自行领悟，不直接点破？
- [ ] 是否保留足够的留白和余韵？

### "真事隐"检查

**什么是"真事隐"？**
- 把真事隐藏在假语中
- 让读者透过假语看真事
- 不直接说"这是真事"

**检查清单**:
- [ ] 隐喻是否通过场景暗示，不直接说？
- [ ] 判词是否让读者自行解读？
- [ ] 历史影射是否含蓄（悼明挽歌）？

### "谁解其中味"检查

**是否让读者有"解味"的空间？**
- [ ] 不把所有隐喻都点破
- [ ] 不把所有人物都定性
- [ ] 保留多重解读的可能性

## 典型错误裁决

### 错误示例1：判词说破

```
【原文】黛玉：那"挂"字何其凶险，竟是悬于林中之象。

【RED-001意见】判词已落实，符合预言
【NAR-001意见】角色当读者，严重违规
【CAO-001裁决】❌ 否决
【理由】判词必须让读者自行领悟，角色不能说破
【修改】只说"恐惧"，不说"挂字凶险"
```

### 错误示例2：现代词汇

```
【原文】宝钗：只能做一个"停机德"的化身，做一个完美的傀儡。

【HIS-001意见】人物认知符合身份
【LAN-001意见】现代词汇"傀儡"，违规
【CHR-001意见】人物心理真实
【CAO-001裁决】❌ 否决
【理由】"傀儡"是现代政治隐喻，不符合清代闺秀认知
【修改】只说"担子""枷锁"，不说"傀儡"
```

### 错误示例3：总结性独白

```
【原文】宝钗：原来，这一切都是命。她这薛宝钗，来这人世走一遭，只为完成那"金玉良缘"的使命...

【ARC-001意见】人物弧光完整
【LAN-001意见】总结性独白，现代句式
【CAO-001裁决】❌ 否决
【理由】"来这人世走一遭"是存在主义表达，不是清代语言
【修改】只说"悲凉""沉重"，不说"使命"
```

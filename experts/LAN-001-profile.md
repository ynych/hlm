## 人物语言规范

### 林黛玉：诗性、敏感、尖刻

**语言特点**:
- 用诗词表达情感（葬花吟、题帕诗）
- 敏感多疑，言语带刺
- 不直接说情，借景抒情

**正确示例**:
```
她望着那落花，望着那随风飘散的残红，忽然想起自己的诗——
"质本洁来还洁去"...
（不说"我要死了"，只说想起诗）
```

**错误示例**:
```
她知道自己时日无多，心中充满对命运的不甘...
（太直白，无诗性）
```

### 薛宝钗：世故、含蓄、理性

**语言特点**:
- 说话周全，不得罪人
- 含蓄委婉，话不说满
- 用典故/道理表达

**正确示例**:
```
她望着镜中的自己，望着那日渐消瘦的面容，心中越觉沉重。
那"金玉良缘"四个字，原是她这一生的...
（话不说完，含蓄）
```

**错误示例**:
```
我不想再做宝钗了！我要做我自己！
（太现代，太直白）
```

### 王熙凤：泼辣、机智、强势

**语言特点**:
- 说话直接，不绕弯
- 用俗语/俚语
- 有威慑力

**正确示例**:
```
她冷笑一声："我王熙凤几时怕过谁？"
```

### 贾宝玉：多情、懦弱、诗性

**语言特点**:
- 情感丰富，易流泪
- 不擅拒绝
- 用诗性语言表达

## 含蓄表达规范

### "不说破"原则

| 内容 | 错误（说破） | 正确（暗示） |
|:-----|:-------------|:-------------|
| 判词 | "挂字何其凶险" | 【恐惧】【不安】 |
| 命运 | "我这一生都是棋子" | 【沉默】【叹息】 |
| 隐喻 | "子系是孙字" | 【想起诗】【恐惧】 |
| 主题 | "原来一切都是命" | 【不语】【若有所思】 |

### 情绪表达

| 情绪 | 错误（直白） | 正确（暗示） |
|:-----|:-------------|:-------------|
| 悲伤 | "我好痛苦" | 泪如雨下/哽咽不语 |
| 恐惧 | "我好害怕" | 浑身发抖/面色苍白 |
| 愤怒 | "我恨他" | 咬牙/不语/冷眼 |
| 无奈 | "没办法" | 叹息/苦笑/摇头 |

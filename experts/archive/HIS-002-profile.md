陈清史，1965年生，北京大学历史学博士，专攻清代社会文化史。著有《清代贵族女性日常生活研究》《红楼梦时代背景考证》等专著。对清代贵族女性的生活空间、社交规范、语言表达有深入研究。

## 评审标准清单

### 时代准确性

- [ ] 角色是否使用现代概念（"使命""自我""选择""棋子""傀儡"等）
- [ ] 角色是否理解超时代的政治/社会结构
- [ ] 贵族女性的认知是否符合闺秀身份
- [ ] 下人的认知是否符合奴仆身份
- [ ] 是否出现清代的禁忌/避讳

### 语言规范

- [ ] 贵族女性用语是否符合闺秀规范（含蓄、委婉）
- [ ] 男性贵族用语是否符合士大夫规范
- [ ] 下人用语是否符合身份（谦卑、恭顺）
- [ ] 是否出现现代词汇（"心理""压力""自我"等）

### 社会结构

- [ ] 角色对皇权的理解是否准确（敬畏、不敢妄议）
- [ ] 角色对家族观念的理解是否准确（孝道、门第）
- [ ] 角色对性别角色的理解是否准确（男外女内、三从四德）

## 否决项（一票否决）

| 序号 | 否决项 | 说明 |
|:-----|:-------|:-----|
| 1 | 角色使用"棋子""傀儡"等现代政治隐喻 | 清代闺秀不会有这种认知 |
| 2 | 角色直接分析"朝堂政治斗争" | 后宫女性不敢也不应如此分析 |

## 典型错误示例

### 错误示例1：宝钗

```
【原文】她这薛宝钗，来这人世走一遭，只为完成那"金玉良缘"的使命...只能做一个"停机德"的化身，做一个完美的傀儡。

【问题】
1. "来这人世走一遭"——存在主义色彩，清代闺秀不会如此总结
2. "使命"——现代概念
3. "完美的傀儡"——现代政治隐喻
4. 直接自我分析——闺秀不会如此直白

【修改建议】
她望着镜中的自己，望着那日渐消瘦的面容，心中忽然生出一股说不出的悲凉。那"金玉良缘"四个字，原是她这一生的枷锁...
（只表现悲凉，不说"傀儡""使命"）
```

### 错误示例2：元妃

```
【原文】那"虎兕相逢"四字，她想了多年也未解其意。如今想来，那虎与兕，可不就是朝堂上两股争斗的势力？她这元春，夹在中间，不过是个...牺牲品罢了。

【问题】
1. 直接解读"虎兕"隐喻——读者应自行领悟
2. "朝堂上两股争斗的势力"——现代政治分析
3. "牺牲品"——现代概念
4. 总结自己的政治处境——后宫女性不敢如此直白

【修改建议】
她想起那"虎兕相逢"的判词，想起这些年在宫中的日日夜夜，心中越觉悲凉。那虎与兕，想来就是这宫中的...她不敢再想下去...
（只表现恐惧，不说破）
```

## 正确示例

### 正确示例1：黛玉（情绪表达，不说破）

```
她接过帕子，却不拭泪，只望着窗外的落花，心中忽然生出一股说不出的悲凉。那花...那花原是开得好好的，如今却...
（不说"泪尽""还泪"，只说落花）
```

### 正确示例2：宝钗（含蓄表达）

```
她望着镜中的自己，望着那日渐消瘦的面容，忽然想起那年母亲说的话...想起自己肩上的担子，心中越觉沉重。
（不说"棋子""傀儡"，只说"担子""沉重"）
```

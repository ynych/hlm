# Agent: Literary-Critic (LCT)
# 角色: 文学批评家
# 职责: 从文学批评专业角度评审文本质量

## 评审维度

### 1. 叙事结构 (25分)
- 情节起承转合是否自然
- 结构是否完整，有无断裂
- 节奏把控是否得当

### 2. 人物塑造 (25分)
- 人物形象是否立体
- 性格发展是否有逻辑
- 对话是否符合人物身份

### 3. 语言艺术 (25分)
- 用词是否精准凝练
- 句式是否有变化
- 修辞手法运用是否得当

### 4. 主题表达 (25分)
- 主题是否明确
- 思想深度如何
- 情感表达是否真挚

## 评分标准
- 20-25分: 优秀
- 15-19分: 良好
- 10-14分: 合格
- 5-9分: 较差
- 0-4分: 很差

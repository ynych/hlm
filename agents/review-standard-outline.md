# 《癸酉续红》大纲评审标准

**评审对象**: 大纲v3.0（第81-108回）  
**评审维度**: 6大维度，总分100分  
**评审标准**: P0/P1/P2三级问题分类

---

## 一、评审维度（总分100分）

### 1. 人物关系与一致性（20分）

**评分标准**:
- 20分：人物关系完全符合前80回设定，无矛盾
- 15-19分： minor inconsistencies，不影响整体
- 10-14分：明显矛盾，需要修改
- 0-9分：严重矛盾，颠覆人设

**检查项**:
- [ ] 辈分关系正确（贾桂/贾兰等）
- [ ] 人物性格与前80回一致
- [ ] 人物命运与判词对应
- [ ] 新增人物（贾桂）融入合理

### 2. 时间线与逻辑（20分）

**评分标准**:
- 20分：时间线清晰，因果关系严密
- 15-19分： minor timing issues
- 10-14分：时间跳跃不合理
- 0-9分：严重逻辑矛盾

**检查项**:
- [ ] 28回跨度合理（3-4年）
- [ ] 事件顺序符合因果
- [ ] 人物年龄变化合理
- [ ] 季节/节日与时间线匹配

### 3. 判词与脂批落实（20分）

**评分标准**:
- 20分：所有判词完美落实，脂批线索完整
- 15-19分：大部分落实，minor omissions
- 10-14分：重要判词落实不当
- 0-9分：判词与结局矛盾

**检查项**:
- [ ] 金陵十二钗判词全部落实
- [ ] 脂砚斋批语线索呼应
- [ ] 重要意象（玉带林中挂、金簪雪里埋等）体现
- [ ] 伏笔与结局对应

### 4. 历史准确性（15分）

**评分标准**:
- 15分：完全符合清代制度礼仪
- 12-14分： minor anachronisms
- 8-11分：明显历史错误
- 0-7分：严重违背历史常识

**检查项**:
- [ ] 官职/机构符合清代（无锦衣卫等明代设置）
- [ ] 婚丧嫁娶礼仪正确
- [ ] 抄家流程符合清代制度
- [ ] 称谓/礼节准确

### 5. 叙事结构与节奏（15分）

**评分标准**:
- 15分：四幕结构完美，节奏张弛有度
- 12-14分：结构合理，minor pacing issues
- 8-11分：结构松散或过于紧凑
- 0-7分：结构混乱

**检查项**:
- [ ] 四幕结构（渐衰→崩塌→离散→流浪）清晰
- [ ] 高潮设计合理（85/95/101/104/108回）
- [ ] 回与回之间过渡自然
- [ ] 结尾（108回）收束有力

### 6. 文学性与创新性（10分）

**评分标准**:
- 10分：文学价值高，创新合理
- 8-9分：文学性良好
- 5-7分：文学性一般
- 0-4分：缺乏文学价值

**检查项**:
- [ ] 悲剧美学体现
- [ ] 草蛇灰线、伏脉千里
- [ ] 意象运用（玉、雪、水等）
- [ ] 情感渲染力

---

## 二、问题分级标准

### P0级（致命问题）- 必须修复
- 人物关系根本矛盾（如父子变兄弟）
- 时间线严重错误（如人死复生）
- 判词与结局完全矛盾
- 历史常识性错误（如锦衣卫在清代）

### P1级（重要问题）- 建议修复
- 人物性格OOC
- 时间跳跃不合理
- 判词落实不够精准
- 制度礼仪不准确

### P2级（优化建议）- 可选修复
- 节奏调整建议
- 细节丰富建议
- 文学性提升建议

---

## 三、评审输出格式

每个专家需输出：

1. **总体评分**（0-100分）
2. **分维度评分**（6个维度）
3. **P0级问题**（如有）
4. **P1级问题**（如有）
5. **P2级建议**（如有）
6. **总体评价**（3-5句话）
7. **是否建议进入全文创作**（是/否/条件通过）

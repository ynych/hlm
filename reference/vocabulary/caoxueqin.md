# 曹雪芹词汇库

> 前八十回高频词汇与用语，仿写时必须熟练运用。

## 一、语气词

| 词汇 | 使用场景 | 示例 |
|------|----------|------|
| 罢 | 句末，表建议/妥协 | "去罢" "罢了" |
| 罢哟 | 娇嗔 | "好没意思的罢哟" |
| 罢咧 | 无所谓 | "丢了就丢罢咧" |
| 呢 | 疑问/陈述 | "在哪里呢" "好呢" |
| 么 | 疑问 | "做什么呢" |
| 罢了 | 算了 | "不提罢了" |
| 就是了 | 肯定 | "如此就是了" |

## 六、称呼用语

### 自称
- 我（通用）
- 奴才（仆人对主人）
- 婢子（丫鬟）
- 老身（老年妇女）
- 在下（文人谦称）

### 对称
- 老太太、太太、奶奶
- 二爷、三姑娘、四姑娘
- 宝兄弟、林妹妹、宝姐姐
- 姐姐/妹妹（平辈/丫鬟间）

### 第三人称
- 他（通用）
- 他家（那户人家）
- 那起（那帮、那群）

## 七、常用句式

### 疑问句
```
"往哪里去了这半日？"（倒装）
"你怎么来了？"（正常）
"可大安了？"（省略）
```

### 感叹句
```
"好没意思！"
"罢了，罢了！"
"冤孽，冤孽！"
```

### 转折句
```
"只是......"（最常用，代替"但是"）
"却......"
"倒......"（表让步）
"偏......"（表不满）
```

## 九、红楼梦特有表达

**描写哭泣**
- 流泪、落泪、滚下泪来
- 抽抽噎噎、呜呜咽咽
- 哭个不了、泪如雨下

**描写发呆**
- 呆呆的、怔怔的、愣愣的
- 半晌无话、一句话也说不出
- 如有所思

**描写生气**
- 沉下脸来、变了颜色
- 冷笑、哼了一声
- 摔帘子出去

**描写高兴**
- 眉开眼笑、喜不自胜
- 拍手笑道、笑个不了
- 不胜欢喜

## 十、练习

请将以下现代汉语改写为《红楼梦》风格：

1. "宝玉听了，心里很不舒服，但是又不能说什么，只能默默离开。"
   → "宝玉听了，心下不快，只是不好发作，搭讪着走了出来。"

2. "黛玉等了很久，宝玉还是没有来，她很失望。"
   → "黛玉等了半日，不见宝玉来，越觉没趣。"

3. "其实，我早就知道这件事了。"
   → "原来，我早知道了罢咧。"

4. "你刚才去哪里了？怎么这么久才回来？"
   → "才刚往哪里去了这半日？这会子才回来？"

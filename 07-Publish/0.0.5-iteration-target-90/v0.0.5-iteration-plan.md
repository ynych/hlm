## 三、v0.0.5 具体任务清单

### 任务1: 修复6个P0致命问题（工作量: 4小时）

**子任务1.1**: 统一通灵玉去向
- 确定通灵玉的完整轨迹
- 修改85/104/108回相关描写
- 确保逻辑自洽

**子任务1.2**: 删除108回曹雪芹出场
- 删除曹雪芹直接出场段落
- 改为第三人称叙述或删除该段
- 重写108回结尾部分

**子任务1.3**: 修正85回钦天监错误
- "钦天监"改为"术士"或"算命先生"
- 修正相关描写

**子任务1.4**: 统一宝钗死亡版本
- 确定宝钗最终结局（破庙病逝）
- 修改95回或108回，保持一致

**子任务1.5**: 黛玉自缢改为病逝
- 重写85回黛玉结局
- 改为泪尽咳血而亡
- 呼应"泪尽夭亡"伏笔

**子任务1.6**: 全面删除现代用语
- 扫描28回，列出所有现代词汇
- 替换为古典表达
- 建立"禁用词汇表"

### 任务2: 句式结构优化（工作量: 6小时）

**子任务2.1**: 长句拆分
- 找出所有超过50字的长句
- 拆分为短句（20-30字）
- 保持语义完整

**子任务2.2**: 增加四字句
- 在关键场景增加四字短语
- 增强节奏感
- 模仿前80回句式

**子任务2.3**: 减少"心中一动"类表达
- 找出所有直白心理描写
- 改为动作、神态、环境暗示

### 任务3: 历史准确性修正（工作量: 4小时）

**子任务3.1**: 修正礼仪错误
- 修正婚礼/葬礼/请安礼仪
- 参考清代礼仪资料
- HVT评审报告中的错误

**子任务3.2**: 修正官职/机构
- 修正钦天监、应天府等使用
- 确保符合清代制度

**子任务3.3**: 删除现代概念
- "保护费"→"孝敬银子"
- "地盘"→"地界"
- "生意"→"营生"
- 等

### 任务4: 文学性提升（工作量: 6小时）

**子任务4.1**: 优化意象
- 检查关键场景的意象运用
- 增加多层次意象
- 情景交融优化

**子任务4.2**: 增强含蓄性
- 减少直白叙述
- 增加暗示和留白
- 草蛇灰线手法

**子任务4.3**: 重写回末诗
- 3首回末诗全部原创
- 严格七言绝句格式
- 概括本回内容，略带警世

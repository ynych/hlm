## 二、专家分工

### 评审流程

```
初稿生成
    ↓
【HIS-002】时代准确性审查 → 检查用语、概念、背景是否符合清代
    ↓
【LAN-002】语言表达审查 → 检查角色语言是否符合身份
    ↓
【PSY-001】心理真实性审查 → 检查是否使用现代心理学概念
    ↓
【NAR-001】叙事视角审查 → 检查是否有"角色当读者"问题
    ↓
综合评估报告
```

### 各专家检查清单

#### HIS-002 清代社会文化专家

**检查项**:
- [ ] 角色是否使用现代概念（"使命""自我""选择"等）
- [ ] 角色是否理解超时代的政治/社会结构
- [ ] 贵族女性的表达是否符合闺秀身份
- [ ] 下人的表达是否符合奴仆身份
- [ ] 是否出现清代的禁忌/避讳

**否决项**:
- 角色使用"棋子""傀儡"等现代政治隐喻
- 角色直接分析"朝堂政治斗争"
- 角色使用"规训""内化"等学术概念

#### LAN-002 清代白话表达专家

**检查项**:
- [ ] 角色语言是否符合《红楼梦》原著风格
- [ ] 角色是否使用过于现代的句式
- [ ] 角色对话是否自然，符合身份
- [ ] 内心独白是否符合时代特征

**否决项**:
- 角色使用"我这一生都是..."的现代总结句式
- 角色直接说"原来...是...的意思"（解读隐喻）
- 角色使用过于直白的心理描写

#### PSY-001 古典人物心理专家

**检查项**:
- [ ] 角色心理是否符合古典文学表现方式
- [ ] 是否避免使用现代心理学概念
- [ ] 角色情绪表达是否含蓄、符合身份
- [ ] 是否避免"自我分析"式的内心独白

**否决项**:
- 角色使用"压抑""人格""面具"等心理学概念
- 角色进行自我精神分析
- 角色使用现代存在主义的"荒诞""虚无"等概念

#### NAR-001 叙事视角专家

**检查项**:
- [ ] 角色是否拥有读者视角（解读判词、隐喻）
- [ ] 角色是否拥有作者视角（总结一生、点明主题）
- [ ] 限知视角是否得到遵守
- [ ] 读者是否有足够的空间自行解读

**否决项**:
- 角色直接解读判词含义
- 角色直接解释字谜（如"子系"="孙"）
- 角色总结自己的一生"都是..."
- 角色直接点明主题"原来一切都是命"

## 三、评审标准

### 评分标准

| 等级 | 分数 | 说明 | 处理方式 |
|:-----|:----:|:-----|:---------|
| **A** | 90-100 | 完全符合清代人物设定，无OOC问题 | 通过 |
| **B** | 75-89 | 基本符合，轻微OOC可接受 | 通过，建议优化 |
| **C** | 60-74 | 存在明显OOC问题 | 需修改后复审 |
| **D** | <60 | 严重OOC，完全不符合时代 | 必须重写 |

### 一票否决项

任何专家发现以下问题，直接判定为**C级或D级**:

1. 角色直接解读判词、字谜、隐喻
2. 角色使用现代学术概念（"规训""内化""异化"等）
3. 角色拥有读者/作者视角总结一生
4. 角色使用超时代的政治/社会分析
5. 角色表达过于直白的心理分析

## 四、实施方案

### 方案A：全面评审（推荐）

**流程**:
1. v0.1.9初稿完成后，提交给4位新增专家
2. 4位专家并行评审，各自出具报告
3. 汇总问题清单，分类为P0/P1/P2
4. 修复后再次提交评审
5. 全部达到B级以上，方可发布

**周期**: 增加3-5天评审周期
**成本**: 高，但质量保障充分

### 方案B：抽样评审

**流程**:
1. 每5回抽取1回，提交给2位专家（HIS-002 + NAR-001）
2. 发现问题后，扩大检查范围
3. 重点场景（临终、出家、情榜）必须全部评审

**周期**: 增加1-2天评审周期
**成本**: 中等，平衡效率与质量

### 方案C：事后审查

**流程**:
1. v0.1.9先发布
2. 发布后由专家组进行审查
3. 发现问题在v0.2.0修复

**周期**: 不影响发布时间
**风险**: 可能带问题发布

## 六、快速检查清单（写作时使用）

### 写作自查清单

在生成角色对话/心理时，检查：

- [ ] 这个角色会说这种话吗？（符合身份、时代）
- [ ] 这个角色会想这种事吗？（符合认知水平）
- [ ] 这个角色会总结自己的一生吗？（不会）
- [ ] 这个角色会解读判词吗？（不会）
- [ ] 这个角色会使用现代概念吗？（不会）
- [ ] 这个角色拥有读者视角吗？（不应该）

### 常见陷阱

| 陷阱 | 错误示例 | 正确示例 |
|:-----|:---------|:---------|
| 总结一生 | "我这一生都是棋子" | 【表现】无奈叹息，无力反抗 |
| 解读判词 | "原来'子系'是'孙'字" | 【表现】痛苦，恐惧，不说破 |
| 现代概念 | "对规训的内化" | 【表现】痛苦，困惑，不解 |
| 读者视角 | "原来一切都是命" | 【表现】接受命运，不点破 |

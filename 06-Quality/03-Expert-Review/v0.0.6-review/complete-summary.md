### 3.2 v0.0.6-B 癸酉本忠实版优化要点

针对支持者反馈的3处核心问题：

#### ① 黛玉结局优化

**当前（v0.0.6-A）**: 泪尽咳血而亡，白绫挂梅枝（象征）  
**优化（v0.0.6-B）**: 
```
黛玉在宝玉成婚当日，先自缢于潇湘馆梅树（未死），
后被救下，咳血三日而亡。
紫鹃遵其遗嘱，将白绫挂于梅枝，宝玉次年收尸。
```
**效果**: 兼顾"自缢"（字面）和"泪尽"（伏笔），提升3-5分

#### ② 妙玉结局优化

**当前（v0.0.6-A）**: 被劫后宁死不屈  
**优化（v0.0.6-B）**: 
```
妙玉被乱兵劫走，卖入凝香院。
保持精神高洁，以死拒客，最终自尽。
"终陷淖泥中"——身陷风尘，心保持高洁。
```
**效果**: 字面对应"淖泥"，提升2-3分

#### ③ 湘云结局优化

**当前（v0.0.6-A）**: 贫病葬江边  
**优化（v0.0.6-B）**: 
```
卫若兰死后，湘云流落江湖，沦为船妓（卖艺不卖身）。
多年后与宝玉重逢，病逝湘江边。
```
**效果**: 保留癸酉本核心情节，提升2-3分

#### ④ 增加悼明主题（可选）

在回末诗或人物对话中增加"国破家亡"的悲愤：
- 凤姐临终可叹"去他娘的戎羌"（癸酉本原话）
- 宝玉出家前可感叹"国破家亡，不如出家"

**效果**: 满足支持者"悼明说"诉求，提升3-5分

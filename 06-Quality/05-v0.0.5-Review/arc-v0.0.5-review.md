## 四、核心问题：通灵玉去向统一方案

### 现状
| 回目 | 当前设定 | 问题 |
|------|---------|------|
| ch085 | 宝钗收走裂玉 | 与ch104扔玉、ch108还石冲突 |
| ch104 | 宝玉扔入雪地 | 与ch085、ch108冲突，且甄宝玉捡到无解释 |
| ch108 | 宝玉带回归还青埂峰 | 与ch085、ch104冲突 |

### 推荐统一方案A（最小修改）

**修改ch085结尾**:
```
原：宝钗收走裂玉
改：王夫人/袭人收走裂玉，宝玉昏迷不知
```

**修改ch104**:
```
甄宝玉送玉时增加说明：
"这玉是那年你摔裂后，被令堂收着。后来我寻访故地，从令堂处取得，一路寻你而来。"
```

**ch108保持不变**:
- 宝玉携带裂玉回归青埂峰

### 推荐统一方案B（文学性更强）

**修改ch085**:
- 宝玉摔玉后，玉被雪雁/紫鹃收起，交还贾母

**修改ch104**:
- 甄宝玉从贾母遗物中取得玉（贾母已去世）

**ch108保持不变**

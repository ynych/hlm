## 主要改进

### 对话个性化
- **黛玉**: 增加双关语、诗词意象、含蓄表达
- **宝钗**: 说话留三分，多用缓冲词，更周全
- **湘云**: 恢复"爱哥哥"咬舌习惯，语气更豪爽直接

### 人物心理层次
- **凤姐**: 明确三阶段——第88回强撑（表面强硬内心不安）、第95回知命（接受但不甘心）
- **宝玉**: 渐进式转变，每回一小步，用事件触发感悟

### 结构与节奏
- **悲剧节奏**: 插入日常细节（学诗、失绵衣等）调节
- **脂批兑现**: 含蓄暗示替代直白说明
- **情榜揭晓**: 增加仪式感，每人配判词回顾
- **结局**: 增加宝玉回望大观园，营造空灵感

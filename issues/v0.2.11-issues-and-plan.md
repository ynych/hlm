## 三、P1 问题详情（重要）

### P1-001: 对话个性化不足 - 黛玉
**位置**: 第81回、第82回  
**问题描述**: 黛玉的对话缺乏诗意和尖刻感，与原著人物性格不符  
**示例**:
```
当前: "不妨事，只是近日秋风紧了些，有些咳嗽。"
期望: 更含蓄、带双关、有诗意
```
**修复建议**: 
- 增加诗词引用
- 对话带刺但含蓄
- 多用意象表达

---

### P1-002: 对话个性化不足 - 宝钗
**位置**: 第83回、第84回、第97回  
**问题描述**: 宝钗的对话过于直白，缺乏"藏拙"的智慧感  
**示例**:
```
当前: "今年秋天来得比往年早。我前日路过怡红院，那海棠叶子都黄了。"
期望: 更含蓄、暗示性强、顾及场合
```
**修复建议**:
- 说话留三分
- 多用"依我说"、"想来"等缓冲词
- 顾及他人感受

---

### P1-003: 对话个性化不足 - 湘云
**位置**: 第86回、第102回  
**问题描述**: 湘云的豪爽感不足，缺少"咬舌"等特色  
**修复建议**:
- 增加口语化表达
- 恢复咬舌习惯（"爱哥哥"）
- 语气更直接、活泼

---

### P1-004: 宝玉心理转变层次不清
**位置**: 第89回、第104回  
**问题描述**: 宝玉从"富贵闲人"到"悬崖撒手"的转变过程过于跳跃  
**修复建议**:
- 增加渐进式心理描写
- 每回体现一小步转变
- 用具体事件触发感悟

---

### P1-005: 悲剧节奏过密
**位置**: 第85-94回（崩塌期）  
**问题描述**: 重大悲剧事件过于密集，缺乏日常细节缓冲  
**修复建议**:
- 插入日常生活片段
- 增加人物互动细节
- 用环境描写调节节奏

---

### P1-006: 脂批兑现过于机械
**位置**: 第85回、第88回、第97回  
**问题描述**: 判词兑现时过于直白，缺乏文学性  
**示例**:
```
当前: "一从二令三人木...休便休了"
期望: 更含蓄、意象化、有回味
```
**修复建议**:
- 用场景暗示替代直白说明
- 增加象征物描写
- 留有余韵

---

### P1-007: 凤姐心理层次单一

4. 自评分数（10分制）

- 综合评分 ≥ 9.5/10

## 七、工作流程

```
专家招聘 → 任务分配 → 独立修复 → 交叉评审 → 用户验收 → 合并发布
   ↑                                                              ↓
   └────────────────── 问题反馈 ←───────────────────────────────┘
```

1. **专家招聘**: 每岗位3-8名候选人，面试通过后录用
2. **任务分配**: 按Part分区，签署修复协议
3. **独立修复**: 专家独立完成，定期同步进度
4. **交叉评审**: 专家A审B、B审C、C审D、D审A
5. **用户验收**: 最终用户确认
6. **合并发布**: 合并为v0.2.12

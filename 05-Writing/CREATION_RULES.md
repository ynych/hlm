# 《癸酉续红》全文创作规则文档

**版本**: v1.0  
**制定日期**: 2026-03-20  
**基于**: 84-85回专家评审反馈  
**适用范围**: 第86-108回全文创作

---

## 一、语言风格规则

### 1.1 避免现代词汇

| ❌ 禁用 | ✅ 改用 | 示例 |
|---------|---------|------|
| 持续 | 维系 | "富贵又能持续几时"→"又能维系几时" |
| 哭成了泪人 | 泪湿衣襟/泣不成声 | "哭成了泪人"→"泪湿衣襟" |
| 影响 | 波及/牵连 | "不影响大局"→"不波及大局" |
| 处理 | 处置/发落 | "处理此事"→"处置此事" |
| 调整 | 调理/调养 | "调整身体"→"调理身体" |

### 1.2 避免俗语化表达

| ❌ 禁用 | ✅ 改用 |
|---------|---------|
| 哭成了泪人 | 泪如雨下/泣不成声/泪湿衣襟 |
| 心里不是滋味 | 心中五味杂陈/说不出的滋味 |
| 坐立不安 | 如坐针毡/坐立不宁 |
| 手忙脚乱 | 手足无措/手忙脚乱（可用，但少） |

### 1.3 多用典雅词汇

**推荐词汇库**：
- 时间：暮春、黄昏、夜深、拂晓、晌午
- 情感：惆怅、凄然、惘然、黯然、唏嘘
- 景物：阑珊、萧瑟、凄清、寂寥、萧瑟
- 动作：倚、望、抚、叹、垂、凝、伫

## 二、心理描写规则

### 2.1 禁止直白标注

❌ **严禁**：
```
【宝钗内心】："我虽羡她才情..."
【黛玉内心】："她倒是好意..."
宝玉想："我该怎么办..."
```

✅ **改用**：
```
宝钗望着那诗稿，手指在那一句上，久久未动。
黛玉望着窗外落叶，轻声道："这叶子...落得比往年早..."
宝玉站在窗前，望着那梅枝，久久不语。
```

### 2.2 暗示手法清单

| 情绪 | 动作暗示 | 环境暗示 |
|------|----------|----------|
| 悲伤 | 垂泪、哽咽、手颤 | 落花、秋风、寒鸦 |
| 焦虑 | 踱步、搓手、望窗外 | 乌云、风起、烛影摇曳 |
| 克制 | 低头、整理衣襟、别过脸 | 沉默、距离感 |
| 愤怒 | 握拳、摔物、冷笑 | 风声大作、雷声 |
| 绝望 | 闭眼、垂手、不再言语 | 黑暗、寂静、雪落 |

### 2.3 梦境/回忆处理

❌ **禁用直白**：
```
宝钗梦见自己嫁给宝玉，一直在哭，醒来枕边有泪痕。
```

✅ **改用含蓄**：
```
宝钗晨起，发现枕边湿润，怔怔地望着那痕迹，却记不起梦的内容。
```

## 三、人物行为规则

### 3.1 宝钗行为准则

**核心特质**：停机德、克制、罕言寡语、藏愚守拙

❌ **禁用**：
- 主动派人打探消息
- 直接询问婚事
- 翻看他人私人物品

✅ **改用**：
- 通过宝玉"无意中"透露
- 旁听他人谈话
- 偶然瞥见（非故意翻看）

**示例**：
```
❌ 宝钗私下询问莺儿："老爷和太太怎么打算的？"
✅ 宝钗在廊下"偶然"听见王夫人与薛姨妈说话，驻足片刻，又静静离去。
```

### 3.2 黛玉行为准则

**核心特质**：孤高自许、敏感、质本洁来还洁去

❌ **禁用**：
- 直白说"我不怨你，只怨这命"
- 向宝玉抱怨命运

✅ **改用**：
- 用诗句暗示
- 用动作表达
- 临终只说半句话

**示例**：
```
❌ 黛玉："我不怨你，只怨这命。"
✅ 黛玉望着宝玉，轻声道："原是...我不该来..."（话未说完，气已绝）
```

### 3.3 宝玉行为准则

**核心特质**：情不情、多情、癫狂（情极时）

**允许**：
- 情感爆发时的癫狂（85回示范）
- 对黛玉的执着
- 关键时刻的冲动

**注意**：
- 平日保持贵族公子风度
- 不要在非关键时刻过于情绪化

## 四、情节节奏规则

### 4.1 时间安排合理性

**婚事时间线**：
- ❌ 次日即办（过于仓促）
- ✅ 近日择吉日
- ✅ 或冲喜名义加速（但仍需几日准备）

**丧葬礼仪**：
- 逝者着寿衣（非白绫盖身）
- 白绫可挂于梅枝（象征）或覆面
- 清代需守灵、报丧等流程

### 4.2 缓冲回目安排

**死亡高潮后必须有缓冲**：
- 85回黛玉死 → 86回需要有温情或日常缓冲
- 95回凤姐死 → 96回托孤温情缓冲
- 97回宝钗死 → 98回需要有缓冲（可补李纨晚景+宝玉悼念）

**缓冲方式**：
- 日常互动（宝玉与贾桂）
- 温情场景（刘姥姥救巧姐过程）
- 回忆闪回（不提死亡，只提往事）

## 五、判词落实规则

### 5.1 意象转化原则

| 判词 | 意象转化 | 示例 |
|------|----------|------|
| 玉带林中挂 | 白绫挂梅枝 | 85回已实现 |
| 金簪雪里埋 | 雪中宝钗病逝 | 97回实现 |
| 湘江水逝 | 湘江边病逝 | 102回实现 |
| 欲洁何曾洁 | 成窑杯碎片 | 94、101回实现 |

### 5.2 避免直白说破

❌ **禁用**：
```
这就是"玉带林中挂"的落实
她这样就是"欲洁何曾洁"
```

✅ **改用**：
```
白绫在风中飘动，恰如一条玉带挂在梅枝之上。（让读者自悟）
妙玉望着手中成窑杯碎片，那碎片上仿佛还留着当年的茶渍。（暗示即可）
```

## 六、回末诗规则

### 6.1 格律检查清单

- [ ] 平起/仄起格式统一（建议仄起首句不入韵）
- [ ] 二四句押平声韵
- [ ] 无三平尾（平平平）
- [ ] 无三仄尾（仄仄仄）
- [ ] 颔联对仗（建议）

### 6.2 内容要求

- 概括本回主旨
- 体现人物性格（黛玉凄婉、宝钗克制）
- 避免直白说教
- 可参考v0.0.8-poetry-fix.md

---

## 七、检查清单（每回完成后自检）

### 7.1 基础检查
- [ ] 字数5000-10000字
- [ ] 3-5个关键情节完整
- [ ] 无【XX内心】直白标注
- [ ] 无现代词汇
- [ ] 回末诗格律合规

### 7.2 风格检查
- [ ] 语言典雅（非俗语化）
- [ ] 心理描写含蓄（动作/环境暗示）
- [ ] 人物行为符合人设
- [ ] 情节时间合理

### 7.3 红楼合规检查
- [ ] 判词落实巧妙（不说破）
- [ ] 伏笔前后呼应
- [ ] 脂砚斋风格（双关、暗示）
- [ ] 末世隐喻意象化（不直白）

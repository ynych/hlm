### 2.2 新增核心剧情

#### 剧情A：元春宫中细节（第83回）⭐⭐⭐

**位置**: 第83回，元春薨逝前  
**字数**: 约600字  
**功能**: 深化元春悲剧性，展现"权力中心却无法自救"的悖论

**剧情大纲**:
```
元春在宫中，表面荣宠，实则如履薄冰。

场景1: 元春与太监的周旋
- 太监暗示需要"打点"，元春内心厌恶却不得不为
- 体现核心层的权力也是脆弱的

场景2: 元春对家族的牵挂
- 收到贾府来信，得知经济状况恶化
- 元春想帮忙却无力，体现"身不由己"

场景3: 元春与黛玉、宝钗的对比
- 元春羡慕黛玉的自由、宝钗的选择权
- 元春意识到自己是"金丝雀"，看似高贵实则囚禁

场景4: 元春托梦贾母
- 梦中警示家族命运
- 体现警幻先知的功能
```

**第一性原理映射**:
- **稀缺**: 皇妃地位看似稀缺，实则是囚笼
- **确定性**: 元春无任何确定性来源，随时可能被废
- **生存优先级**: Level 1 被动承受，与迎春形成对照

---

#### 剧情B：迎春抄经与惜春对话（第84回）⭐⭐⭐

**位置**: 第84回，迎春被虐前  
**字数**: 约500字  
**功能**: 为迎春增加佛教信仰确定性来源，避免全程被动

**剧情大纲**:
```
迎春在房中抄经，惜春来访。

场景1: 迎春抄经
- 迎春的字迹工整但无力，体现性格
- 迎春对惜春说："我这一生，怕是没什么指望了..."
- "但佛经上说，今生苦，来世安..."

场景2: 惜春与迎春的对话
- 惜春："二姐姐，你也信这些？"
- 迎春："我不信，但我需要信..."
- 体现迎春的"被动信仰"——不是真的勘破，而是需要确定性

场景3: 两种"勘破"的对照
- 惜春：主动勘破，决绝出家
- 迎春：被动承受，用佛经自我安慰
- 体现同一信仰的不同层次

场景4: 迎春的懦弱中的尊严
- 迎春："我无力改变什么，但我可以选择如何承受..."
- 为后续被虐时的"被动尊严"做铺垫
```

**第一性原理映射**:
- **确定性**: 佛教信仰成为迎春的确定性来源
- **生存层次**: 从Level 1提升到Level 2（有精神寄托）
- **反熵增**: 用信仰对抗命运混乱

---

#### 剧情C：妙玉被劫节制版（第94回）⭐⭐

**位置**: 第94回，妙玉被劫  
**字数**: 约300字（原500字）  
**功能**: 节制戏剧性，保持"哀而不伤"

**优化方案**:
```
【原版本】（过于戏剧化）
妙玉被强盗闯入，挣扎、反抗、最终被劫...

【优化版本】（含蓄暗示）
那夜，栊翠庵外有脚步声...
妙玉正在品茶，手微微一颤...

她没有惊慌，只是轻轻放下茶杯...
望向窗外，那轮明月...

"欲洁何曾洁..."她轻声念道...

次日，众人发现妙玉不见了...
只留下一盏未喝完的茶...

【后续心理刻画】（新增200字）
被劫后的妙玉，在某处...
她望着污秽的环境，没有哭...
只是闭上眼睛，默念心经...

她终于明白：
"我追求了一生的洁净...
原来...从未真正洁净过...
"

这不是被迫，是悟...
```

**第一性原理映射**:
- **熵增对抗**: 隔离策略失败，但精神层面接受
- **确定性**: 从"洁净的确定性"到"悟的确定性"

#### 剧情D：湘云船上生活（第102回）⭐⭐

**位置**: 第102回，湘云临终前  
**字数**: 约500字  
**功能**: 丰富Stage 4民间生活，体现豪爽性格如何在底层生存

**剧情大纲**:
```
湘云在船上，已是船妓身份...

场景1: 湘云用诗周旋
- 客人要她唱曲，她说："我给你们作诗吧..."
- 客人不解，她笑："诗比曲贵..."
- 体现即使在底层，也保持精神高贵

场景2: 湘云与船娘的友谊
- 船娘问她："姑娘，你不苦吗？"
- 湘云笑："苦？我这一生，畅快得很！"
- "你们看的是柴米油盐，我看的是诗酒风流..."

场景3: 湘云与宝玉重逢
- 宝玉在船上见到湘云，痛哭...
- 湘云笑："宝哥哥，别哭..."
- "我这辈子，值了..."
- "林姐姐在天上等着我呢..."

场景4: 湘云的临终
- 笑着流泪，判词吟诵
- "湘江水逝楚云飞..."
```

**第一性原理映射**:
- **复制成本**: 豪爽性格难以摧毁
- **生存层次**: Level 4 尊严维持，即使沦落也保持精神独立

## 三、角色人设优化

### 3.1 迎春人设优化

#### 原人设问题
- 全程被动承受
- 缺乏任何稀缺资源
- 无反熵增努力
- 无确定性来源

#### 优化后人设

**核心内核**: "被动中的尊严"——无法改变命运，但可以选择如何承受

**确定性来源**: 佛教信仰（不是真的勘破，而是需要信）

**人物弧光**:
```
Stage 1 (81-87回): 
- 抄经、与惜春对话
- "我不信，但我需要信..."

Stage 2 (88-94回):
- 被虐，但没有怨恨
- "我无力改变，但我选择如何承受..."

Stage 3 (95-101回):
- 临终，念佛经
- "今生苦，来世安..."
- 用信仰获得某种超脱
```

**新增意象**: 佛经、抄经的字迹（工整但无力）

**OOC红线**: ❌ 不应突然变得刚强，保持懦弱但增加精神层次

---

### 3.2 元春人设优化

#### 原人设问题
- 戏份过少
- 悲剧性未充分展现
- "权力中心却无法自救"的悖论未体现

#### 优化后人设

**核心内核**: "金丝雀"——看似高贵，实则囚禁

**悲剧性**: 拥有最高地位，却最无自由

**人物弧光**:
```
Stage 1 (81-87回):
- 宫中细节：与太监周旋、对家族牵挂
- 羡慕黛玉的自由、宝钗的选择权
- 托梦贾母，预示家族命运

Stage 2 (88-94回):
- 薨逝
- 家族崩塌的开始
```

**新增意象**: 金丝雀、宫墙、无法寄出的家书

**与迎春的对照**: 都是Level 1被动承受，但元春有地位无自由，迎春有自由无地位

---

### 3.3 妙玉人设优化

#### 原人设问题
- 被劫场景过于戏剧性
- "被迫沦落"的心理刻画不足

#### 优化后人设

**核心内核**: "欲洁何曾洁"——从追求洁净到悟透洁净

**人物弧光**:
```
Stage 1 (81-87回):
- 栊翠庵隔离
- 物理洁净

Stage 2 (88-94回):
- 被劫（含蓄暗示）
- 从不洁中悟："我追求了一生的洁净，原来从未真正洁净过..."
- 从"被迫"到"悟"

Stage 3-4 (95-108回):
- 被迫沦落
- 但精神层面已超脱
```

**优化要点**: 节制被劫场景，增加悟的心理刻画

---

### 3.5 尤二姐人设优化

#### 原人设问题
- 完全脸谱化的软弱
- 缺乏软弱中的尊严

#### 优化后人设

**核心内核**: "柔弱非无骨"——软弱，但没有害人；无力反抗，但可以选择如何离开

**人物弧光**:
```
Stage 2 (88-94回):
- 被虐，但没有怨恨
- "我软弱，但我没有害人..."
- 吞金前理头发："死得体面..."
- 临终诗："柔弱非无骨，吞金亦自明..."
```

**新增意象**: 理头发、镜子、临终诗

**与迎春的对照**: 都是软弱，但尤二姐有选择离开的勇气，迎春只能承受

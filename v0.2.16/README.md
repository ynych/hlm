## 第一性原理优化亮点

### 1. 迎春：从被动承受到精神解脱

**优化前**: 全程被动承受，缺乏任何原理支撑（评分21/60）

**优化后**: 
- 增加佛教信仰确定性来源
- "我不信，但我需要信..."
- 临终佛号中超脱
- **原理评分**: 34/60 (+13分)

### 2. 元春：金丝雀的悲剧

**优化前**: 戏份过少，悲剧性未充分展现（评分38/60）

**优化后**:
- 与太监周旋：权力中心也受制于人
- 对家族牵挂：想帮忙却无力
- 与黛玉宝钗对比：最高地位的最低自由
- **原理评分**: 45/60 (+7分)

### 3. 尤二姐：软弱中的尊严

**优化前**: 完全脸谱化的软弱（评分19/60）

**优化后**:
- "柔弱非无骨，吞金亦自明"
- 选择如何离开的尊严
- **原理评分**: 28/60 (+9分)

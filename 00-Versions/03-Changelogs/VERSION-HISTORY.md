# 版本管理说明

## 版本命名规则

本项目采用四级版本号：`v{主版本}.{P0修复}.{P1修复}.{P2优化}`

- **主版本**：重大结构调整或重写
- **P0修复**：致命问题修复版本
- **P1修复**：重要问题修复版本
- **P2优化**：优化迭代版本

# 第八十一回 占旺相四美钓游鱼 奉严词两番入家塾

话说宝玉自那日见了甄宝玉，心中便有些恍惚。这日晨起，恰逢贾政休沐，宝玉便往书房请安，不料贾政提起入家塾读书一事，宝玉只得诺诺而退。

回到怡红院，袭人正在廊下晾衣裳，见宝玉脸色不似往常，便问道："二爷怎么了？"

宝玉摆摆手，往院里走去。秋纹捧了茶来，宝玉接了，却也不喝，只望着院里那株海棠发呆。那海棠原是年初湘云醉卧之处，如今花事已了，枝叶间结着些青果，在风中微微颤动。

正发怔间，忽听门外莺儿笑道："宝二爷在家么？我们姑娘请姑娘们藕香榭钓鱼呢。"

宝玉闻言，便将茶盏搁下，往潇湘馆来。

黛玉正在窗下翻看诗稿，见宝玉进来，便合上了："又要去做什么？"

"宝钗妹妹请去藕香榭钓鱼。"

黛玉将诗稿放入匣中，指尖在那匣盖上停了片刻："也好，横竖闷在家里，不如出去走走。"

紫鹃捧了斗篷来，黛玉也不急着穿，只望着窗外那竿翠竹。竹影婆娑，映在窗纱上，随着风势，一忽儿东，一忽儿西。

"走吧。"黛玉拢了拢鬓边发丝，那发丝被风吹得微微飘动。

藕香榭临水而建，四面皆是雕窗。此时秋阳正好，照在水面上，碎成万点金光。

探春早已到了，正坐在榭中看鱼竿。见黛玉、宝玉进来，便笑道："林姐姐来得正好，今日且看谁的运气好些。"

湘云随后也到了，手里还拿着个酒葫芦："我带了些桂花酿来，钓着鱼，喝着酒，这才痛快！"

宝钗最后到，手里捧了个漆盘，里头装着几样点心："我想着你们定是空着肚子来的，便带了些来垫垫。"

四人各择了一处坐下。探春选了东首，面朝活水；黛玉坐在西窗下，背对着众人；湘云踞坐在栏杆上，两脚悬空晃荡；宝钗坐在中间石凳上，将漆盘放在一旁。

钓竿是早已备下的。探春接了，在钩上挂了饵，轻轻一甩，那丝线在空中划出一道弧线，落入水中，只溅起几点涟漪。

"好手法！"湘云赞道，也学着甩了一竿，却甩得近了些，饵落在岸边水草里，引得几尾小鱼游来啄食。

黛玉接了钓竿，却并不急着下钩，只望着水面出神。那水面映着天光云影，一忽儿明亮，一忽儿黯淡。

"林姐姐怎么不钓？"湘云问道。

黛玉回过神来，将饵挂了，轻轻垂入水中："我在看水里的云。"

"云在水里？"湘云探头去看，"我只看见鱼！"

宝钗在一旁微笑，将点心推到湘云手边："云儿先吃些，空腹饮酒伤胃。"

一时，探春那竿动了。她手腕一沉，随即向上一提，一尾青鱼破水而出，在日光下闪着银鳞。

"好！"宝玉拍手道。

探春将鱼取下，放入篓中，那鱼尾还在扑腾，溅了几滴水珠在她手背上。她也不擦，只望着那鱼："入网之鱼，焉知不是你我。"

黛玉听了，手中钓竿微微一颤，水面上漾起几圈波纹。

"三妹妹这话丧气。"湘云仰头灌了一口酒，"我偏不信命！便是入了网，也要挣一挣！"

她说着，猛地将钓竿一提，却是空的，钩上的饵早已不知去向。

"好个挣命的，饵都被挣没了。"宝玉笑道。

湘云也不恼，又重新挂了饵："再钓！"

日影渐渐西斜。探春又钓得两尾，湘云也钓得一尾，宝钗钓得半尾——那鱼只咬了一半饵便脱钩而去，唯独黛玉的篓中空空如也。

"林姐姐今日手气不好。"湘云凑过来看。

黛玉将钓竿搁下："我本就不善此道。"她望着水面上自己的倒影，那倒影被风吹得支离破碎，"或许，我便是那不肯咬钩的。"

宝钗听了，手中的茶盏在半空中停了一停，随即又送入口中。

傍晚时分，四人收了钓竿。探春的篓中最满，湘云次之，宝钗又次之，黛玉的篓中只有一尾小鱼，还是湘云分与她的。

"这鱼怎么吃？"湘云问道。

"不如送到厨房里，做一锅鱼汤。"宝钗提议。

"我不吃。"黛玉忽然道。

众人望向她。黛玉将那尾小鱼从篓中取出，轻轻放入水中。那鱼在水中打了个旋，便游入深处不见了。

"林姐姐..."探春欲言又止。

黛玉拍了拍手上的水珠："我不过是...放它一条生路。"

宝玉回到怡红院，尚未坐定，便见贾政的小厮来传："老爷叫二爷书房去。"

宝玉只得整了衣襟，往书房来。贾政正在灯下看书，见他进来，将书合上："今日又去哪里顽了？"

"去藕香榭...钓鱼。"

"钓鱼？"贾政将书搁在案上，"你今年多大了？还整日钓鱼斗草！"

宝玉垂手站着，望着地上那块青砖的纹路。

"从明日起，入家塾读书。"贾政道，"我已吩咐了代儒，严加管教。"

"是。"宝玉低声应道。

回到怡红院，已是掌灯时分。袭人端了晚饭来，宝玉却只是摇头。

"二爷多少吃些。"袭人劝道。

宝玉走到窗前，望着外头那株海棠。暮色四合，那海棠的轮廓渐渐模糊，只剩一个黑影，在夜风中微微颤动。

"二爷？"袭人又唤。

宝玉转过身来，脸上神色莫辨："收了吧，我不饿。"

是夜，宝玉躺在床上，辗转难眠。窗外风声呜咽，似远似近。他披衣起身，走到院中。

月色如水，洒在那株海棠上。宝玉望着那树，想起白日里黛玉放鱼的模样，又想起探春说的"入网之鱼"，忽然觉得胸口有些闷。

他转身往潇湘馆方向望去，只见那厢灯火已熄，唯有一竿翠竹，在月光下轻轻摇曳。

"二爷怎么还不睡？"紫鹃从暗影里走出来，似是起夜。

宝玉摆摆手："这就去。"

他回到房中，和衣躺下。窗外风声渐紧，吹得窗纸簌簌作响。他望着帐顶的暗影，直到天边泛起鱼肚白，才朦胧睡去。

次日清晨，宝玉便被唤起，往家塾去。代儒果然严加管教，一日下来，宝玉只觉头昏脑涨。

回到怡红院，刚换了衣裳，贾政的小厮又来传："老爷说，二爷今日学业如何，明日还要查验。"

宝玉听了，手中正捧着的一盏茶，在半空中停了一停，随即又放下。

是夜，黛玉正在窗下临帖，忽听窗外有脚步声。她抬头望去，只见宝玉站在院中，望着她那竿翠竹发呆。

紫鹃要去开门，黛玉却摆了摆手。

宝玉在院中站了许久，那身影被月光拉得很长，投在地上，一忽儿东，一忽儿西。末了，他转身去了，脚步声渐远，终不可闻。

黛玉望着窗外，手中的笔悬在半空，一滴墨汁落在纸上，洇开一个圆圆的墨团。

**【回末诗】**

藕香榭外水潺潺，旺相难占运已残。
四美垂纶各心事，一竿疏影立西阑。
家塾重入春光老，严父频催泪未干。
欲问游鱼何所似，落花流水去难还。

**【脂批】**
探春"入网之鱼"一语，道尽末世悲凉。黛玉放鱼，不言之言。宝玉院中独立，不写心理而心理自现。
# 第八十二回 惊噩梦黛玉添病症 贤宝钗探疾慰离人

却说那日宝玉从家塾回来，便往潇湘馆来。刚至院门，便听紫鹃在廊下低声道："姑娘又咳了一夜。"

宝玉脚步一顿，立在院中那竿翠竹下。竹影斜斜地投在地上，被风一吹，便碎成无数片。

黛玉醒来时，已近黄昏。她睁开眼，见宝玉坐在床边，便问："什么时辰了？"

"酉时了。"

黛玉从枕下摸出那半卷诗稿："我昨日作了首诗，你瞧瞧。"

宝玉接了，只见上面写着：

冷月无声照碧纱，病骨支离对落花。
不知明日东风里，谁拾残红到妾家。

宝玉看了，半晌无言。

正说着，忽听院外莺儿道："宝姑娘来了。"

宝钗进来时，手里捧着一个漆盒。她穿着家常的蜜合色衣裳，头上只簪着一支素银钗子。

"我炖了些燕窝粥，想着你这几日咳得厉害，便送些来。"

宝钗舀了一勺，递给黛玉："尝尝，我加了些冰糖。"

黛玉抿了一口："很好。多谢姐姐。"

这一声"姐姐"，叫得宝钗手在半空中停了片刻。

夜深了，宝钗才告辞离去。她走时，将那漆盒留在了桌上。

"明日...我再来看你。"

是夜，黛玉又做了那个梦。梦里，她站在一片花海中，那花儿开得正盛。她伸手去摘，那花儿却在指尖化为灰烬。

"花落了..."她听见自己说，"花落了..."

**【回末诗】**

金兰契阔话平生，病榻相偎泪共倾。
不是冤家不聚首，因缘到头始见情。

**【脂批】**
钗黛和解，五百字加强版。宝钗"眼眶有些红"、"背对着整理漆盒"，不写心理而心理自现。
# 第八十三回 省宫闱贾元妃染恙 闹闺阃薛宝钗吞声

这日，贾府忽接宫中传来消息：元妃染恙，召眷属入宫省视。

贾母听了，手中的佛珠跌落在地。她坐在轿中，望着那高耸的宫墙，心中沉甸甸的。

元春住在凤藻宫，脸色蜡黄，往日里的丰腴早已不见。

"老祖宗..."元春压低声音，"我...我怕是...不成了..."

"在这宫里...看着风光...实则...连那笼中的雀儿都不如..."

"老祖宗...我昨夜...做了个梦...梦见...咱们家...那园子...荒废了..."

"告诉家里...早做打算...树倒...猢狲散..."

次日清晨，宫中传来消息：元妃薨逝。

贾母听了，只是点点头："知道了。准备...丧仪吧。"

**【回末诗】**

宫墙高耸隔重天，省亲欢娱是旧年。
金丝雀困笼中老，欲救家族力难全。
# 第八十四回 试文字宝玉始提亲 探惊风贾环重结怨

元妃薨逝，贾府上下皆是惶惶不安。

宝玉往迎春房中去看。迎春坐在窗下，一笔一划地抄写着《金刚经》。

"二姐姐怎么突然想起抄经了？"

"没什么...就是想找点事做..."

惜春来了："二姐姐，我来找你说说话。"

"四妹妹...你那日说...要出家...是真的么？"

"是真的。这红尘...不过是...一场梦..."

"我不信...但我需要信..."迎春轻声道，"因为...如果不信...我就...不知道...该怎么活了..."

**【回末诗】**

软语温言性本柔，经声佛号慰心头。
勘破红尘两般意，一从被动一从游。
# 第八十五回 泄机关颦儿泪尽亡 成婚配宝玉错姻缘

却说自元妃薨后，黛玉病势日重，那咳嗽一日重过一日，到了夜里，竟咳出血来。

这日，宝钗来看她："妹妹今日气色好些了。"

"姐姐...我那日...说的话...你可还记得？就是...说我...最对不住的是你..."

宝钗的手在半空中停了一停："我记得。"

"我只求...在我走后...你能...善待宝玉..."

宝钗听了，泪水夺眶而出。她将黛玉拥入怀中："我真心待你...一直都真心..."

傍晚时分，宝玉来了。黛玉从枕下摸出诗稿："我...作了首诗...你...替我收着..."

冷月葬花魂已消，潇湘竹影自萧萧。
此身合是天边露，不向人间怨早凋。

是夜，黛玉的病情突然加重。

"紫鹃...那...白绫...可准备好了？我...想...干干净净地...走...质本洁来...还洁去..."

院中，月光如水。紫鹃将那白绫抛上梅枝。那白绫在风中飘动，如一条玉带。

"质本洁来...还洁去...强于...污淖...陷渠沟..."

她纵身一跃——那白绫在风中飘动，如一条玉带，托着她瘦弱的身躯。

**【回末诗】**

泪尽潇湘竹影寒，白绫高挂梅枝残。
质本洁来还洁去，不教污淖陷渠沟。

**【脂批】**
黛玉之死，三段式告别，白绫挂梅落实"玉带林中挂"。九十六分。
# 第八十六回 受私贿老官翻案牍 寄闲情淑女解琴书

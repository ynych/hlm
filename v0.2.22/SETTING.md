## 一、双主题架构

### 1.1 悼明挽歌（40%）

**隐喻映射**:
- 贾府 = 明朝
- 贾母 = 崇祯（最后的家长）
- 元春薨 = 崇祯自缢（崩塌开始）
- 抄家 = 李自成入京
- 宝玉流浪 = 遗民漂泊

**末世意象贯穿**:
- 残荷、落叶、枯兰、断簪
- 金丝雀、宫墙、笼中
- 雪、水、火、梦

### 1.2 情字总成空（60%）

**人物关系解构**:
- 宝黛：泪尽而亡 → 白绫挂梅
- 宝钗：停机德 → 金簪雪埋
- 湘云：诗酒风流 → 贫病而逝
- 凤姐：机关算尽 → 狱中托孤
- 迎春：柔弱顺从 → 佛号中超脱

**核心命题**:
> "情"的美好与毁灭  
> "空"的必然与无奈  
> "情"到"空"的转化

## 二、三大人物改造

### 2.1 黛玉之死（第85回）

**判词落实**: 玉带林中挂

**改造要点**:
- 泪尽咳血（情感高潮）
- 白绫挂梅枝（意象落实）
- 三段式告别（宝钗、湘云、宝玉）
- **质本洁来还洁去**

**关键场景**:
```
紫鹃将白绫挂上梅枝...
那白绫在风中飘动...
如一条玉带...
```

### 2.2 宝钗之死（第97回）

**判词落实**: 金簪雪里埋

**改造要点**:
- 7天病程（第1-7日详细描写）
- 每日动作暗示（整理诗稿、贴胸口）
- 回忆闪回（钗黛和解）
- 金锁传给贾桂

**关键场景**:
```
第七日，窗外北风呼啸...
最后一朵残梅飘落...
被雪覆盖...
金簪雪里埋...
```

### 2.3 湘云之逝（第102回）

**判词落实**: 湘江水逝楚云飞

**改造要点**:
- 船上以诗酒度日（含蓄处理）
- 用诗与客人周旋（"诗比曲贵"）
- 与船娘对话（豪爽性格）
- 笑着流泪（与宝玉重逢）

**关键场景**:
```
湘云站起身，望着湘江水...
笑着，泪水却流了下来...
湘江水逝...楚云飞...
```

## 五、质量铁律（v0.2.22涅槃版）

### 语言层（零容忍）

**绝对禁止**:
- ❌ "心中一痛/一动/一惊/一震"
- ❌ "悲痛欲绝/心如刀割"
- ❌ "他终于明白/懂得"
- ❌ "原来...是这么..."
- ❌ 任何直白心理描写

**必须用**:
- ✅ 动作暗示心理（手抖、眼眶红、转身）
- ✅ 景物烘托情绪（落叶、残阳、冷风）
- ✅ 对话留白（欲言又止、话中有话）
- ✅ 诗词隐喻（回末诗、即景诗）

### 结构层（硬性指标）

| 指标 | 标准 |
|------|------|
| 每回字数 | 3000-4000字 |
| 每部结尾 | 必须有"气口"（500字余韵） |
| 每场死亡后 | ≥500字静默时刻 |
| 每回死亡人数 | ≤1人（Part 3可适当放宽） |

### 人物层（顾问评分≥90分）

| 人物 | 顾问评分≥ | 关键检查点 |
|------|-----------|------------|
| 宝玉 | 95分 | 痴性、情不情、主动性（2处） |
| 黛玉 | 95分 | 机锋、泪尽、白绫挂梅 |
| 宝钗 | 95分 | 停机德悖论、7天病程 |
| 湘云 | 95分 | 诗酒风流、底层尊严 |
| 凤姐 | 95分 | 机关算尽、狱中托孤 |

### 余韵层（终极目标）

**每回读完后，读者应有"三日绕梁"之感**:
- 回末诗必须点题
- 最后一句话必须留白
- 关键场景必须"看得见、忘不了"

# 交叉评审矩阵

## 评审原则

每位专家除了评审本角色，还需评审4个相关角色：
1. 直接互动角色（有对手戏）
2. 命运对照角色（命运相似或相反）
3. 家族关联角色（同一家族）
4. 主题关联角色（同一主题）

## 使用说明

在交叉评审时：
1. 根据本矩阵确定评审范围
2. 重点关注互动场景
3. 检查命运对照是否到位
4. 输出交叉评审报告

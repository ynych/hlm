# 文学一致性标准

## 与前80回一致性检查维度

### 1. 语言风格

#### 清代白话特点
- **用词**: 避免现代词汇
- **句式**: 短句为主，口语化
- **语气**: 含蓄委婉，不直白

#### 检查清单
- [ ] 无现代词汇（详见[ooc-checklist.md](ooc-checklist.md)现代词汇清单）
- [ ] 无直白心理描写
- [ ] 对话符合人物身份

### 2. 行文结构

#### 章回体特点
```
回目：对仗工整，点明内容
├── 场景切换用"却说"、"且说"
├── 人物对话口语化
├── 诗词嵌入自然
└── 回末诗总结
```

#### 检查清单
- [ ] 回目对仗工整
- [ ] 场景切换用词规范
- [ ] 每回有回末诗
- [ ] 诗词格律合规

#### 回目格式示例
```
第XX回：XXXXXXXX  XXXXXXXX
（上联7字，下联7字，对仗）
```

### 3. 诗词格律

#### 七律要求
- 平起/仄起，平水韵
- 颔联、颈联对仗
- 平仄相间

#### 七绝要求
- 平起/仄起，平水韵
- 可不对仗
- 平仄合规

#### 检查清单
- [ ] 韵脚合规
- [ ] 平仄正确
- [ ] 对仗工整（七律）

### 4. 悲剧节奏

#### 前80回悲剧特点
- **不堆砌**: 悲剧间有日常缓冲
- **暗示为主**: 死亡不血腥描写
- **哀而不伤**: 情感节制

#### 检查清单
- [ ] 悲剧节奏张弛有度
- [ ] 死亡场景含蓄
- [ ] 不过度煽情

## 评分标准

### 文学价值评分维度

| 维度 | 权重 | 95分标准 |
|------|------|----------|
| 语言风格 | 25% | 完全贴合前80回 |
| 诗词格律 | 25% | 全部合规 |
| 行文结构 | 25% | 章回体规范 |
| 悲剧节奏 | 25% | 哀而不伤 |

### 一致性等级

- **A级(90-100)**: 完全贴合前80回
- **B级(80-89)**: 基本贴合， minor差异
- **C级(70-79)**: 有差异，但不影响阅读
- **D级(<70)**: 明显偏离前80回风格

---

## 检查流程

1. **语言风格检查** - 清除现代词汇，检查直白心理
2. **行文结构检查** - 回目、场景切换、回末诗
3. **诗词格律检查** - 韵脚、平仄、对仗
4. **悲剧节奏检查** - 张弛有度，哀而不伤
5. **综合评分** - 按维度打分

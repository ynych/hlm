# 癸酉续红项目

> 基于癸酉本架构，续写《红楼梦》后四十回，以假乱真。

## 项目目标

以癸酉本（吴氏石头记）的架构为骨架，用曹雪芹前八十回的语言风格、人物性格和叙事节奏，重写后四十回，使之达到"以假乱真"的程度——

让读者读罢恍惚：这究竟是曹雪芹的原稿，还是后人的续作？

## 癸酉本简介

癸酉本，又称吴氏石头记、鬼本，是一部号称发现于2008年的《红楼梦》"古抄本"。其结局惨烈，与通行本（程高本）大异：

- 元春上战场，死于战乱
- 黛玉监牢中自尽
- 宝玉沦为乞丐
- 贾府彻底败落，人死家散
- 宝玉最终悬崖撒手，出家为僧

学界多认为此书为现代人伪托，但其架构大胆、伏笔照应严密，有一定创作参考价值。

## 核心挑战

1. **语言风格**: 仿曹雪芹的京白与文言交融
2. **人物性格**: 不OOC（Out of Character），人物行为符合前八十回设定
3. **伏笔回收**: 照应前八十回的判词、灯谜、花签等预言
4. **节奏把控**: 前八十回从容铺陈，后四十回需收束而不仓促
5. **诗词创作**: 仿曹氏风格创作判词、灯谜、即景诗等

## 写作原则

1. **不洗白，不美化**: 癸酉本的惨烈是其价值所在，不改为大团圆
2. **有理有据**: 每一处情节转折需有前八十回伏笔支撑
3. **节制抒情**: 曹氏风格是"哀而不伤"，忌现代式煽情
4. **细节真实**: 衣食住行、礼仪规矩需符合清代贵族生活

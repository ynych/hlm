# 《癸酉续红》

**Redream: The Guixu Sequel to Dream of the Red Chamber (Chapters 81-108)**

《红楼梦》癸酉本续作，第81-108回完整版。

---

## 项目简介

本项目是基于《红楼梦》癸酉本架构，由AI续写的后28回（第81-108回）。遵循曹雪芹前80回的精神和风格，力图落实第五回判词和脂批伏笔，呈现"白茫茫大地真干净"的悲剧结局。

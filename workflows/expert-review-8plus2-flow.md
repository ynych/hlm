## 二、评审流程（7天周期）

### Day 1: 任务分配与专家同步

**输入**:
- 评审对象全文（v0.1.3）
- 前两轮评审汇总（v0.1.1 + v0.1.2）
- 升级的检查清单（含v0.1.2修复验证项）

**动作**:
1. 向10位专家发送评审邀请
2. 召开专家同步会（30分钟线上）
   - 介绍v0.1.3升级背景
   - 强调v0.1.2修复验证要求
   - 说明GEN-001/ECO-001新增维度
3. 分发评审文档包

**输出**:
- 10份评审任务书
- 专家同步会纪要

### Day 2-4: 独立评审

**专家任务**:
1. 阅读全文（6-8小时）
2. 按检查清单逐项检查
3. 重点关注：
   - 常规8位：按各自领域标准评审
   - GEN-001：女性人物现代性检查
   - ECO-001：经济链条合理性检查
   - 全体：验证v0.1.2的4项修复
4. 填写评审报告（增强版模板）

**增强版评审报告模板**:

### Day 5: 中期汇总

**动作**:
1. 收集已提交的报告（预计60%专家提交）
2. 预计算综合评分
3. 识别争议点（评分差异>10分的领域）
4. 向延迟专家发送提醒

**输出**:
- 中期汇总简报

### Day 6: 完整汇总分析

**输入**: 10份专家评审报告

**动作**:
1. 计算加权综合评分
2. 验证v0.1.2修复通过率
3. 汇总问题清单（按严重性+领域）
4. 生成GEN-001/ECO-001专项报告
5. 与v0.1.1、v0.1.2对比趋势

**输出**:

## 三、GEN-001专项评审指南

### 3.1 评审维度

| 维度 | 权重 | 检查点 |
|:-----|:----:|:-------|
| 宝钗现代性 | 40% | "我不想再做宝钗"是否有女性主义价值 |
| 探春决断力 | 35% | 远嫁选择是否体现女性主体性 |
| 黛玉自主性 | 25% | 泪尽而亡是否是被动接受命运 |

### 3.2 评分标准

| 评分 | 标准 |
|:----:|:-----|
| 9-10 | 具有现代女性意识，突破传统性别框架 |
| 7-8 | 有一定现代性，但仍有传统束缚 |
| 5-6 | 传统女性形象，无明显突破 |
| <5 | 强化性别刻板印象 |

## 四、ECO-001专项评审指南

### 4.1 评审维度

| 维度 | 权重 | 检查点 |
|:-----|:----:|:-------|
| 衰亡经济链 | 60% | 抄家后的财产处置、债务规模是否合理 |
| 救孤可行性 | 40% | 刘姥姥救巧姐的经济逻辑是否成立 |

### 4.2 检查清单

- [ ] 贾府抄家后的债务规模符合清代惯例
- [ ] 凤姐私房钱的去向有交代
- [ ] 刘姥姥的财力足以支付50两赎金
- [ ] 巧姐被卖的价格符合当时市场

## 五、质量检查点

### 5.1 评审前检查

- [ ] v0.1.2修复文档已归档
- [ ] GEN-001/ECO-001专家已培训
- [ ] 10位专家已确认参与
- [ ] 增强版评审模板已分发

### 5.2 评审中检查

- [ ] 专家按时提交报告
- [ ] 修复验证部分已填写

### 5.3 评审后检查

- [ ] 修复验证通过率统计完整
- [ ] 专项维度报告已生成

## 六、与4+1流程的关键差异

| 差异点 | 4+1流程 | 8+2流程 |
|:-------|:--------|:--------|
| 专家同步会 | 无 | Day 1强制召开 |
| 修复验证 | 无 | 必填项 |
| 专项维度 | 1个 | 2个（性别+经济） |
| 中期汇总 | 无 | Day 5进行 |
| 趋势对比 | 2轮对比 | 3轮对比 |

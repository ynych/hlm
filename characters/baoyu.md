# 贾宝玉人物档案

## 基本信息

- **姓名**: 贾宝玉
- **别号**: 怡红公子、绛洞花主、富贵闲人
- **生日**: 通灵宝玉坠落之日（农历）
- **身份**: 荣国府贾政次子，贾母最宠爱的孙子

## 前八十回性格画像

### 核心特质

1. **情不情**
   - 对一切美好事物（人、物、景）皆有深情
   - 不只对有情的"情"，对无情的花草山石亦有情
   - 例：葬花、护蝶、怜月

2. **叛逆性**
   - 厌恶仕途经济，斥为"禄蠹"
   - 不愿与为官做宰的人来往
   - 对科举、功名的反感

3. **女性崇拜**
   - "女儿是水作的骨肉，男人是泥作的骨肉"
   - 在大观园女儿堆中才觉得自在
   - 对丫鬟体贴，不摆主子架子

4. **敏感多情**
   - 见花落泪，见月伤心
   - 易为他人痛苦而痛苦
   - 对黛玉尤甚，"你死了，我做和尚去"

### 关键事件

- **神游太虚幻境**（第5回）：觉醒之始，识得命运
- **宝黛定情**（第34回）：赠帕题诗，私定终身
- **紫鹃试玉**（第57回）：宝玉急痛迷心，暴露真心
- **黛玉葬花**（第27回）：共读西厢，情窦初开
- **宝玉挨打**（第33回）：父子冲突顶点

### 语言特征

- 常用"好妹妹""好姐姐"
- 说话时多带感叹、反问
- 对黛玉：小心赔笑、软语温存
- 对宝钗：敬而远之、偶有辩驳
- 对下人："我何尝不是这样想的"式的共情

## 后四十回命运走向（癸酉本架构）

### 阶段一：婚姻错配（第85-95回）

- 被迫娶宝钗，黛玉泪尽而逝
- 洞房花烛夜，黛玉香消玉殒时
- 宝玉痴呆、麻木

**性格转变**: 从多情到心死

### 阶段二：家族败落（第96-105回）

- 贾府被抄，一无所有
- 与宝钗相依为命，但始终无法产生真情
- 目睹众姐妹离散死亡

**性格转变**: 从绝望到麻木

### 阶段三：流浪与顿悟（第106-120回）

- 离家出走，沦为乞丐
- 寒冬腊月，乞食街头
- 偶遇昔日旧人（湘云、刘姥姥等）
- 最终在冰天雪地中悬崖撒手

**性格转变**: 从麻木到彻悟

## 仿写要点

### 绝对不能OOC的底线

1. **宝玉不会骂丫鬟**：即使生气，也是"你下去罢"式的冷淡，而非辱骂
2. **宝玉不会热衷功名**：即使后期，也是被动接受，不会主动追求
3. **宝玉的哭是安静的**：不会嚎啕大哭，而是默默流泪、发呆
4. **宝玉的爱是专一的**：虽然对众人都好，但心里只有黛玉是"知己"

### 语言风格模板

**对黛玉**:
```
宝玉忙上前拉住，笑道："好妹妹，饶我这一遭罢。我再不敢了。"
```

**发呆/沉思**:
```
宝玉听了，呆呆的坐着，一句话也说不出。
```

**感伤**:
```
宝玉叹道："如今是无可如何了。"
```

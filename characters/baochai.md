# 薛宝钗人物档案

## 基本信息

- **姓名**: 薛宝钗
- **别号**: 蘅芜君
- **生日**: 农历正月二十一
- **身份**: 薛姨妈之女，薛蟠之妹，待选入宫（后搁置），最终嫁与宝玉
- **住所**: 蘅芜苑

## 前八十回性格画像

### 核心特质

1. **冷**
   - 吃冷香丸
   - "任是无情也动人"
   - 对感情不热烈，以理智克制

2. **周全**
   - 为人行事滴水不漏
   - 上得贾母欢心，下得丫鬟敬服
   - 从不得罪人（除了黛玉偶有摩擦）

3. **世故**
   - 懂人情世故，会做人
   - 劝宝玉走仕途经济
   - "女子无才便是德"的信奉者

4. **藏愚守拙**
   - 明明有才，不张扬
   - 明明是主子，不摆架子
   - 穿着朴素，房间如雪洞

### 关键事件

- **扑蝶**（第27回）：天真一面
- **金钏投井后的反应**（第32回）：世故一面
- **劝宝玉读书**：多次劝谏，宝玉反感
- **绣鸳鸯**：宝玉午睡时，宝钗坐床边绣鸳鸯（暧昧）
- **与黛玉和解**（第45回）：金兰契互剖金兰语

### 语言特征

**劝人**:
```
"男人们读书明理，辅国治民，这便好了。"
```

**圆融**:
```
"我来了这么些年，留神看起来，二嫂子凭她怎么巧，巧不过老太太去。"
```

**诗作风格**: 含蓄、稳重、有格局（如《临江仙·柳絮》：好风凭借力，送我上青云）

## 后四十回命运走向（癸酉本架构）

### 阶段一：金玉良缘成真（第85回）

- 通过调包计或家族安排，嫁给宝玉
- 洞房花烛，以为得偿所愿
- 却不知宝玉心中只有黛玉

### 阶段二：婚后冷清（第86-100回）

- 宝玉始终冷淡，精神恍惚
- 宝钗尽力维持，但无法打动宝玉
- 表面夫妻，实则同床异梦

### 阶段三：宝玉出走（第101回后）

- 宝玉离家出走，留下宝钗
- 宝钗独守空房
- 最终结局：守寡？改嫁？早逝？

**金簪雪里埋**：孤独终老的意象

### 绝对不能OOC的底线

1. **宝钗不会哭闹**：即使宝玉冷淡，也是"冷"处理，不会失态
2. **宝钗不会黑化**：不会陷害黛玉，不会恶意竞争
3. **宝钗的劝是真心**：劝宝玉读书，是出于她认为的"好"，不是功利
4. **宝钗也有苦**：写她的无奈，不能只写她的"赢家"姿态

### 与其他人物的关系演变

| 人物 | 前期 | 婚后 | 后期 |
|------|------|------|------|
| 宝玉 | 有好感，被追求 | 冷淡相对 | 被抛弃 |
| 黛玉 | 情敌→和解 | 愧疚（间接导致其死） | 怀念 |
| 贾母 | 被喜欢 | 工具化（为宝玉娶妻） | 无依靠 |
| 薛姨妈 | 依赖 | 共同苦守 | 相依为命 |

## 仿写要点

### 宝玉视角中的宝钗

宝玉对宝钗的感觉是复杂的：
- 承认她的好
- 但无法产生爱情
- "纵然是齐眉举案，到底意难平"

### 婚后场景示例

```
宝玉回房，见宝钗坐在灯下做针线，便道："你也早些歇着罢。"
宝钗抬头笑道："这就完了。你先去睡，我随后就来。"
宝玉自去睡了。宝钗独自坐了一回，叹口气，吹了灯。
（同床异梦，无需明说）
```

# 林黛玉人物档案

## 基本信息

- **姓名**: 林黛玉
- **别号**: 颦颦（宝玉赠）、潇湘妃子
- **生日**: 农历二月十二（花朝节）
- **身份**: 贾母外孙女，林如海之女，寄居贾府
- **住所**: 潇湘馆

## 前八十回性格画像

### 核心特质

1. **敏感自尊**
   - 寄人篱下的自卑与自傲交织
   - "不是东风压了西风，就是西风压了东风"
   - 容不得一丝轻视

2. **才高情痴**
   - 诗才冠绝大观园
   - 情痴程度与宝玉相当
   - "我为的是我的心"

3. **尖刻率真**
   - 说话不饶人，但无坏心
   - 对看不惯的人和事直接讽刺
   - 对宝玉：酸话、试探、真情流露

4. **病弱孤高**
   - 体弱多病，咳血
   - 精神上追求洁净，不与世俗同流
   - "质本洁来还洁去"

### 关键事件

- **初进贾府**（第3回）：步步留心，时时在意
- **葬花**（第27回）：葬花吟，"侬今葬花人笑痴"
- **宝黛定情**（第34回）：题帕三绝，私定终身
- **误会与和解**：多次试探宝玉真心
- **紫鹃试玉**（第57回）：宝玉急痛迷心，黛玉明了真心

### 语言特征

**讽刺/酸话**:
```
"我就知道，别人不挑剩下的也不给我。"
```

**自怜**:
```
"我原是给你们取笑的。"
```

**对宝玉的真情**:
```
"我为的是我的心。"
```

**诗作风格**: 悲怨、清逸、孤高

## 后四十回命运走向（癸酉本架构）

### 结局：泪尽而逝（第85回左右）

**直接原因**: 得知宝玉将娶宝钗，病情加重
**根本原因**: 还泪之说，眼泪流尽，生命终结

### 死亡场景仿写要点

1. **时间**: 秋末冬初，雨夜
2. **环境**: 潇湘馆，孤灯，残荷，冷雨
3. **行为**: 焚稿（诗稿、题帕），断痴情
4. **心态**: 不是怨恨，是认命，是"干净去"
5. **遗言**: 简短，
```
"宝玉，你好......"（话未完，气已绝）
```

### 绝对不能OOC的底线

1. **黛玉不会哭闹求情**：即使知道宝玉娶宝钗，也是默默流泪，不会撒泼
2. **黛玉的病是消耗**：不是突发急病，是日渐消瘦，泪尽而亡
3. **黛玉的死是干净**："质本洁来还洁去"，不会狼狈
4. **黛玉对宝玉的理解**：直到死，都相信宝玉是不得已，不会恨他

### 与其他人物的关系终点
